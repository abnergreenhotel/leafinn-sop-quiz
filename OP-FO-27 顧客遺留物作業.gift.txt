$CATEGORY: OP-FO-27 顧客遺留物作業

::OP-FO-27-Q01::當房務人員在房間內拾獲顧客遺留的一般衣物時，根據 SOP 流程，第一步應優先執行哪項作業？ {
~立即將物品送到行李房上方的置物籃存放#存放是後續步驟，應先確認房號並嘗試與顧客取得聯繫。
~直接填寫遺失(留)物表單並進行交接#在未聯繫顧客確認處置方式前，不應僅完成表單填寫。
=確認房號並聯繫顧客確認是否需要保留#標準流程的首要步驟是確認來源房號並與客聯繫，以決定後續處理方式。
~在物品上黏貼寫有房號的便條紙#標籤上應註記的是取物人姓名與手機，且聯繫客人的順序在標記之前。
~通知櫃檯主管請其代為撥打電話#一般遺留物處理可由一線同仁直接聯繫，除非是貴重物品或特殊客訴才需主管介入。
}

::OP-FO-27-Q02::關於「黑貓貨到付款」的寄件服務，在與顧客溝通時，下列哪項資訊的提醒是正確的？ {
~運費大約為 $200$ 至 $250$ 元，且當日可達#費用預估與寄送時間均與規範不符，規範中提到的費用約為 $120-150$ 元。
=提醒運費約 $120-150$ 元，需時 $3-5$ 天寄達#此為 SOP 明確要求的提醒內容，包含預估運費與作業時間。
~飯店會負擔此運費，請顧客提供正確地址即可#服務項目為「貨到付款」，意即運費需由收件人負擔，而非飯店負擔。
~若適逢連假黑貓暫停收件，則無法提供寄送服務#規範指示若遇到連假或黑貓暫停時，可轉往 7-11 進行寄件，而非拒絕服務。

}

::OP-FO-27-Q03::若房務同仁在房內拾獲「已開封的飲料」或「易腐敗食品」，應如何處理？ {
~視同一般物品保留 $6$ 個月，等待客人領回#食品具衛生風險且易變質，保留半年不符合衛生與空間管理規範。
~放置於行李房遺留物籃子內，以便隨時拿取#一般物品（衣物等）才放於行李房，食品應存放於冰箱或直接丟棄。
=不留過夜，可於大夜巡檢時至員休丟棄#針對易腐敗及衛生用品，SOP 規定不留過夜並由大夜同仁執行丟棄作業。
~只要是食品，不論包裝是否完整，一律直接丟棄#若客人有特別指定日期領取，且食品可存放時，仍可保留至當天。
~由拾獲的同仁當場處理或自行領用#所有物品處理需依循遺留物 SOP，不可由個人私自處置或自行使用。
}

::OP-FO-27-Q04::當顧客來電詢問是否有遺失物品，且房務同仁回報「未在房間內看到」時，下列哪種應對話術最符合規範？ {
~「我們打掃過兩次都沒看到，您是不是記錯放在哪了？」#這種語氣帶有質疑顧客的意味，違反了「不可讓顧客覺得在懷疑他」的注意事項。
~「沒看到喔，如果有看到的話我們早就聯繫您了。」#回答過於武斷且缺乏服務誠意，未展現會持續追蹤的態度。
=「不好意思，找過房間沒有看到，或是您有印象放在哪邊嗎？我們會再持續尋找。」#此話術展現了協助意願，詢問細節並承諾持續尋找，語氣委婉。
~「房務同仁說沒看到，所以應該是您帶走了。」#將責任推回給顧客會造成負面觀感，並非標準應對方式。
~「請您留下聯絡電話，如果有房務同仁偷拿，我們會再報警處置。」#提及內部同仁操守問題非常不專業，且非標準作業對話內容。

::OP-FO-27-Q05::在「館方遺失物品處理」的情境中（如飯店備品短少），與客人的通話原則為何？ {
~應強硬告知顧客飯店財產不可侵占#態度強硬會引起客訴，應採委婉詢問的方式。
=必須非常留意說話的態度跟話術，不可讓顧客覺得在懷疑他#這是處理館方物品短少時的核心注意點，旨在避免誤會或爭議。
~直接向顧客索取賠償金，若不付費則報警處理#應先確認物品去向，而非直接視為竊盜索賠。
~不需要聯繫顧客，直接從刷卡保證金扣款即可#擅自扣款會造成法律爭議，必須先聯繫顧客確認情況。
~若是飯店物品失蹤，只需填寫報廢單，不必詢問顧客#規範要求需聯繫顧客並小心詢問放置位置。
}

::OP-FO-27-Q06::關於「貴重物品」（如現鈔、手機、護照）的處理方式，下列敘述何者錯誤？ {
~須請主管及房務同仁一同尋找，最好錄影存證#此為正確做法，旨在保護同仁並釐清事實。
~可回報主管並詢問是否可存放於辦公室保險箱#此為正確流程，貴重物品應存放於高安全度地點。
~保留期限為 $6$ 個月#貴重物品與一般物品的保管期限均為 6 個月。
=若保留期滿無人領回，可交由拾獲人自行領回使用#貴重物品若期滿無人認領，應移交警方處理，而非直接給予拾獲人。
~若尋找未果，必須立即回報主管#涉及貴重物品的遺失投訴，回報主管是標準通報機制。
}

::OP-FO-27-Q07::使用德安 PMS 系統新增遺留物資料時，有關「A區」的操作細節，下列何者正確？ {
~遺失日期應填寫顧客入住的那一天#遺失日期應填寫「登記當日」，即發現物品的那天。
=遺失備註應填寫物品目前的放置地點#系統規範明確指出，備註欄位應用於記載物品目前存放之處，方便搜尋。
~不需要點選「查詢住客」，直接手打姓名即可#點選「查詢住客」可自動帶入退房日期與房號，確保資料準確性。
~A區主要是用來填寫「拾獲者」的相關資訊#A區是填寫遺失資訊，B區才是填寫拾獲者（如同仁姓名）的資訊。
~系統會自動抓取所有資料，使用者不需輸入任何文字#使用者仍須依序輸入遺失物品、地點及備註等詳細欄位。
}

::OP-FO-27-Q08::若無法透過電話聯繫上顧客，發送簡訊或電子郵件通知時，應包含哪些內容？ {

::OP-FO-27-Q12::若「一般物品」保留期滿 $6$ 個月且無人領回，經主管核准後，不可採取的處理方式為何？ {

~由拾獲人領回#經主管核准後，拾獲人是有權領回一般物品的。
~轉贈予其他有需要的單位#SOP 中提到可「轉贈」，屬於合規處理。
}
