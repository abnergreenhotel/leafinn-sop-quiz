$CATEGORY: OP-FO-30 公區維護作業

::OP-FO-30-Q01::在執行「每班例行巡檢」時，關於巡視方式的標準要求，下列敘述何者正確？ {
~為節省時間，可以站在櫃台內觀察周邊環境即可#作業規範明確要求巡檢需實際走動查看，而非僅憑目測，以確保無死角遺漏。
=必須實際走動巡視各個區域，不可僅憑目測櫃台周邊#實際走動能發現目測無法察覺的細節，如氣味、照明或地面微小污損，是確保品質的基礎。
~只有在尖峰時段才需要實際走動，一般時段目測即可#無論是否為尖峰時段，標準流程皆要求實際走動查看以維持一致的環境水準。
~巡檢時只需針對公共廁所進行走動查看#巡檢範圍包含大廳、門口、電梯、廁所、休息區及外圍動線，並非僅限於廁所。
~若現場有主管巡視，夥伴即可不必執行該班次的走動巡檢#每班巡檢是夥伴的例行職責，不因主管是否巡視而免除作業程序。
}

::OP-FO-30-Q02::當你在大廳進行「例行清潔工作」時，若現場有顧客在場，應採取何種服務態度與行動？ {
~立即停止所有清潔動作，直到顧客離開為止#完全停止清潔可能導致環境品質下降，應在不影響顧客的前提下繼續作業。
~請顧客暫時移動到其他區域，以便能徹底清潔該處#飯店服務應以顧客體驗優先，要求顧客配合清潔而移動會降低服務滿意度。
=以不打擾顧客動線為原則，靈活調整清潔範圍#在維護環境整潔的同時，確保顧客的活動空間與感受不被干擾是服務標準的核心。
~加速清潔動作，即使產生較大噪音也應快速完成#清潔時的噪音與急促動作會影響顧客的休息品質與環境氛圍。
~無視顧客存在，按照既定計畫完成每一項清潔任務#完全無視顧客會顯得不夠體貼，清潔應隨現場動態調整作業方式。
}

::OP-FO-30-Q03::關於公區備品的補充原則，下列哪一項做法符合「公區備品補充」的標準規定？ {
~為了環保與成本控制，需等衛生紙完全用完再行更換#標準規範禁止等備品完全用完才補充，以免造成顧客使用不便。
~只需在每日早班交班前進行一次全面補充即可#備品補充應視現場消耗狀況隨時調整，而非僅限於特定班次。
=隨時注意餅乾數量，若數量不足應立即補充#大廳餅乾是迎賓區的重要環節，標準要求需隨時注意數量並適時補充以滿足顧客需求。
~迎賓區用品若快用完時，先將其收起以保持陳列整齊#備品短缺時應及時補充而非收起，確保功能性完備才是陳列標準的要求。
~備品補充只需注意數量，擺放位置可隨作業方便調整#補充時需確認擺放整齊並符合「館內陳列標準」，以維持視覺美感。
}

::OP-FO-30-Q04::當進行「區域專案維護」（如植栽區加強整理）時，若需要封閉部分區域，該如何正確執行？ {
~由作業人員自行決定封區時間，不需告知其他單位#專案維護涉及安全與動線，必須經過適當的確認與協調流程。
=先確認作業時段與安全性，並與現場營運動線協調#為了降低對顧客的干擾，任何可能影響動線的維護都必須考慮營運時段與安全防範。
~僅在夜間執行，因此不需考慮與營運動線的協調問題#即使在夜間執行，仍需與櫃台或相關同仁協調，確保現場安全與應變。
~先將封鎖線圍起來後，再回報主管進場作業#應先確認時段與安全後再執行封閉動作，以避免突發的動線混亂。
~直接請房務部負責封區與安全確認即可#專案維護的執行者需負起確認作業時段與安全的責任，而非單方面推給其他部門。
}

::OP-FO-30-Q05::巡檢時發現公共區域有無法立即排除的污損或漏水，最正確的處理流程是什麼？ {
~先假裝沒看見，等下一個班次的人員來處理#異常狀況必須立即回報，推遲處理可能會導致問題擴大或損及安全。
=立即回報並視情況聯繫房務、工務或主管，並設置提醒警示#回報、尋求專業支援與設置警示是處理異常的三大核心步驟，確保問題能被追蹤且不影響顧客安全。
~不需回報，自行嘗試修理直到修好為止#超出能力的清潔或故障應尋求專業部門（如工務、房務）處理，以防造成二次損壞。
~只在交班本記錄即可，不需現場設置警示#無法立即排除的異常若不設置提醒或警示，可能會導致顧客意外受傷。
~等待每月的區域專案維護時一併處理#漏水或設備故障屬於「異常回報與追蹤」範疇，應即時處理而非延至專案維護。
}

::OP-FO-30-Q06::關於「確認與交接」作業，下列哪一項行動是下班前的關鍵標準程序？ {
~只要把公區燈光關掉就可以離開#公區維護包含多項檢查與準備工作，單純關閉燈光不符合交接標準。
~僅需口頭告訴下一班同仁大致狀況即可#未完成事項與異常案件必須「寫入交班」，以確保有紀錄可查且權責分明。
=下班前再巡視一次重點公區，並將未完成事項與待補耗材寫入交班#最後的巡視能確保交接時的公區水準是良好的，並讓接班者能快速掌握待辦重點。
~將所有備品藏起來，以免下一班太快用完#交接應致力於公區維護結果可被接手，惡意隱藏備品違反職業道德與作業標準。
~若本班次很忙，未完成的巡檢可不寫入交班#無論原因為何，未完成的事項都必須確實交接，這也是維護整體品質的關鍵。
}

::OP-FO-30-Q07::根據標準作業流程，下列哪一個時間點是進行公區巡檢的必要時刻？ {
=每班開始與尖峰前#在班次開始及人潮高峰前巡檢，能預先發現問題並確保提供最完善的環境給顧客。
~僅在早班開始時進行一次#規範明確標示為「每班」例行巡檢，意指早、中、晚班皆需執行。
~顧客投訴環境髒亂後才進行#巡檢應是「主動預防」而非「被動補救」，需按照班別規律執行。
~每個月執行一次大規模巡檢#每班的巡檢屬於「例行」性質，與每月一次的「專案維護」不同。
~只有在主管巡場前才需要預演巡檢#巡檢應是為了維持服務品質與顧客安全，而非僅為了應付主管檢查。
}

::OP-FO-30-Q08::在例行清潔工作中，對於「大廳桌上植栽」的維護要求為何？ {
~每日噴灑亮光劑以保持翠綠#作業說明中僅提到依班別執行「澆水」，過度使用化學藥劑可能損害植栽。
=依班別執行澆水作業#維持植栽生命力是公區美化的重點，規範要求夥伴在清潔時一併注意水分補充。
~將植栽移至門口曬太陽#除非是特定的專案維護或主管要求，否則例行工作中僅需執行澆水與基本整理。
~每兩小時澆一次水以確保濕潤#澆水應適度，頻率應依照植栽特性與班別計畫，過量可能造成爛根或溢水污損。
~植栽維護屬於外包廠商工作，夥伴不需處理#「大廳桌上植栽澆水」被明確列入夥伴的例行清潔工作中。
}

::OP-FO-30-Q09::執行公區巡檢時，除了確認環境整潔外，還包含哪些重要的檢查項目？ {
=氣味、照明與安全狀況#優質的環境包含視覺（整潔）、嗅覺（氣味）與安全感，這些都是巡檢的關鍵要素。
~櫃台內的私人物品擺放#巡檢的核心在於「公共區域」，櫃台內私人物品並非此項作業的巡查重點。
~確認同仁的出勤打卡時間#人員考勤不屬於公區硬體維護巡檢的範疇。
~檢查客房內的備品是否補齊#本規範專指「公區維護」，客房內部屬於房務部的其他標準作業程序。
~統計路過門口的行人數量#行人數量統計與環境維護、安全檢查無關，非巡檢主要目的。
}

::OP-FO-30-Q10::若在清潔過程中導致「地面濕滑」，根據注意事項，首要動作應為何？ {
~大聲提醒路過的顧客小心滑倒#口頭提醒效果有限且可能造成干擾，應以具體的視覺警示為主。
=立即設置告示（警告牌）#即時設置物理告示能有效預防滑倒意外，是飯店公共安全的基本要求。
~停止清潔動作，等待地面自然風乾#若不設置告示，風乾期間仍有顧客受傷的風險，應先示警。
~請其他同仁站在該處引導顧客#以人力引導在人力資源分配上不具效益，標準做法是使用告示牌。
~若現場沒有顧客，可以不必設置告示#環境隨時可能有顧客或同仁經過，只要地面濕滑就必須即時設置告示。
}
