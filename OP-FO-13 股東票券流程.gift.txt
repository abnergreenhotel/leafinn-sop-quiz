$CATEGORY: OP-FO-13 股東票券流程

::OP-FO-13-Q01::當客人詢問是否能在「連續假期」期間使用股東住宿券預約訂房時，根據標準作業流程應如何處理？ {
~直接為客人建立訂單，並加收連假差額#資料明確規定連假不開放使用，並非加價即可使用。
~請客人自行聯繫發券單位確認連假資格#這並非標準處理流程，應由一線夥伴根據作業規範直接回覆。
=委婉告知國定假日及連續假期不開放使用住宿券#根據規範，國定假日與連續假期屬於不開放使用的區間，夥伴需進行委婉說明。
~視當日住房率決定是否接受訂房#流程中並未提到可視房況彈性開放，應嚴格遵守不開放的規範。
~告知可以訂房但需以原價計算，不可折抵#雖然可改以一般訂房處理，但針對「使用住宿券」的請求應直接說明規範限制。
}

::OP-FO-13-Q02::若客人預計在「假日」入住，關於房費的告知與計算方式，下列何者正確？ {
~房價以原價計算，無須告知折後金額#假日入住仍有折扣，且流程強調需主動告知折後總額。
=房價以 6 折計算，並需主動告知折後應付總額#規範要求假日入住應以 6 折計算，且為了避免爭議必須主動報價。
~房價以 5 折計算，入住時再核對金額即可#規定的折扣數為 6 折而非 5 折。
~僅需告知平日不加價，假日則依現場價格為準#這會導致報價不明確，應直接依據 6 折計算並告知。
~假日入住必須全額預付，否則無法計算折扣#雖然要處理費用，但計算標準與預付是兩件事。
}

::OP-FO-13-Q03::在票券初審階段，為避免電話接聽產生誤差，建議請客人透過什麼方式傳送票券照片以便核對？ {
~發送電子郵件至訂房組信箱#雖然可行，但非流程中建議的最方便通訊管道。
~傳真至飯店前台#傳真影像通常不清晰，不利於核對細節。
=傳送至官方 LINE 帳號#官方 LINE 是流程建議最方便、清楚且能即時核對正反面照片的管道。
~上傳至飯店官網的留言板#官網留言板通常不具備即時傳圖與對話的功能。
~透過簡訊傳送多媒體訊息 (MMS)#MMS 格式限制較多且不夠普及，並非建議方式。
}

::OP-FO-13-Q04::辦理票券初審時，下列哪一項不是「執行說明」中明確列出的核對重點？ {
~登記券碼是否正確#這是初審的基本動作，必須確認券碼。
~檢查「使用須知」規範#確保客人持有的票券符合當前使用規範。
~核對「適用館別印章」是否正確清晰#這是確認票券適用性的關鍵查核點。
=確認票券的購買來源與發票號碼#流程中僅提到核對券面資訊與印章，並未要求追溯購買來源或發票。
~確認票券正、反面照片皆已提供#流程明確要求請訂房者拍攝正反面傳送。
}

::OP-FO-13-Q05::在資料蒐集與備註階段，除了詢問交通方式，還必須詢問哪一項重要資訊？ {
~客人的用餐禁忌#雖然重要但非本流程中列出的標準查核項。
=預計抵達時間#流程明確要求詢問預計抵達時間以利安排。
~過去是否有入住紀錄#此資訊並非此流程的必要問項。
~是否需要加購早餐#流程重點在於交通與特殊硬體需求。
~隨行家屬的身分證字號#除非辦理入住，否則訂房階段無須蒐集所有家屬字號。
}

::OP-FO-13-Q06::針對假日的訂房預約，在最後確認與提醒階段，必須進行哪項與財務相關的操作？ {
~要求客人全額匯款總房費#流程提到的是「剩餘房費」且可採授權或匯款。
~無需處理費用，入住再付即可#假日有特別規定需先處理剩餘費用。
=需先授權或匯款剩餘房費#這是針對假日入住的特殊財務規定，用以確保預約。
~請客人提供信用卡卡號作為保證但不刷取#流程明確指明要「授權或匯款」，而不只是提供資料。
~收取 1000 元作為作業手續費#流程中並無手續費的規定。
}

::OP-FO-13-Q07::在「最後確認與提醒」步驟中，夥伴務必提醒客人入住當日應遵守哪項規定，否則會影響優惠核銷？ {
~必須攜帶身分證件#身分證是基本要求，但此處流程強調的是「票券」相關核銷。
~必須於 15\:00 前辦理入住#入住時間非此核銷流程的重點提醒。
=必須攜帶票券正本#這是核銷最關鍵的憑證，沒帶將影響優惠。
~必須出示訂房成功的 LINE 截圖#截圖不能取代實體票券核銷。
~必須現場填寫股東滿意度問卷#流程中並未要求此項操作。
}

::OP-FO-13-Q08::根據「入住辦理階段」的標準流程，股東票券在系統操作上有何特殊規定？ {
~必須先入全額訂金方可入住#股東票券有特殊的免入訂金規定。
=無須入訂金，直接選擇入住即可#流程中明確標註股東票券無須入訂金的操作路徑。
~系統需手動調整房價為 0 元#流程並未指示手動改價，僅說明訂金處理方式。
~必須先執行退房再重新入住#這是錯誤的系統逻辑。
~系統會自動抓取 LINE 照片核銷#系統無法自動處理 LINE 照片，仍需人工核對實體券。
}

::OP-FO-13-Q09::在處理異常狀況時，若發現收到的實體住宿券有「任何塗改痕跡」，應如何認定？ {
=視為票券瑕疵，可能影響核銷#塗改通常代表票券效力有疑慮，屬於異常處理範圍。
~請客人在塗改處簽名確認即可#規範並未允許透過簽名補強塗改後的票券效力。
~只要券碼清晰，塗改其他位置沒關係#流程中提到「任何塗改痕跡」都是異常狀況。
~要求客人支付票券工本費以換領新券#此非流程內定的異常處理方式。
~請客人聯繫原股東進行背書#這與標準異常處理流程不符。
}

::OP-FO-13-Q10::下列哪一項是「訂房預約階段」中，最後確認與提醒時「必須再次核對」的項目？ {
~生日、身分證字號、戶籍地址#這些資訊在預約階段非首要核對項目。
=入住日期、姓名、電話、備註事項#這些是確保訂單正確性最基礎且關鍵的資訊。
~購買票券的日期與金額#重點在於入住資訊而非票券購買細節。
~同行者人數與保險資料#非流程中規定的核對項目。
~前次入住的房號與滿意度#雖然可增加服務品質，但非本流程標準步驟。
}
