$CATEGORY: OP-FO-12 續住作業流程

::OP-FO-12-Q01::當住客來到櫃台表示想要辦理「續住」時，櫃台人員的第一個作業步驟應為何？ {
=確認該住客的房號#在進行任何房務或系統操作前，首要任務是準確核實住客目前所居住的房號。
~直接告知續住折扣#在確認是否有房可續住及核對身份前，不應直接討論房價優惠。
~請住客先支付房費#支付房費應在確認有空房並安排好房號之後才進行，並非第一步驟。
~詢問是否需要清消房型#清潔需求應在確認住客可以續住後，再進一步詢問其出門時間與需求。
~聯絡房務部確認房間狀況#應先確認住客房號並由櫃台系統查詢空房狀態，而非第一時間連絡房務。
}

::OP-FO-12-Q02::根據續住作業流程，關於續住客人的房間安排，應遵循下列哪項原則？ {
~強制要求客人換到較大的房型#流程中並未要求升等房型，且應盡量減少客人搬遷的不便。
=安排同一個房號入住，儘量避免換房#安排原房續住可以提升住客便利性，同時減少房務部清理退房與準備新房的工作量。
~優先安排至剛打掃完畢的新房間#作業指導書明確指出應「避免換房」，以客人的續住便利為優先。
~依據當日退房狀況隨機分配房號#隨機分配會增加住客換房的頻率，不符合作業流程中「安排同一房號」的要求。
~若有空房則必須更換至不同樓層#作業流程建議儘量不換房，除非該房型已無空房或有特殊需求。
}

::OP-FO-12-Q03::在詢問續住客人的清潔需求時，作業指導書規定最晚的續掃時間為何？ {
~中午 12 點前#中午 12 點通常是退房截止時間，但續掃服務有更彈性的時限規定。
~下午 1 點前#此時間並非標準作業指導書中所規定的續掃截止時間。
=下午 2 點前#根據流程話術，應提醒客人最晚的續掃服務提供時間是下午兩點。
~下午 3 點前#下午三點通常是入住高峰期，房務部已須全力處理入住房型，故續掃截止時間較早。
~晚上 6 點前#晚間時段房務人員配置通常較少，無法提供大規模的續掃服務。
}

::OP-FO-12-Q04::當確認客人續住後，櫃台人員應立即執行的後續動作不包括下列何者？ {
~回報房務群組住客是否需要打掃#這是標準流程第 4 步與注意事項第 1 點明確要求的必要動作。
~告知房務人員住客預計出門的時間#告知出門時間有助於房務部安排清潔排程，是必要的流程步驟。
~與客人約定支付續住房費的時間#確認續住後應與客人確認付款細節，這是確保款項入帳的標準動作。
=馬上幫客人辦理退房並重新核發房卡#續住作業應著重於「延續」而非重新辦理退房手續，應避免不必要的行政流程。
~查詢該房型是否還有剩餘空房#確認續住的可行性必須先透過查詢空房來達成，是核心作業之一。
}

::OP-FO-12-Q05::若系統查詢後發現「目前已無空房」可供客人續住，櫃台人員應如何回應？ {
~請住客自行上訂房網站確認#櫃台應代表館方直接給予回覆，而非推諉給外部平台。
=「您好，目前已無空房，非常抱歉」#這是指導書中明確規範的標準遺憾話術，簡潔且具禮貌。
~請住客至其他分館詢問有無房間#在未確認其他館別狀態前，不應隨意給予住客不確定的承諾或指引。
~建議住客在大廳等候是否有其他人退房#這不是專業的客務處理方式，且會造成住客困擾。
~直接忽略客人的請求，處理下一位住客#這違反基本服務禮儀，且未完成該作業流程的回應動作。
}

::OP-FO-12-Q06::關於續住作業中的房費支付規定，下列哪一項敘述最符合流程指引？ {
~必須在辦理續住的當下立即結清款項#流程指引建議可以與客人「約定時間」，在該時間前至櫃台支付即可。
=可與客人約定時間，在該時間前至櫃台支付#這是流程中賦予櫃台人員的執行說明，增加了收款作業的彈性。
~一律等到最終退房那天再合併結算#為了降低財務風險，應在續住當日或約定時間內收取費用，而非全數延後。
~續住費用僅能透過線上轉帳支付#流程提到應請客人「至櫃台」支付，並未限制支付方式。
~續住一律不收費，視為贈送#續住屬於營收項目，雖然可給折扣，但仍需收取房費。
}

::OP-FO-12-Q07::在續住作業中，關於「告知房務部」的注意事項，下列何者正確？ {
~只需告知房號，不需告知清潔需求#清潔需求與出門時間是房務部安排人力的重要依據，必須告知。
~等到客人真的出門後再聯絡房務#注意事項第一點強調「確認續住後須馬上告知房務」，以便提前規劃。
=確認續住後須馬上告知房務，並說明清潔需求與出門時間#這是注意事項的核心，能確保櫃台與房務部資訊同步，確保服務品質。
~如果客人不需要打掃，就不必告知房務部#即使不需要打掃，房務部也需要知道該房為續住房，以免誤闖或誤報退房。
~僅需在交班本上登記，不需要傳到群組#執行說明中提到應「回報房務群組」，以確保即時性。
}

::OP-FO-12-Q08::當客人詢問續住價格時，根據作業指導書，櫃台人員應如何處理？ {
~告知續住一律恢復原價#注意事項第二點提到「續住可給房價折扣」，這是有助於提升續住率的誘因。
=告知續住可給予房價折扣#這是公司為了鼓勵客人延長住宿所提供的優惠政策。
~告知續住需加收 20% 的服務費#流程中並未提到加價規定，而是提到可以給予折扣。
~請客人自己上網比價，以網路價為準#現場人員應具備處理續住價格的權限或依據流程給予折扣。
~告知續住價格必須比第一晚入住時更高#這不符合流程中鼓勵續住並提供折扣的經營方向。
}

::OP-FO-12-Q09::情境：住客陳先生表示要續住，櫃台查詢後確認有空房，且陳先生表示他要先去吃早餐，大約下午 1 點才會出門，需要打掃。下列何種回報方式最正確？ {
~回報房務群組：房號 XXX 續住，不需打掃#這與客人的需求（需要打掃）不符，會造成客訴。
=回報房務群組：房號 XXX 續住，下午 1 點後清潔#完整包含房號、續住狀態、出門時間及清潔需求，符合標準作業程序。
~等下午 1 點陳先生真的出門後再說#這違反了「確認續住後須馬上告知房務」的原則。
~告知陳先生 1 點太晚了，房務沒辦法掃#最晚續掃時間是下午 2 點，下午 1 點仍在可服務的時間範圍內。
~僅告知房務部有續住，不提清潔時間#未告知清潔時間會導致房務人員無法有效排程，可能發生與客人在房內碰撞的情況。
}
