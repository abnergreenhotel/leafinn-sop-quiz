$CATEGORY: OP-FO-03 電話禮儀規範

::OP-FO-03-Q01::當外部廠商撥打電話指名要找執行長或總經理時，櫃台夥伴的首要任務是什麼？ {
=協助過濾是否為廣告或房仲等非重要訊息#針對高階主管的來電，規範中特別要求需過濾無關業務的廣告資訊，以避免干擾其工作。
~立即將電話轉接到執行長辦公室#未經確認就直接轉接可能會讓主管接到不必要的廣告或推銷電話。
~直接告知廠商主管正在開會，請其自行掛斷#這是不專業的回應方式，應根據規範詢問並記錄留言資訊或進行過濾。
~向廠商索取主管的個人手機號碼以核對身份#櫃台不應反向要求廠商提供主管個資，應是由廠商主動說明來意。
~將電話轉接至人事部門進行初步審核#規範中並未提到轉接至人事部，過濾資訊是櫃台人員在電話接聽時應執行的職責。
}

::OP-FO-03-Q02::外客來電詢問：「我的朋友住在那裡，請問他在房間嗎？」根據電話禮儀規範，櫃台應如何反應？ {
=秉持隱私原則，不主動證實「房客是否存在」#為了保障住客隱私，不論房客是否在場，都不應向外人確認其居住事實。
~直接告知房客的房號，並詢問是否需要轉接#規範中嚴禁主動提供房號，這會嚴重侵犯住客隱私。
~查閱住房名單後，若確有此人則告知其目前在房內#即便房客在房內，也不應直接證實房客的存在，應依照規範流程先行核對身份並嘗試聯繫。
~請對方提供房客的房間電話，若對不上則掛斷#此做法過於生硬且不具服務品質，應引導對方留下留言。
~告知對方房客剛好外出，請晚點再撥#告知「房客外出」等於間接證實了房客確實住在該處，違反了隱私保護規範。
}

::OP-FO-03-Q03::在記錄廠商欲轉達給主管的留言資訊時，下列哪一項資訊是相較於訪客留言「額外」需要記錄的？ {
=公司行號#對於廠商留言，規範明確要求記錄其所屬公司，以便主管判斷業務相關性。
~留言者的姓名#無論是廠商還是訪客留言，姓名都是最基本的必備資訊。
~聯絡手機#聯絡方式是所有留言回覆的基礎，不分廠商或訪客皆需記錄。
~留言時間#記錄留言當下的時間是確保資訊時效性的標準作業步驟。
~轉達內容#了解具體事項是留言的核心目的，不分對象皆為必填項目。
}

::OP-FO-03-Q04::當對象要找的人正在接聽另一線電話無法切換時，在作業流程中屬於哪一種適用情況？ {
=業務忙碌中#規範定義「當事人正在接聽另一線電話，無法立即切換」屬於業務忙碌的情境。
~暫時離位#暫時離位通常指開會、用餐或處理現場事務等不在座位上的狀態。
~非當班時間#非當班時間是指同事尚未上班、已下班或休假。
~處理現場緊急事務#這屬於暫時離位的一種具體原因，與正在接聽另一線電話的情形不同。
~系統故障#規範中並未將系統問題列入此項作業流程的常見適用情況中。
}

::OP-FO-03-Q05::若外客表示無法聯繫上房客 XXX，請求櫃台幫忙，櫃台標準的話術第一步應是什麼？ {
=「您好，為您確認一下，請問您要找的房客全名怎麼稱呼？另外也請教您貴姓呢？」#作業流程的第一步是「確認房客/訪客身份」，以確保後續聯繫的準確性。
~「好的，他現在就在房間裡，請稍候我為您轉接。」#這直接證實了房客在場，違反了不主動證實房客存在與隱私保護的規定。
~「請問您知道他的房號嗎？如果不知道的話我無法幫您查詢。」#櫃台應主動提供協助並引導流程，而非以不知道房號為由拒絕服務。
~「他現在不在，您要不要直接留下手機號碼，我請他回電？」#尚未嘗試聯繫房客前，不應直接判定其不在，且應先核對雙方身份。
~「請問您找他有什麼急事？我們這裡不方便隨便轉接電話。」#話術應保持禮貌並依程序確認身分，而非預設拒絕的立場。
}

::OP-FO-03-Q06::在掛斷電話前，為了確保資訊傳達正確，櫃台夥伴應該執行哪一個動作？ {
=將所記錄下的資訊複誦一遍進行再確認#複誦資訊能有效避免記錄錯誤，確保轉達給留言對象的內容精確無誤。
~詢問對方是否滿意本次的通話服務#雖然禮貌很重要，但規範強調的核心是資訊傳達的準確性而非滿意度調查。
~告知對方目前櫃台非常忙碌，可能會延遲轉達#這會給人專業度不足的印象，應承諾會如實傳達而非找藉口預留延遲空間。
~要求對方在線上等待，直到聯繫上對象為止#若對象目前不便接聽，應記錄留言請其回電，而非讓來電者無止盡等待。
~提醒對方下次可以直接撥打分機，不需經過櫃台#規範中並未要求此動作，且有時基於安全過慮，仍需經過櫃台處理。
}

::OP-FO-03-Q07::當來電對象確定不在位子上，櫃台提議留言的話術應該包含下列哪些細節？ {
=聯繫電話、具體轉達事項、建議的回電時間#規範中明確建議詢問聯絡電話、具體事項及建議的回電時間，以提升後續溝通效率。
~對方的住家地址、身份證字號、出生年月日#這些資訊屬於過度蒐集，與轉達留言的必要性無關。
~對方的電子郵件地址與社交軟體帳號#電話回電仍以電話號碼為主，除非有特殊需求，否則不屬於標準詢問內容。
~主管或房客目前的具體行蹤（如在餐廳用餐）#櫃台應保護當事人隱私，不應向外人透露具體的行程細節。
~詢問對方是否願意改打其他同事的電話#這不符合專業的留言轉達流程，應先以記錄留言給原對象為主。
}

::OP-FO-03-Q08::如果櫃台人員不確定某通廠商電話是否為廣告，應採取什麼行動？ {
=詢問主管進行確認#規範指出，若不確定是否為需要過濾的資訊，應請示主管確認。
~為了保險起見，一律視為廣告拒絕#這可能會漏掉重要的業務訊息，造成公司損失。
~直接將電話轉給執行長讓其自行判斷#這失去了櫃台過濾資訊的功能，會增加高階主管負擔。
~請廠商先寄送電子郵件，之後再打來#雖然這是一種過濾方式，但規範指引的首選做法是詢問主管確認。
~將該廠商列入黑名單，以後不再接聽#在未經確認前就封鎖廠商是不專業的行為。
}

::OP-FO-03-Q09::對於「非當班時間」的定義，下列哪一項敘述「不符合」規範中的描述？ {
=該名同事正在處理現場緊急事務#「處理現場緊急事務」在規範中被分類為「暫時離位」而非「非當班時間」。
~該名同事尚未上班#尚未到班屬於非當班時間的範疇。
~該名同事已經下班#已經離開工作崗位結束當日勤務屬於非當班時間。
~該名同事正在休假中#排定休假的人員顯然不在當班狀態。
~撥打電話時，該名同事並未排定班表#未排班等同於未上班或休假，皆屬於非當班時間。
}

::OP-FO-03-Q10::在「轉達留言 (廠商找主管)」的執行說明中，哪一項是關於廣告資訊處理的重點？ {
=過濾資訊是否為廣告#這是執行說明中的第二點，強調需篩選無效資訊以維護主管辦公效率。
~向廣告商收取轉達服務費#這不符合飯店或辦公室運作的常規，且規範中無此要求。
~記錄廣告商的統一編號#既然是廣告資訊，通常直接過濾即可，不需詳加記錄其統編。
~將廣告訊息彙整後每週呈報一次#規範中並無此要求，廣告訊息通常不需要通知主管。
~主動向廣告商推薦其他房客作為客源#這嚴重違反住客隱私且與職責背道而馳。
}
