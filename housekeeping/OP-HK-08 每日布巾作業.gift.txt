$CATEGORY: OP-HK-08 每日布巾作業

::OP-HK-08-Q01::當你在核對「洗旺紅單」時，發現送來的大毛巾每綑數量不一。根據作業標準，每一綑大毛巾應該包含幾條？ {
=10 條#大毛巾與足布的標準綑紮數量均為 10 條，以便於快速點收與記錄。
~20 條#這是小毛巾一綑的標準數量，而非大毛巾的配置。
~50 條#50 條是枕套的標準綑紮數量，不適用於大毛巾。
~15 條#此非 SOP 中定義的任何布巾類別綑紮標準數值。
}

::OP-HK-08-Q02::小明正在準備當日的房務作業，他發現倉庫內的「擦毛」僅剩下 15 條。此時小明最正確的後續處理方式為何？ {
=立即回報洗旺群組並提醒廠商多送#當擦毛低於 20 條的預警水位時，應及時聯繫廠商補貨以防作業短缺。
~先將就使用，等下午全部用完再回報#延遲回報可能導致後續夥伴無物可用，不符合預警回報機制。
~直接修改工作表上的需求量#修改需求量無法解決實體備品不足的問題，且會導致數據統計錯誤。
~向其他樓層夥伴借調，不需報備#非正式的借調會導致各樓層剩餘量統計失準，且未解決總量不足的問題。
}

::OP-HK-08-Q03::在整理客房時，你發現一條乾淨的被套上有明顯未洗淨的殘留污漬。請問該如何處置這條被套？ {
=在汙點處打結並放置於 6 樓布巾間的回洗區#在汙漬處打結是為了標記位置方便廠商重新處理，並應集中於指定的回洗區。
~直接丟入一般的回收髒毛桶中#未打結標記會使洗衣廠難以辨識特定汙點，可能導致再次洗不乾淨。
~放在當層樓的備品車下層等待收走#回洗布巾應專門放置於 6 樓指定區域，避免與當日髒毛混淆或遺失。
~自行先用清水搓洗後再繼續使用#現場夥伴不應自行處理洗滌，且可能造成布巾無法完全乾燥而產生異味。
}

::OP-HK-08-Q04::若你在更換枕套時，發現枕套邊緣有嚴重破損缺口。關於此破損布巾的處理，下列敘述何者正確？ {
=先回報群組讓廠商知悉，再放至報廢區等待評估#破損布巾需先回報以便向廠商反應，並由主管評估後進入報廢程序。
~將破損枕套計入當日回收髒毛總量中#SOP 明確規定破損品不應列入當日髒毛數量統計。
~直接打結後丟入 6 樓回洗區#破損並非汙漬，回洗無法修復，應走報廢流程而非回洗流程。
~為了節省成本，反摺後繼續給客人使用#提供破損布巾會嚴重影響服務品質與飯店形象。
}

::OP-HK-08-Q05::當日工作結束後，房務夥伴必須將記錄好的工作表交到哪裡，以便辦事員進行後續登記？ {
=2 樓員休室桌上#這是統一收集工作紀錄的指定地點，確保數據能被準確統計。
~6 樓布巾間入口處#雖然 6 樓是布巾存放區，但非辦事員收集行政表單的指定位置。
~直接交給當班的主管簽收#主管僅負責簽名確認，實際登記流程需將表單放置於指定的 2 樓位置。
~放在備品車上隨車交班#表單放置於車上容易遺失或被遺忘，且無法讓辦事員即時統計。
}

::OP-HK-08-Q06::關於工作車（備品車與布巾桶）在走道定位的標準，下列哪一項描述符合安全與美觀要求？ {
=切齊門框定位，備品車在前、布巾車在後#一字型排開並切齊門框可維持動線暢通，且特定的前後順序符合工作邏輯。
~為了方便拿取，將車子橫向停放在電梯口#橫向停放會阻礙電梯出入動線，造成安全隱憂。
~布巾車與備品車左右並排靠牆#並排會占用過多通道寬度，不符合一字型排開的規範。
~布巾桶放在前面，以便隨時丟入髒毛#SOP 規定備品車在前、布巾車在後，這與樓梯上來的動線方向有關。
}

::OP-HK-08-Q07::在填寫「每日布巾作業表」時，表單下方的「回洗布巾總量」應該如何填寫才正確？ {
=僅填寫發現有污漬而打結回洗的數量#回洗量專指因汙漬需重複洗滌的品項，與破損報廢品應區分開來。
~包含當日所有回收的髒毛數量#髒毛數量應填寫在另一欄位，回洗是指「乾淨卻有汙點」需重新處理的部分。
~將汙損品與破損品全部加總填入#破損品不列入當日髒毛或回洗統計，應另外回報。
~如果當天沒有破損，就填寫 0#回洗與破損是不同的概念，此選項混淆了兩者的定義。
}

::OP-HK-08-Q08::小華從樓梯上來右轉準備開始作業，他應該如何將工作車定位？ {
=將車輛定位於樓梯與電梯中間通道，備品車在前#這符合 SOP 定位方向規範，確保工作人員推進時的便利性與走道秩序。
~將車子推進客房內以避免阻礙走道#工作車體積大，進入客房會造成空間侷促且可能損壞房內家具。
~將布巾車停在電梯旁，備品車停在房門口#車輛應集中定位管理，分散停放會造成管理困難且顯得凌亂。
~隨意停放，只要不擋到房門即可#缺乏統一停放標準會影響飯店專業形象及緊急疏散安全。
}
