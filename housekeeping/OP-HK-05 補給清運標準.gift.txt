$CATEGORY: OP-HK-05 補給清運標準

::OP-HK-05-Q01::在開始當班作業前進行備品盤點時，若發現茶包的外包裝有些許汙損，但內容物看起來完整無缺，應如何處理？ {
~擦拭外包裝後繼續放上備品車提供給客人#即使內容物完好，外包裝髒污的備品仍會影響客人對衛生品質的信任，不應提供。
~將該批汙損備品放置於備品車最底層備用#放置於底層仍可能被誤用，且 SOP 明確規定汙損備品不可提供給顧客使用。
=將汙損的備品挑出，不提供給顧客使用#根據標準，任何破損、受潮、過期或外包裝髒汙的備品皆屬於不合格品，不得提供給顧客。
~先放進客房，待客人反應後再行更換#這會導致客訴並降低服務品質，應在準備階段就排除不合格的物品。
}

::OP-HK-05-Q02::在進行備品車補給作業時，為了兼顧推行安全與物品保護，下列哪一種擺放方式最正確？ {
~將較重的清潔劑瓶裝物放在備品車上層以便取用#重物放在上層會導致車體重心不穩，增加推行時傾倒的風險。
~將容易受壓損的衛生紙、棉花棒放在最底層以節省空間#輕物或易壓損物品應放在上層，避免被下方物品擠壓變形。
=將重物放置於下層，輕物與易壓損物品放置於上層#下重上輕的原則能確保備品車重心穩定，並保護易損備品的完整性。
~為了效率，將所有清潔劑與客房備品混合擺放在同一格#清潔劑應與客用品分開，避免化學氣味殘留或發生滲漏造成交叉污染。
}

::OP-HK-05-Q03::房務員在走廊補給客房備品時，關於備品車的位置安排，下列何者是必須遵守的安全規定？ {
~為了作業方便，備品車應盡可能貼近客房門口#雖然貼近門口方便取物，但若阻擋了客人的行進動線或逃生空間則不符合安全規範。
~只要不擋到走廊中央，可以暫停在消防栓前方#消防設備前方必須保持淨空，絕對不可被任何物品阻擋，以因應緊急狀況。
=備品車不得阻擋逃生門、消防設備與客人動線#確保安全通道與應變設備的可及性是旅館安全管理的最高原則。
~備品車只要靠牆停放，即使擋到電梯口也沒關係#電梯口是重要的通行區域，阻擋此處會嚴重干擾客人進出與逃生安全。
}

::OP-HK-05-Q04::在樓層備品間進行補貨作業時，為了避免備品因存放過久而過期或受潮，應遵循哪項原則？ {
~將新領回的備品直接堆疊在舊有庫存的最前方#這會導致舊品被推到後方長期無法使用，增加過期或變質的風險。
=採用先進先出原則，將舊品放在前方，新品放在後方#先進先出能確保舊的備品優先被取用，維持備品的新鮮度與有效性。
~依據外箱的大小進行排列，大的紙箱放在最外面#僅依大小排列無法達成效期管理，可能導致物料浪費。
~新舊品混合堆放，只要標示清楚數量即可#混合堆放會造成取貨時的混亂，難以落實庫存效期管理。
}

::OP-HK-05-Q05::房務員在整房清運垃圾時，發現客房垃圾袋內裝有破碎的玻璃杯，應如何正確處理？ {
~直接將垃圾袋封口，並儘速送往垃圾集中區#直接封口無法防止碎玻璃刺破袋子，可能造成運送者或後續處理人員受傷。
~將玻璃碎片與其他垃圾混在一起，增加緩衝力#混放會使分類與後續處理變得危險，且無法有效防止穿刺。
=將玻璃碎片分開包覆並加上明確標示#針對尖銳物應採取獨立包裝與警示標示，以維護搬運與清理過程的人員安全。
~先將垃圾袋暫放在客房外走廊，等下班前再統一處理#垃圾不得暫放在走廊，會影響走廊觀瞻並可能發生滲漏或散發異味。
}

::OP-HK-05-Q06::關於客房垃圾收集的標準作業，當垃圾袋達到多少容量時應進行更換以防搬運過程中破裂？ {
~垃圾袋全滿且稍微擠壓至實滿#全滿的垃圾袋在封口與搬運時極易因受壓而破裂，增加清理負擔。
=垃圾袋裝滿約八分滿時#維持八分滿能留出適當的封口空間，並降低因內容物過重或尖銳擠壓導致的破裂風險。
~只要還沒溢出來，都可以繼續裝填#這是一種不負責的做法，極可能導致異味散發或垃圾掉落。
~每小時固定更換一次，不論垃圾量多寡#這不符合作業效率，且 SOP 是以容量（八分滿）作為更換標準。
}

::OP-HK-05-Q07::在清運髒布巾的過程中，為了確保客人的財務安全與設備完整，哪一個步驟最重要？ {
=清運前必須翻查布巾內是否夾帶遺留物或房卡#徹底檢查能避免客人的遺留物（如房卡、遙控器、飾品）隨布巾被送往洗滌，降低損失。
~為了節省時間，直接將整床布巾捲起放入袋中#未經檢查直接捲起，極易導致遺留物遺失或損壞。
~將所有髒布巾（含特殊污染）全部混合裝袋運送#特殊污染布巾需獨立處理，混放會導致其他布巾交叉污染。
~若布巾太重，可以先在走廊地面拖行一小段距離#髒布巾嚴禁拖地，這會導致布巾更加髒污或受損。
}

::OP-HK-05-Q08::房務員若在整理客房時發現有沾染大量血跡或排泄物的「特殊污染布巾」，應如何分類處理？ {
~與一般回洗布巾混合裝袋，讓洗衣廠統一處理#混合處理會導致其他乾淨程度尚可的布巾被污染，影響洗滌效果。
~自行用客房清潔劑刷洗乾淨後再送洗#這不屬於房務員的補給清運作業範疇，應依照規範進行分類標示。
=將特殊污染布巾獨立裝袋並標示清楚#特殊污染物品需要特定的消毒或強力洗滌流程，獨立標示才能確保洗滌廠正確處理。
~將污染布巾與擦毛（擦拭巾）混放在同一個袋子#擦毛與回洗布巾的性質不同，混放會造成洗滌分類的困難。
}

::OP-HK-05-Q09::完成垃圾集中區（子母車區）的清運作業後，下列哪一項不是確認清運完成的標準？ {
~子母車的桶蓋應處於閉合狀態#桶蓋閉合是標準要求之一，可避免異味散發與蚊蟲滋生。
~集中區地面應乾淨無散落垃圾#地面整潔是確認清運品質的重要指標。
~現場應無明顯的異味散出#無異味代表清運徹底且地面滲漏物已被清除。
=將清運用的推車留在子母車旁以便下一位同事使用#這是不正確的做法。SOP 規定清運工具使用後需清潔歸位，不可隨意遺留。
}

::OP-HK-05-Q10::當你在當班期間發現某樓層的牙刷耗損量異常的高，遠超預期數量時，應如何執行補給紀錄與交接？ {
~直接補足數量即可，不用特別記錄以免麻煩#這無法解決根本問題，可能忽略了客房管理或人為浪費的潛在原因。
~只需在群組簡單提到「東西少很多」#紀錄應包含日期、樓層、品項、數量及原因，不完整的資訊無法提供後續分析。
=紀錄異常消耗並追蹤原因，回報至群組#落實紀錄與回報能幫助主管追蹤成本異動，並確認是否有其他作業問題需要調整。
~向下一班口頭提醒多補一點就好#僅口頭交接容易遺忘且無記錄可查，不符合標準的交接程序。
}
