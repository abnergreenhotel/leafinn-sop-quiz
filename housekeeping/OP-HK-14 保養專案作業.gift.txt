$CATEGORY: OP-HK-14 保養專案作業

::OP-HK-14-Q01::在進行週保養時，針對木質房門框與浴室內的白鐵把手，應使用哪一種藥劑進行擦拭保養？ {
=必麗澤#根據 SOP 規範，木質與白鐵材質的表面（如門框、開關、把手等）應統一使用必麗澤進行細部除塵保養。
~穩潔#穩潔主要用於外窗與玻璃材質，對於木頭牆面或白鐵把手的保養效果並非 SOP 建議的標準。
~漂白水#漂白水具腐蝕性，通常用於浴室牆面與地板除垢，不適合用於日常木質家具或門框的擦拭。
~檸檬酸#檸檬酸主要用於半月保養中的快煮壺與杯子浸泡除垢，而非用於表面家具的擦拭。
}

::OP-HK-14-Q02::夥伴在執行「半月保養」的快煮壺與杯子清潔時，下列哪項操作最符合品質標準？ {
=集中於浴缸內用檸檬酸浸泡半小時，沖洗後確保無水漬#SOP 規定快煮壺與杯具需用檸檬酸浸泡約 30 分鐘，且完成後不可殘留水漬以維持衛生美觀。
~使用必麗澤噴灑壺身內外，再用乾布擦乾#必麗澤屬於表面保養劑，不可用於接觸飲用水的快煮壺內部清潔。
~為了徹底除菌，應加入漂白水與清潔劑混合浸泡#清潔劑不可混用，且快煮壺除垢應優先使用檸檬酸，使用漂白水可能造成殘留風險。
~浸泡後直接自然風乾，無需刻意擦拭#SOP 特別強調「不可有水漬」，自然風乾容易產生水痕，不符品質規範。
}

::OP-HK-14-Q03::進行「每月浴室重保養」使用漂白水刷洗時，為了避免旅客不適與確保現場安全，下列哪一項做法是錯誤的？ {
=為了讓空氣流通，應同時開啟客房空調運作#SOP 明確指出應關閉冷氣以避免整個空調系統充滿漂白水味，並需全程開窗通風。
~保養完成後需在浴室噴灑除臭劑#這是正確的做法，SOP 要求保養完畢需噴除臭劑以掩蓋藥劑味道，提升旅客舒適度。
~作業過程中必須全程打開窗戶保持通風#這是 SOP 規定的安全步驟，旨在減少化學藥劑氣味在室內的濃度。
~針對牆面 Silicone 與地板細縫進行全面除垢#這是月保養的重點工作之一，目的是確保浴室各處細節都經過深度清潔。
}

::OP-HK-14-Q04::在進行每月「更換給皂劑」的保養作業時，下列哪項做法最能兼顧衛生與成本控制？ {
=將舊液倒掉並清洗乾淨後，再補充半瓶左右的新品#SOP 規定需清洗瓶身後補充，且預計保養前一週以半瓶為主，可避免更換時造成過多浪費。
~直接在剩餘的皂液上補充新品，確保瓶身全滿#SOP 要求需先倒掉並清洗乾淨，直接添加可能導致細菌滋生或新舊液混雜。
~僅在給皂劑完全耗盡時才進行清洗與更換#這是月保養的固定項目，旨在維護品質，而非等耗盡才處理。
~為了清潔方便，直接將給皂機外殼拆下送洗#SOP 強調的是內部殘液的處理與瓶口清潔，並非大範圍拆卸外部硬體。
}

::OP-HK-14-Q05::針對客房床墊的「季保養（翻床）」，現場夥伴應如何正確執行？ {
=將床墊進行方向對調（轉向），但不可翻面#SOP 備註欄明確提醒「床只能轉向對調，不能翻面」，以符合床墊設計的正確維護方式。
~為了讓兩面磨損均勻，每季應將床墊翻面使用#這違反了 SOP 備註中「不能翻面」的警示，可能損壞特定結構的床墊。
~翻床時若發現保潔墊乾淨，可不送洗以節省工時#SOP 規定季保養包含更換保潔墊並送洗，屬於固定保養流程。
~翻床作業應在每年的 2 月與 8 月進行#SOP 規定的季保養月份為 1、4、7、10 月。
}

::OP-HK-14-Q06::週保養中關於冷氣與通風設備的清潔，下列操作何者正確？ {
=清洗冷氣迴風網並風乾，抽風機內部則需用牙刷刷淨#根據 SOP，冷氣迴風網需清洗風乾，而抽風機外蓋拆下後，內部需配合牙刷進行細部清潔。
~冷氣出風口處灰塵較多，應噴灑大量清潔劑後擦乾#SOP 提醒藥劑應適量，避免過多，且出風口處主要使用必麗澤進行保養。
~抽風機只需擦拭外蓋，不可拆卸以避免損壞#SOP 指示抽風機外蓋應拆下清洗，才能確保內部清潔度。
~LED 投射燈應使用濕抹布直接擦拭燈珠#SOP 指出 LED 投射燈應翻轉後，利用油漆刷進行除塵，而非濕擦。
}

::OP-HK-14-Q07::在 1 月份的保養計畫中，除了季保養外，還包含了哪一項獨有的「年度保養」項目？ {
=浴室維修孔區域的天花板除塵#根據年度保養表單，1 月份需執行天花板除塵，特別是針對浴室維修孔改區域。
~更換客房內的滅火器藥粉#SOP 僅提到滅火器外觀使用必麗澤保養，並未提及內部藥粉更換流程。
~全面清洗客房窗簾與床裙#床裙與窗簾屬於「半年保養」項目，分別於 2 月與 9 月執行。
~更換天花板的所有冷氣口濾網#濾網清潔屬於「週保養」的例行範疇，並非年度保養才處理。
}

::OP-HK-14-Q08::夥伴在執行半年保養時，使用烘被機處理枕頭與棉被，其主要目的是什麼？ {
=透過高溫殺滅塵蟎與減少霉味的產生#SOP 註明烘枕烘被半小時後換面，是為了減少因潮濕而產生的霉味。
~單純加速寢具烘乾，方便立即恢復使用#雖然烘乾可能有此效果，但 SOP 內明確提到的作業目標是「減少霉味產生」。
~取代定期洗滌枕頭的流程#烘乾是為了保養防霉，不應視為完全取代清洗的替代方案。
~增加枕頭的蓬鬆度與外觀美感#雖然蓬鬆度會提升，但 SOP 核心強調的是氣味與濕度管理的保養效益。
}
