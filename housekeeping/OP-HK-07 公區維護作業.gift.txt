$CATEGORY: OP-HK-07 公區維護作業

::OP-HK-07-Q01::夥伴在 8F 洗衣間執行維護作業時，除了表面擦拭和清理垃圾，針對烘衣機的後續操作，哪一項最符合 SOP 的設備維護標準？ {
=清理烘衣機的棉絮殘留並檢查是否有客人遺留物#清理棉絮是確保烘衣機效能與防火安全的關鍵步驟，同時檢查遺留物能提升服務品質。
~僅需擦拭烘衣機外殼，內部滾筒每週清潔一次即可#忽視棉絮清理會導致散熱不良，並非僅靠外觀擦拭就能達到維護標準。
~將烘衣機門開啟通風，不需要特別檢查濾網#通風雖好，但 SOP 明確要求處理棉絮，若不清理濾網會影響下一位客人的使用體驗。
~在烘衣機內噴灑芳香噴霧，維持空間氣味#SOP 未提及在烘衣機內噴灑化學物質，且棉絮與遺留物的清潔才是重點任務。
}

::OP-HK-07-Q02::當你在清潔 1F 公共廁所時，除了刷洗馬桶與洗手檯，根據指導書，哪些耗材是必須主動巡檢並視情況補充的？ {
=廁所捲筒衛生紙與香香噴瓶#這是維持公廁基本機能與空間舒適度的必要項目，也是 SOP 中明確註記的注意事項。
~洗手乳與拋棄式擦手紙#雖然這些是常見耗材，但 SOP 針對 1 樓公廁特別強調的是捲紙與香香噴瓶。
~拋棄式馬桶座墊紙與垃圾袋#垃圾袋屬於垃圾桶清潔的一部分，並非 SOP 特別標示需要「留意補充」的亮點項目。
~洗髮精與沐浴乳備品#公共廁所通常不提供洗澡備品，這屬於客房維護範疇，不符合公廁作業標準。
}

::OP-HK-07-Q03::在處理電梯清潔時，針對不同的材質表面，下列哪一種清潔劑的使用方式是正確的？ {
=鏡面使用穩潔，門片與按鍵等其餘部位使用必麗澤#針對材質選用正確的清潔劑（穩潔去油汙、必麗澤保養金屬/塑膠面）能確保清潔效果且不傷表面。
~所有部位皆統一使用穩潔，以求清潔效率#穩潔主要針對玻璃，對電梯門片或其他金屬部件可能無法達到良好的保養效果。
~電梯門片使用穩潔，鏡面使用必麗澤#這是錯誤的配對，必麗澤使用在鏡面上會留下油膜，導致視線模糊。
~按鍵部分使用清水擦拭，禁止使用必麗澤#SOP 指示除了鏡面以外的部分皆使用必麗澤，包含按鍵與門片。
}

::OP-HK-07-Q04::在客人退房尖峰時段結束後，針對電梯內容易堆積髒汙的微小細節，應如何依照 SOP 進行加強清潔？ {
=使用吸塵器將門下方的溝槽及腳踏墊吸乾淨#電梯溝槽極易積塵，吸塵能確保電梯門開合順暢並維持地面整潔。
~用濕抹布反覆擦拭門下方軌道即可#濕抹布可能將灰塵推入縫隙深處，SOP 明確要求應使用吸塵器吸淨。
~噴灑除臭劑在腳踏墊上，蓋過灰塵氣味#除臭劑無法解決實體髒污，且腳踏墊積塵會影響公區觀感。
~只需在每週五固定進行一次深度吸塵#SOP 指出在客人退房後應進行此項清潔，並非僅每週一次。
}

::OP-HK-07-Q05::房務夥伴在維護各樓層飲水機時，發現接水處有泡麵渣或茶葉，根據作業流程，應執行的關鍵動作為何？ {
=拿起集水盤清潔，並將出水口刷洗乾淨#出水口與集水盤是衛生死角，刷洗出水口能避免殘渣滋生細菌或影響水質衛生。
~僅需將集水盤內的積水倒掉，擦乾外觀即可#只倒水無法清除殘渣，且 SOP 要求必須特別針對出水口進行刷洗。
~在飲水機旁張貼告示請客人配合，不執行刷洗#告示是預防手段，但維護作業仍應包含實體刷洗動作以維持現狀整潔。
~更換整個集水盤與出水口濾網#除非設備損壞，否則標準作業是以「清潔」與「刷洗」為主，並非更換耗材。
}

::OP-HK-07-Q06::針對各樓層的天井欄杆，為了避免灰塵或生物留下的痕跡影響美觀，清潔時應特別注意哪個位置？ {
=欄杆下方是否有蜘蛛絲殘留#欄杆下方是視覺容易忽略的角落，清理蜘蛛絲能確保環境看起來精緻且無死角。
~欄杆連接地面的螺絲孔位#雖然也是細節，但 SOP 中特別點出需要注意的地方是「下方」的「蜘蛛絲」。
~欄杆最高處的手扶面#手扶面雖需清潔，但並非 SOP 備註中需要「特別注意」的特殊異常狀況。
~欄杆間隙的垂直面#此處為一般清潔區域，並非文件強調的隱蔽性清潔重點。
}

::OP-HK-07-Q07::若目前的房況非常忙碌，而今天是排定執行「腳踏墊清潔」的月份（如 3 月）的第一週星期四，夥伴應如何依規定彈性調整？ {
=可依房況彈性提前或延後一天執行#非例行性清潔任務允許依據現場房況進行前後一天的微調，以兼顧服務品質與維護進度。
~必須在當天完成，否則視為作業疏失#SOP 已賦予執行者彈性，強制當天完成可能會造成人力過度負荷或影響其他急迫任務。
~自動取消該季度的清潔，待下個季度再處理#腳踏墊清潔為每季一次的任務，取消會導致設備污損加劇，不符合維護邏輯。
~將該任務直接外包給外部清潔公司#SOP 中並未提及可因房況忙碌而直接轉由外包處理，夥伴應自行調整時間。
}
