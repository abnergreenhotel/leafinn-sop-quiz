$CATEGORY: OP-HK-04 客房除塵清潔

::OP-HK-04-Q01::房務員在整理客房提籃內的備品時，關於紙拖鞋與鞋拔的擺放，下列哪一項做法符合正確的視覺標準？ {
=紙拖鞋字面朝下並置中擺放，鞋拔側邊靠向保險箱並切齊邊緣#這種擺放方式能確保備品外觀整齊一致，並透過與保險箱切齊的對位方式提供標準化的視覺感受。
~紙拖鞋字面朝上方便客人辨識，鞋拔隨意放在提籃內即可#字面朝上不符合文件規定的隱藏字面標準，且鞋拔隨意擺放會顯得雜亂，降低客房質感。
~紙拖鞋需靠左擺放，鞋拔則應靠向提籃的最右側以區隔空間#標準要求拖鞋應置中，且鞋拔應有明確的參考基準點（保險箱）而非隨意靠側。
~紙拖鞋字面朝下，鞋拔則需放在兩雙拖鞋之間以示區隔#將鞋拔放在拖鞋之間並非標準做法，會破壞拖鞋置中擺放的平衡感。
}

::OP-HK-04-Q02::在整理客房的 Mini Bar 時，即使發現杯具（玻璃杯、馬克杯）看似乾淨且客人未曾使用，房務員仍應執行哪項動作？ {
~僅需確認杯蓋是否有蓋好，若有蓋好則不需重複檢查#僅檢查杯蓋無法確保杯身本身的清潔度，環境微塵或前次留下的指紋仍可能存在。
=仍需檢查是否有指紋、棉絮或灰塵，並使用杯子專用擦毛擦拭#細節檢查能確保每一位入住客人接觸到的容器都是無瑕疵的，且專用抹布能避免交叉感染。
~為了節省作業時間，只要杯具擺放位置沒變動即可跳過#跳過檢查可能導致下一位客人發現杯子上有指紋或灰塵，造成服務品質的負面印象。
~使用手邊現有的客用毛巾快速擦拭杯緣，確保無灰塵即可#客用毛巾不應作為清潔工具使用，這違反了衛生規範，應使用專用的擦毛。
}

::OP-HK-04-Q03::房務員在執行冰箱區域的除塵作業時，除了冰箱內部、上方及兩側縫隙，哪一個「視線死角」是 SOP 特別強調要注意的？ {
~冰箱門把手的內側#門把手雖重要，但並非 SOP 中特別註記需注意積塵的特定「死角」區域。
~冰箱內部的照明燈罩#照明燈罩通常位於冰箱內，較不易積聚環境灰塵，非外部除塵的重點提醒事項。
=冰箱下方的空間#冰箱下方容易積聚棉絮與灰塵，卻常因位置低而被忽略，是維持客房深度清潔的關鍵細節。
~冰箱後方的散熱葉片#雖然散熱片也會積塵，但 SOP 主要是針對房務清潔可觸及的冰箱周邊（含下方）進行規範。
}

::OP-HK-04-Q04::在準備客房書桌的面紙盒時，下列哪一項設置最符合飯店的服務標準？ {
=面紙量維持在 8-9 分滿，且最上層的衛生紙需進行折角處理#維持 8-9 分滿可確保面紙好抽取且不顯空洞，折角則能傳遞「已完成整理」的服務細節。
~面紙盒必須補到全滿，以防客人在使用中途用完#面紙盒若過滿反而會導致抽取困難，甚至造成面紙撕裂，影響使用體驗。
~只要盒內還有剩餘面紙，不論剩多少都暫不補充#若面紙量過低，會讓客人感到飯店未落實房務巡檢，且可能造成使用上的不便。
~面紙應維持 8-9 分滿，但為求效率不需特別折角#折角是飯店業中常見的服務標記，代表此區域已通過房務員的檢查與整理。
}

::OP-HK-04-Q05::房務員在進行地板清潔作業時，為了有效地收集毛髮並確保清理動線不造成重複污染，正確的操作流程為何？ {
~由房門口往室內擦拭，並在床底收尾#由外而內的動線會將髒污帶進房內，且最後在床底收尾不便於垃圾的最後處理。
=由室內到外採 Z 字形拖法，將毛髮集中至門口後統一處理#Z 字形能確保最大覆蓋率，由內而外的動線則能引導髒污至門口收尾，避免踩踏已清處區域。
~在房內採取畫圓式拖法，將毛髮集中在房間正中央#畫圓拖法容易漏掉角落，且將毛髮集中在房間中央會讓後續的收尾處理變得困難。
~先拖乾淨走道區域，再處理床邊與桌下的死角#這種做法缺乏系統性動線，容易導致死角的毛髮被帶回到已拖乾淨的走道上。
}

::OP-HK-04-Q06::關於客房內行李掛衣架的除塵，房務員應注意哪些組成部分的清潔？ {
~只需擦拭層板表面即可#掛衣架結構包含鋼架，若僅擦拭層板，鋼架上的灰塵仍會沾染客人的行李或衣物。
=層板以及連接的黑色鋼架均需除塵#行李掛衣架由不同材質組成，全方位的除塵才能確保整體設施的潔淨感。
~只針對黑色鋼架除塵，因為層板通常有行李擋住看不見#層板是放置私人物品的主要位置，若不清潔將直接影響衛生，不能因被行李遮擋而省略。
~優先清潔衣架掛鉤，鋼架只需每週定期清理一次#除塵清潔應是客房整理的日常項目，不應將結構部分拆分週期處理。
}

::OP-HK-04-Q07::在清潔 Mini Bar 的杯子時，SOP 對於「抹布的使用」有非常嚴格的禁忌，下列哪一項做法是完全錯誤的？ {
~使用專用的單獨擦毛來擦拭#這是符合規範的正確做法，專用擦毛能確保杯具的衛生標準。
~絕對不可以拿取客用毛巾或擦拭過馬桶的毛巾來擦拭杯子#這是 SOP 中的明確禁忌事項，強調了衛生防線，避免細菌交叉污染。
=若杯子專用擦毛不夠時，暫時使用尚未拆封的乾淨客用毛巾代替#即使是乾淨的客用毛巾，其材質、纖維特性可能不適合擦杯具，且違反了工具分類使用的原則。
~檢查杯具是否有棉絮留下，並確保專用擦毛本身是乾燥且清潔的#這是檢查杯具清潔品質的必要步驟，能確保杯身不留殘跡。
}
