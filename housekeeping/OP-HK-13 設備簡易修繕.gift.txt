$CATEGORY: OP-HK-13 設備簡易修繕

::OP-HK-13-Q01::當房務人員在整房時發現燈具發出焦味且有輕微閃爍現象，根據修繕判斷原則，下列哪一項是正確的處理方式？ {
~先嘗試更換同規格的燈泡看看是否能解決。#SOP 規定若燈具有焦味、異味或跳電情形，應立即停止處理並回報，以避免觸電或火災風險。
=立即停止處理並回報值班主管。#涉及異味、冒煙或漏電等安全疑慮時，不屬於簡易修繕範圍，必須立刻通報主管處理。
~拿取 5 樓工具室的工具拆開燈座檢查內部線路。#簡易修繕以不拆機、不接線為原則，拆卸燈座內部結構已超出房務可處理的範疇。
~為了不影響整房進度，先記錄在群組等整房完再去回報。#安全問題具有急迫性，且 SOP 要求涉及異常狀況需「立即」回報，不可因趕房而延誤。
}

::OP-HK-13-Q02::在檢查冷氣功能時，若將溫度設定為 25度，使用測溫槍測量出風口的溫度。下列哪一個數據代表冷氣運作「不正常」，需要進行回報？ {
~出風口溫度為 20度#SOP 標準規定設定 25度 時，出風口應為 20度，此為正常範圍。
~出風口溫度為 19度#溫度低於20度 代表冷房效果良好，並非故障情形。
=出風口溫度為 24度#若出風口溫度接近設定溫度（如 24度），未達到預期的 20度溫差，顯示冷房效果不足。
~只要感覺有風吹出來就算正常，不需看溫度。#SOP 明確規定了量化標準（設定25度，出風口應為 20度），不能僅憑個人感覺判斷。
}

::OP-HK-13-Q03::關於遙控器的簡易排除作業，下列哪一個操作行為符合 SOP 的安全與維護規範？ {
~發現電池沒電時，將一顆新電池與舊電池混用以節省資源。#SOP 註意事項明確禁止混用新舊電池，以免導致電池漏液或損壞設備。
~若發現電池漏液，應徒手直接將電池取出並丟棄。#處理漏液電池具腐蝕風險，SOP 要求必須配戴手套處理並回報。
=更換電池後，需確認規格相同並測試電視是否可正常操作。#更換同規格電池並進行實際測試，是確認修繕成功的標準步驟。
~按鍵卡住時，使用專業工具拆開遙控器外殼清理內部。#簡易修繕以不拆機為原則，過度拆卸可能導致設備毀損。
}

::OP-HK-13-Q04::若住客反映浴室蓮蓬頭與軟管的「連接處」發生漏水現象，夥伴應優先執行哪項檢查？ {
~直接將整組蓮蓬頭與軟管換新。#連接處漏水通常可簡易排除，不應在未確認原因前就直接申請報廢換新。
=確認連接處內部的墊片是否位移或掉落。#SOP 指出軟管連接處漏水通常與墊片位置有關，重新裝回即可解決。
~使用強酸藥劑清洗連接處的防水膠圈。#SOP 禁止使用未經允許的強酸強鹼藥劑，這可能會損壞設備材質。
~因為涉及水管問題，不進行任何檢查直接通報工務。#確認墊片是否位移屬於房務可處理的「簡易排除」範圍，應先行判斷。
}

::OP-HK-13-Q05::在處理馬桶輕微堵塞後，關於工具的使用與收納，下列哪一項做法是正確的？ {
~使用完馬桶吸把後直接放回工具間即可。#未經清潔的工具會造成衛生問題與異味，且容易在移動過程中滴水。
~處理完畢後不需沖水測試，直接回報已完成。#SOP 要求處理後需沖水測試排水是否順暢，未測試完成不得回報修復。
=將吸把清洗乾淨並用垃圾袋裝套，避免滴水到客房地毯。#為了維護客房環境清潔，吸把必須清乾淨並妥善包裹，防止二次污染。
~若一次吸不通，立即倒入強鹼藥劑幫助疏通。#嚴禁使用未經允許的化學藥劑，以免發生危險或損壞管路。
}

::OP-HK-13-Q06::房務人員發現客房內的冰箱完全不冷，使用測溫槍測量後發現溫度為  18度，此時應採取的動作為何？ {
~自行拆開冰箱後方的背蓋檢查壓縮機。#SOP 明確規定不拆除冰箱、吹風機或保險箱之內部結構。
=確認門是否關緊、電源及溫度旋鈕後，記錄並回報主管。#冰箱溫度高於 15度需回報，且應先進行外部基本檢查（門、電源、旋鈕）。
~認為 18度尚可接受，不需進行任何回報。#SOP 規定冰箱溫度若在 15度 以上即需回報，這已超出正常冷藏標準。
~將冰箱內的食物全部移出丟棄後，直接在群組回報已修復。#溫度異常尚未排除前，不能回報已修復，且處理住客物品需謹慎。
}

::OP-HK-13-Q07::根據修繕回報紀錄規範，若 302 房電視訊號異常，經房務人員重新啟動機上盒後測試正常。下列哪一個是正確的紀錄格式？ {
~302 房電視好了。#此格式缺少狀況描述、處理方式與測試結果，資訊不全。
=302 + 電視 + 無訊號 + 重啟機上盒 + 測試正常。#此格式完整包含了房號、設備、狀況、處理方式及測試結果，符合 SOP 要求。
~電視壞了，我已經處理完，請主管確認。#此格式缺少房號與具體的處理步驟，且 SOP 要求應先測試完成再回報。
~302 房報修，請工務過來看。#若已簡易排除且測試正常，則不需直接請工務協助，且格式不符規範。
}

::OP-HK-13-Q08::當你在客房進行簡易修繕時，如果發現問題無法在多少時間內排除，應即時回報請求協助以避免影響進度？ {
~5 分鐘內。#SOP 給予的標準排除時間稍長於 5 分鐘。
=10 分鐘內。#SOP 建議以 10 分鐘內能排除的小問題為主，若卡太久會影響整房進度。
~20 分鐘內。#20 分鐘對於單間房的整房流程來說過久，會嚴重延誤進度。
~沒有時間限制，修好為止。#房務作業需考量整體時效，不能為了單一維修而無限期延後其他工作。
}

::OP-HK-13-Q09::在更換損壞的燈泡後，汰換下來的舊燈泡應如何妥善處置？ {
~直接丟入客房的垃圾桶內。#燈泡屬於需回收物料且具破碎危險，不應直接丟入一般垃圾。
~放在走道待清潔人員收走。#私自將廢棄物置於走道會影響美觀與安全。
=放置於 5 樓工具室的回收區。#SOP 明確規定汰換之燈具需放至 5 樓工具室的回收區統一處理。
~帶回員工休息室，等下班再處理。#這不符合公司規定的物料處理流程。
}

::OP-HK-13-Q10::住客反映保險箱無法開啟，且裡面存有住客的貴重物品，房務人員應如何處理？ {
=確認電池與顯示後，若無法排除即回報主管到場。#涉及住客財物時，SOP 規定必須有主管在場，以確保責任釐清與安全。
~使用萬能鑰匙自行幫客人開啟。#SOP 並未授權房務人員獨自開啟內含住客物品的保險箱。
~嘗試拆開保險箱背板進行解鎖。#SOP 嚴禁拆卸保險箱內部，這屬於專業修繕範圍。
~請住客稍後再試，等主管下班後再處理。#保險箱異常可能影響住客行程，應即時依流程處理並請主管協助。
}
