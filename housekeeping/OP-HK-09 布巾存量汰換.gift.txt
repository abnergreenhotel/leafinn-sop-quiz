::OP-HK-09-Q01::房務員在整理備品時，發現一條浴巾雖然沒有破損，但表面出現了洗不掉的明顯泛黃。根據現場作業規範，這條浴巾應如何處理？ {
~因為沒有破洞，可先放置在備品架的最底層供緊急備用#即使備品不足，也不得將外觀影響住客感受的不合格布巾放回乾淨架上。
=立即挑出並放置於報廢區，等待主管進一步評估#作業流程要求不合格或不確定的布巾需立即挑出，並由主管評估後續處理方式。
~直接將其剪角並做記號，轉為清潔用的抹布使用#降級使用或報廢的判斷需經過主管確認，第一線人員不應自行決定處理方式。
~送回洗衣廠加強漂白，不需要特別記錄或挑出#不合格布巾必須立即挑出避免誤用，且應由主管確認是否能透過回洗恢復品質。
}

::OP-HK-09-Q02::當主管核准將一批老舊毛巾「降級」為清潔抹布後，房務夥伴在管理上應注意哪項細節？ {

=必須進行標示或分區存放，確保不會再次流回到客用品#明確的區隔與標示能防止流程漏洞，保障客用布巾的純淨度。

::OP-HK-09-Q03::房務員在進行「庫存紀錄與回報」作業時，除了登記汰換的品項與數量，還必須特別記錄布巾布標上的哪一項資訊？ {
~布巾的產地#雖然布標有產地資訊，但 SOP 重點在於追蹤布巾的使用年限與損耗原因。
~洗滌溫度限制#洗滌標示是洗衣廠的參考資訊，非汰換紀錄時必須回報的關鍵統計數據。
=布巾出廠年份#記錄出廠年份有助於評估布巾的耐用度及判斷是否因正常老化而汰換。
~洗衣廠的名稱#合作洗衣廠通常固定，在每筆汰換紀錄中登記年份對於品質分析更有意義。

::OP-HK-09-Q04::飯店正值住房高峰期，房務部暫時從庫存區借調了布巾。根據流程，後續應該如何完成補回與記錄？ {
~連假後直接補回即可，不需特別回報#流程規定拿取皆須回報說明並確實記錄數量，以維持數據準確性。
=於連假後補回庫存區，且拿取時須回報說明並記錄數量#確實的借調紀錄與回報是維持安全庫存管理的必要動作。
~只要下週洗衣廠送回新貨，就不需要補回庫存區#庫存區的數量應維持定額，暫時借調必須完成「補回」的程序。
~借調庫存不需記錄，只需記錄正式汰換的數量#任何庫存變動都應記錄，否則會造成盤點數據不符。

::OP-HK-09-Q05::在髒布巾送洗前的核對流程中，除了統計數量，哪一項動作是正確的流程步驟？ {
=確認污漬位置與髒污程度，並填寫洗旺收據本#預先確認污漬狀況有助於洗衣廠後續處理，並確保回收時的核對準確。
~將所有布巾先在房務間浸泡消毒後再統計#現場作業主要負責統計與確認污漬，浸泡作業通常不在現場 SOP 範圍內。
~直接裝袋送走，數量由洗衣廠點清回報即可#SOP 要求送洗前需統計數量並填寫收據，不能僅依賴洗衣廠數據。
~為了速度，只需要統計總件數，不需要區分品項#細分品項與數量是統計回洗與盤點的基礎，不應簡略。
