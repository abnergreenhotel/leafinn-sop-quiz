$CATEGORY: OP-HK-01 房務進房禮儀

::OP-HK-01-Q01::當您準備進入客房進行清潔作業時，發現房門外的「請勿打擾（DND）」燈號亮起，以下哪一項是符合規範的操作？ {
~輕聲敲門三下，確認房客是否只是忘記關掉燈號#即使是輕聲敲門，也會打擾到設定「請勿打擾」的旅客，違反了該燈號的基本功能與服務禮儀。
=切記不可按門鈴或敲門，直接略過該房間#尊重客人的隱私與需求，當燈號亮起時，作業標準明確規定嚴禁進行任何干擾動作。
~按兩次門鈴，若無人回應再進房，以確保清潔進度#按門鈴會直接中斷客人的休息，且在 DND 狀態下自行進房屬於嚴重的服務違規。
~先在門口大聲喊 Housekeeping，等待十秒後再決定#口頭告知身分的前提是必須在可執行進房流程的情況下，DND 狀態下任何聲音示警都是不被允許的。
}

::OP-HK-01-Q02::在執行進房前的示警流程時，關於敲門與按門鈴的節奏，下列敘述何者最正確？ {
=連續敲擊門板 3 下，每次敲擊間隔約 2 秒，並同時口頭告知身分#固定的敲擊次數與規律的間隔能給予房客足夠的反應時間，且同步告知身分能消除房客對門外聲響的疑慮。
~快速敲門 5 下以引起注意，並連續按門鈴直到有人回應#過於急促或連續的聲響會造成房客心理壓力，不符合飯店服務的禮儀規範。
~敲門與按門鈴應分開執行，先按門鈴等待回應後再敲門#作業規範要求敲門與按門鈴應視為一套連續的示警組合，而非互斥或階段性的動作。
~為了保持安靜，僅需口頭告知 Housekeeping 而不需敲門#單純口頭告知可能因房客睡覺或洗澡而無法被聽見，必須結合物理敲擊或門鈴聲。
}

::OP-HK-01-Q03::房務員在第一次敲門並喊話後，即使沒有聽到房內有動靜，為什麼仍必須等待 10 至 20 秒才能進行下一步？ {
~為了讓房務員有時間在手報表上做紀錄#紀錄雖然重要，但等待的主要目的是基於對客人現場狀況的包容與尊重。
=這是為了確保房客若正在洗澡或睡覺，也有足夠時間對門外的聲音做出反應#適當的等待時間能避免在房客行動不便（如洗澡中）時強行進入，從而保護客人的隱私與安全感。
~這是為了確認走廊上是否有其他主管在巡視#等待流程是針對房內客人的互動標準，與外部主管巡視無關。
~為了觀察門縫是否有光線，判斷房內是否有人的輔助手段#觀察光線並非標準流程中的主要等待原因，且非所有房間皆能透過此方式判斷。
}

::OP-HK-01-Q04::若經過兩到三次的敲門與確認流程後，房間內依然沒有任何回應，房務員應如何執行「進入檢查」？ {
~直接將房門全開，快速進入房間檢查浴室與床位#直接全開門會讓房內狀況完全暴露，若判斷錯誤將造成極大的服務失誤。
=使用萬用卡（Master Key）將門稍微打開，並再次喊話確認#「稍微打開」是最後一道防線，配合再次喊話能給予房客最後的提醒機會，並確認房內是否真的無人。
~站在門外大聲撥打房內分機，確認無人接聽後再刷卡進入#雖然撥打電話也是一種方式，但這並非該 SOP 文件所規範的進房禮儀步驟。
~不需再喊話，刷卡後立刻進入並鎖上房門開始清掃#進房後立即鎖門會增加意外碰撞或誤入的風險，且省略喊話違反了確認程序。
}

::OP-HK-01-Q05::房務員誤以為房內無人而打開房門，卻發現客人在房內時，下列哪一項後續處理流程最符合 SOP 要求？ {
=立刻向客人致歉並退出，隨即回報櫃檯與值班主管，由飯店後續處理#致歉與撤出是立即的補救，而回報主管能確保飯店能針對此失誤對客人進行適當的安撫或處理。
~向客人說明現在是清掃時間，詢問客人是否可以現在開始清理#當發生誤開門的驚擾時，應以致歉與離開為優先，而非強行或就地進行服務溝通。
~關上門後裝作沒事發生，繼續前往下一間房作業，避免尷尬#不回報會導致飯店失去彌補客人的機會，且客人若客訴會使事態擴大。
~只要客人沒有生氣，就只需口頭交接給下一班夥伴即可#作業規範明確要求必須回報至櫃檯與主管，不能視客人反應而私下吃案。
}

::OP-HK-01-Q06::為了避免發生「誤開房門」的情況，房務夥伴在敲門與刷卡進入前，最重要的預防檢查動作是什麼？ {
~確認自己的名牌是否有配戴整齊#名牌與儀容是基本要求，但與預防「進錯房間」的實務操作關聯性較低。
=務必確認房號與目前的房況（如：續住、退房、空房）#確認房號與房況能從源頭減少錯誤發生的機率，是進房前必須執行的核心注意事項。
~檢查萬用卡的電池是否有電#這屬於設備維護範疇，無法預防因人為疏忽導致的誤開房門。
~確認走廊的逃生門是否處於開啟狀態#這屬於公共安全檢查，與進入客房的標準作業流程無直接因果關係。
}
