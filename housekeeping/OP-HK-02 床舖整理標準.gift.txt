$CATEGORY: OP-HK-02 床舖整理標準

::OP-HK-02-Q01::房務夥伴在整理完 501 號房的床備品後，準備前往下一間房。根據作業標準，關於布巾數量的登記記錄，下列哪一種做法最正確？ {
=完成該間客房的拆床作業後，立即填寫對應房號的數量。#及時記錄能確保數量的準確性，避免後續因忙碌或遺忘而造成誤差。
~為了節省時間，等該樓層所有房間都拆完後再統一填寫。#這雖然看似效率高，但容易導致數量記憶混淆或填寫錯誤，不符合一間寫一間的標準。
~先口頭記住各房數量，等收工回到備品間再一併登記。#口頭記憶不可靠，且延後登記會大幅增加錯誤率，不符合現場作業程序。
~只記錄有異常損壞的布巾房號，數量正常的房間不需填寫。#作業標準要求確實填寫對應房號的數量欄位，而非僅記錄異常情況。
}

::OP-HK-02-Q02::在進行套枕頭套的作業時，為了讓枕頭能更輕易地塞進枕套並保持外觀平整，夥伴應優先採取哪項動作？ {
=先以手刀下壓枕頭中間，再提起枕頭左右角塞入。#透過下壓與提角的動作可縮小體積，讓填充過程更流暢且定位更精準。
~將枕頭對折後直接用力推入枕頭套底部。#對折可能導致內部纖維不均或枕套受力過大損壞，不如手刀下壓法專業。
~將枕頭套反過來捲起，再像穿襪子一樣套上枕頭。#這種方法在處理大型飯店枕頭時效率較低，且難以確保枕角對齊。
~先將枕頭套撐開，直接抓住枕頭長邊塞入。#直接塞入容易造成枕頭在套內扭曲，且操作起來較為費力。
}

::OP-HK-02-Q03::夥伴完成鋪床後進行最後檢查，兩顆枕頭立式擺放對靠在床頭板上。關於枕頭套「開口」的方位，哪一項符合標準？ {
=兩顆枕頭的開口應朝向內側，呈現互相面對的狀態。#開口朝內可以隱藏開口處，讓視覺重心集中在床舖中央，呈現整齊美觀的效果。
~兩顆枕頭的開口一律朝向房門方向，方便客人使用。#這會導致一邊開口露在外側，視覺上顯得凌亂且不對稱。
~開口應一律朝向床墊底端，避免客人側睡時看到。#枕頭是立式擺放，開口朝下會讓枕頭重心不穩且不便於收納餘布。
~開口朝向兩側外圍，讓床頭區域看起來更寬敞。#開口外露會直接看到內部的枕芯，不符合飯店美學的精緻度要求。
}

::OP-HK-02-Q04::當你在鋪設被套時，為了防止客人在睡眠翻身時，內部的「被芯」與「被套」發生分離而導致保暖不均，應執行哪一項關鍵動作？ {
=確保被芯塞進被套後，將被角確實打上活結。#打活結能有效固定被芯，且在後續拆床時也易於解開，平衡了實用與效率。
~使用強力膠帶將被芯四角貼合在被套內側。#膠帶會損壞布料且不衛生，不符合飯店專業房務規範。
~將被角打成死結，確保長期使用都不會鬆脫。#死結會導致洗滌拆卸困難，造成後續夥伴作業負擔，應使用活結。
~只需將被芯鋪平，靠棉被本身的重量壓實即可。#單靠重量無法防止滑動，容易造成被芯位移，影響客人的睡眠舒適度。
}

::OP-HK-02-Q05::房務夥伴接到通知，502 房有嬰兒隨行，需要提供「澡盆組 (PKG)」。根據標準，除了澡盆本身，你還必須確認哪一項備品已包含在內？ {
=水瓢#根據 SOP 項目 6.4，澡盆組 (PKG) 明確包含澡盆與水瓢兩樣工具。
~拋棄式小毛巾#雖然洗澡需要毛巾，但它不屬於 SOP 中定義的澡盆組固定配件。
~嬰兒專用洗髮精#洗劑通常屬於個人清潔備品，不包含在澡盆組的硬體配套中。
~防滑地墊#澡盆組標配僅列出澡盆與水瓢，防滑墊並非此組合的標準配件。
}
