$CATEGORY: OP-HK-03 浴廁清掃標準

::OP-HK-03-Q01::房務員在執行浴廁清掃時，為了確保效率與清潔品質，正確的刷洗先後順序應為何？ {
~浴缸 > 玻璃 > 蓮蓬頭 > 馬桶 > 洗手台#此順序與 SOP 要求相反，這會導致已清潔區域在後續作業中再次受到污染。
=洗手台 > 馬桶 > 蓮蓬頭 > 玻璃 > 浴缸#這符合 SOP 要求的由外而內清潔邏輯，確保清潔路徑的一貫性與衛生。
~馬桶 > 洗手台 > 浴缸 > 蓮蓬頭 > 玻璃#馬桶應在洗手台之後清潔，且浴缸應作為刷洗流程的最後一站。
~玻璃 > 蓮蓬頭 > 洗手台 > 馬桶 > 浴缸#雖然浴缸排在最後，但玻璃不應作為清潔起點，應遵循從洗手台開始的順序。
}

::OP-HK-03-Q02::在整理洗手台區域時，關於抽屜內部的清潔與整理標準，下列何者正確？ {
~抽屜需關閉整齊，並在把手處貼上已清潔封條#SOP 未提及封條要求，且明確規定抽屜的擺放狀態以便查驗。
~抽屜內部只需確認吹風機收整，不需要除塵#SOP 明確規定抽屜內部需要進行清潔除塵作業。
=需將抽屜打開備查，並確保內部清潔無塵#這符合 SOP 要求將抽屜打開以便後續督導查驗除塵與收整狀況的標準。
~抽屜內只要沒有垃圾，不需要特意打開給主管看#這違反了 SOP 中「需將抽屜打開備查」的具體執行說明。
}

::OP-HK-03-Q03::關於免治馬桶的清潔重點，除了刷洗內座外，夥伴最容易忽略但也必須確實清潔的部位是？ {
~沖水把手與水箱蓋上方#雖然這也需要清潔，但 SOP 特別強調的是出水孔與細部髒污的檢查。
=出水孔處及馬桶下緣有無髒污#這是 SOP 特別提醒的死角，包括出水孔、馬桶下緣及坐墊是否有血跡或污漬。
~馬桶蓋後方的軸心縫隙#SOP 主要是強調出水孔與底座區域，軸心並非 SOP 提到的關鍵檢查點。
~牆壁噴濺到的水花#這屬於牆面清潔範疇，非馬桶本體清潔標準中強調的細節。
}

::OP-HK-03-Q04::夥伴在整理淋浴空間時，針對蓮蓬頭的清潔與維護，哪項作業是正確的？ {
~只需擦拭金屬表面，確保光亮無水漬即可#僅擦拭表面是不夠的，SOP 要求需有瀝乾的動作以確保衛生。
=清潔時需將蓮蓬頭轉開瀝乾後再裝回#這能防止蓮蓬頭內部積水產生異味或衛生問題，是 SOP 中的明確要求。
~蓮蓬頭軟管不可轉動，以免損壞密封墊片#這與 SOP 要求轉開瀝乾的指引相違背。
~只要確認出水順暢，就不需要拆開處理#瀝乾內部積水是標準作業流程的一環，不論出水是否順暢都應執行。
}

::OP-HK-03-Q05::在檢查洗髮精、沐浴乳的容量時，何時需要進行補充？ {
~容量低於瓶身二分之一時#SOP 規定了更具體的參考基準線，而非粗略的二分之一。
~瓶內清潔劑快見底時再補充即可#這會導致客人使用不便，不符合飯店服務標準。
=容量低於貼紙下緣位置（紅線處）時#這是 SOP 規定的精確補充時機，確保客人有充足的量可使用。
~每天固定補滿，不論剩餘多少#雖然這也是一種做法，但 SOP 主要是設定了低於貼紙下緣的補充門檻。
}

::OP-HK-03-Q06::整理雙人房的漱口杯時，若採用「直式擺法」，正確的顏色擺放順序應為？ {
~白色在外、黑色在內#這與 SOP 中規定的雙人房直式擺放順序相反。
~兩支黑色在外、兩支白色在內#雙人房通常只有兩支杯子，此敘述較像四人房的情況且擺法不符。
=黑色在外、白色在內#這是 SOP 針對雙人房直式擺法明確設定的視覺標準。
~隨機擺放，只要杯口擦拭乾淨即可#旅館非常講究視覺的一致性，SOP 有明確規定顏色順序。
}

::OP-HK-03-Q07::夥伴更換完垃圾桶袋子後，關於塑膠袋收口的細節，正確的做法為何？ {
~收口處應朝向客人進門最明顯的方向#這樣會影響視覺美觀，讓客房顯得不夠精緻。
~將多餘的袋口塞入垃圾桶底部隱藏#這不是 SOP 要求的特定位置，收口方向應有固定準則。
=塑膠袋向內收的地方需要朝向牆壁方向#這能將不美觀的收口隱藏起來，確保客人視角看到的是平整的桶面。
~不需特別在意方向，只需套緊桶緣#忽視細節會降低客房質感，SOP 明確要求收口應向後（牆壁）。
}

::OP-HK-03-Q08::在洗手台旁擺放「三小瓶」與「洗手乳」時，由內而外的正確順序應為何？ {
~洗手乳 > 身體乳 > 衣物手洗精 > 香香噴瓶#此順序為由外而內，方向與 SOP 要求完全相反。
=香香噴瓶 > 衣物手洗精 > 身體乳 > 洗手乳#這符合 SOP 規定的排列邏輯，確保備品擺放的統一美感。
~身體乳 > 洗手乳 > 香香噴瓶 > 衣物手洗精#此排列完全不符合 SOP 要求的特定項目順序。
~衣物手洗精 > 香香噴瓶 > 洗手乳 > 身體乳#這混淆了噴瓶與洗劑的擺放邏輯。
}

::OP-HK-03-Q09::關於浴缸置物架的擺放位置與清潔標準，下列敘述何者正確？ {
~置物架應放在浴缸正中央，以利左右對稱#SOP 要求的是 1\:2 的比例位置，而非正中央。
~只需沖洗表面，不需特別清理圓孔處#SOP 特別強調縫隙與圓孔處需進行細清與除霉。
=放置於靠近大圓弧處約 1\:2 的位置，且圓孔需無黴垢#這完全符合 SOP 中對於置物架擺放比例與細部清潔的要求。
~放置於浴缸前端，方便客人坐下時拿取#SOP 有明確指定 1\:2 的特定擺放區域以維持美觀。
}

::OP-HK-03-Q10::浴室清潔最後完成時，對於地板與踏墊的收尾工作，何者符合 SOP？ {
~地板濕拖完畢後即可離開，讓其自然風乾#這可能會讓客人滑倒，且 SOP 提到需要用滾滾處理毛髮。
=踏墊需整齊捲好，放置在毛巾籃的左邊#這是 SOP 關於備品擺放 8.1 項中的明確收納位置規定。
~踏墊應平舖在浴室門口，不需捲起#SOP 要求的收納方式是「整齊捲好」，並非平舖。
~只要把地板毛髮撿乾淨，不一定要用滾滾#SOP 明確指引需使用「滾滾」來徹底清除細微毛髮。
}
