$CATEGORY: OP-HK-06 備品車維護

::OP-HK-06-Q01::在走廊整理客房時，關於備品車的停放位置，下列哪一種做法最符合安全規範？ {
~為了拿取方便，將備品車停放在逃生通道口#逃生通道必須時刻保持暢通，放置備品車會造成安全隱患。
=將備品車靠邊停放，或是停在正在清理的房門口#靠邊停放能確保旅客通行空間，且不阻礙逃生動線，是兼顧效率與安全的做法。
~將備品車橫停在走廊中央，以防旅客誤入工作區域#橫停在走廊中央會完全阻斷客人與同仁的行動路線，並非標準做法。
~只要不影響自己工作，隨意停放在走廊任何位置即可#隨意停放可能造成客人跌倒或在緊急狀況時阻礙疏散。
}

::OP-HK-06-Q02::若在整理備品車時，發現車輪輪胎出現故障或層架有變形狀況，夥伴應如何處理？ {
~利用車上的十字起子嘗試自行拆卸修復#SOP 明確規定不可自行拆修，以免造成設備二次損壞或人員受傷。
~先繼續使用，等車子完全不能推動時再處理#帶故障作業會增加工作難度，甚至可能損壞地面或造成職業災害。
=立即通報主管，並依指示進行後續處理#發現設備異常應立即通報專業管理人員，以確保工作安全與設備完好。
~將故障的車子推到逃生樓梯間藏起來#這不僅沒有解決問題，還違反了不能阻礙逃生通道的安全規定。
}

::OP-HK-06-Q03::在擺放備品車上的布巾時，關於「大毛巾」與「床單」的配置，下列敘述何者正確？ {
~重物應放在最上層，方便隨手拿取#重物置於上層會導致重心不穩，增加備品車翻倒的風險。
=床單應放置在第二層，而大毛巾應放置在第三層#重物（如大毛巾）置於下層可確保車體重心穩固，且符合指定的分類標準。
~為了美觀，將所有大毛巾與床單混合堆疊在第一層#混合堆疊會導致取用不便，且違反了分層擺放的標準作業流程。
~將大毛巾懸掛在右側的紙拖鞋掛鉤處#右側掛鉤有其指定用途，懸掛厚重毛巾會影響其他備品的放置空間。
}

::OP-HK-06-Q04::當工作結束準備進行備品補充時，針對茶包與餅乾的點檢，最關鍵的注意項目為何？ {
~確認包裝袋上的顏色是否有退色現象#雖然外觀重要，但包裝色澤通常非 SOP 強調的品質檢核重點。
=檢查茶包與餅乾的日期是否快到期#確保食品安全是備品維護的核心，快到期的食品需及時處理以維護服務品質。
~確認餅乾是否為客人最喜歡的口味#口味通常為採購決定，現場夥伴應優先確保產品的效期與新鮮度。
~將所有茶包重新排列，確保顏色交錯擺放#美觀雖好，但比起效期檢查，視覺排列並非確保品質的最關鍵步驟。
}

::OP-HK-06-Q05::夥伴若需要使用「木質保養劑」來整理客房家具，應在備品車的哪一個區域取用？ {
~備品車的最上層紙盒內#最上層主要放置小件備品與文書用品，清潔劑通常不放於此區。
~備品車的左側袋子中#左側主要規劃為垃圾袋與回收袋的放置區域。
=備品車的右側懸掛或放置處#右側是指定放置清潔藥劑、保養劑與除塵工具的空間，方便操作時拿取。
~第一層的開封面紙旁#面紙與化學藥劑混合擺放可能造成交叉污染，不符合配置規範。
}

::OP-HK-06-Q06::關於工作結束後的收尾作業，下列哪一項描述符合標準作業流程？ {
~將剩餘的髒布巾直接留在車上，交由下一班處理#未處理的髒布巾會產生衛生問題，且不符合交接與整理規範。
~將備品車推入倉庫後即可直接打卡下班#下班前應完成布巾繳回與垃圾清理，而非僅是移動車子。
=需將布巾放回 6 樓布巾室，處理垃圾並更換新垃圾袋#完整的收尾包含物料歸位與垃圾清運，這能確保備品車在下一個時段處於可用狀態。
~只需將車上的垃圾袋打結，不需要拿去處理#垃圾必須徹底移除並換新，不能留在工作區域或車上。
}

::OP-HK-06-Q07::如果客房內的「起皂器」螺絲鬆脫了，夥伴應該在哪裡找到合適的工具進行簡易鎖緊？ {
~在備品車的第一層布巾下方尋找#第一層主要放置布巾類，將細小工具藏於其下會增加尋找困難且不專業。
=在備品車的上層區域找尋十字起子與螺絲#上層規劃了放置常用工具（如起子、酒精、螺絲）的區域，以應對客房內的小型維修需求。
~到右側的回收袋中翻找看看#回收袋內應只有回收物品，不應混入工具與零件。
~詢問櫃檯是否有備用的維修工具#備品車上層已有配置標準維修工具，應優先使用車上資源以提高效率。
}

::OP-HK-06-Q08::在備品車的分層配置中，第一層除了開封面紙與垃圾袋外，還應該擺放哪些物品？ {
~被套與枕頭套#大型寢具類（如被套）通常放置在較下層（第二層），以利收納。
=小毛（小毛巾）與足布（腳踏墊）#小毛與足布體積較小且使用頻率高，放置於第一層符合人體工學與分類邏輯。
~捲紙與咖啡包#捲紙與咖啡包屬於消耗性備品，應放置於最上層的配置格中。
~除塵撢子與水垢清潔劑#清潔工具與藥劑應統一放置在備品車右側，避免與乾淨布巾接觸。
}
