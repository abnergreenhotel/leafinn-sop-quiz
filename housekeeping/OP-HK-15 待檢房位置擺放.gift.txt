$CATEGORY: OP-HK-15 待檢房位置擺放

::OP-HK-15-Q01::在準備待檢房時，針對保險箱與拖鞋籃的擺放位置，下列哪一項操作最能節省檢查人員的時間並符合標準？ {
~將保險箱門全開，並將拖鞋籃完全拉出至走道。#過度拉出設備可能會阻礙走道動線，標準要求僅需拉出部分即可確認內容。
=將保險箱打開三分之一，並將拖鞋籃也拉出三分之一。#拉出三分之一的比例足以讓檢查人員一眼辨識內部整潔與備品齊全，同時保持房內整齊。
~保險箱保持關閉，僅將拖鞋籃稍微拉出五公分。#關閉保險箱會導致檢查員需額外動手開啟，不符合待檢房「易於視覺檢查」的原則。
~將保險箱打開一半，拖鞋籃則留在櫃子內不拉出。#拖鞋籃若不拉出，檢查員將難以確認籃內底部是否有毛髮或灰塵。
}

::OP-HK-15-Q02::房務夥伴在準備待檢房的沙發區時，將沙發墊「側立」擺放的主要目的是什麼？ {
~為了讓沙發墊通風，避免產生異味。#雖然通風有益，但側立擺放的核心重點在於細節的清潔檢視。
~為了展示沙發墊的厚度與品質，提升客房質感。#待檢房的佈置是以「清潔稽核」為導向，而非純粹的美觀展示。
=為了方便檢查人員確認墊子下方是否有毛屑或垃圾。#側立墊子能暴露出平時隱藏的縫隙，確保檢查時不會遺漏任何細小的垃圾。
~為了防止沙發墊變形，保持其蓬鬆度。#這與待檢房的檢查作業標準無直接關聯。
}

::OP-HK-15-Q03::在佈置書桌上的書面資料（如顧客意見表、住房須知）時，下列哪種擺放方式正確？ {
~將所有資料整齊疊放在遙控器下方，保持桌面清爽。#疊放會遮蔽下方的內容，檢查員無法確認資料是否齊全或有毀損。
=將意見表與須知內容攤開放，確保所有資訊皆可視。#攤開放擺設能讓檢查人員快速確認備品內容，且給予顧客隨手可得的服務感。
~將資料放入抽屜內，避免桌面顯得凌亂。#待檢房要求書桌上的特定品項需擺放於桌面上以便檢查。
~僅擺放迎賓卡，其餘須知與意見表則立在玻璃盅旁。#立放雖然可見，但未符合標準中要求的「攤開放」佈置細節。
}

::OP-HK-15-Q04::若執行房務作業當天天氣轉為陰雨綿綿，關於客房窗戶的通風處理，應如何操作？ {
~維持開啟一個拳頭寬度，以確保空氣流通。#下雨天若開啟窗戶，雨水可能濺入室內損壞窗框或地板。
~改為全開窗戶，加快水氣蒸散速度。#此舉會導致室內大面積受潮，是錯誤的判斷。
=不開啟窗戶，但仍需將客房窗簾拉開。#天雨時不開窗是為了保護客房資產，但拉開窗簾是為了讓光線進入以便檢查。
~窗戶與窗簾皆緊閉，避免室外潮濕空氣進入。#窗簾仍應拉開，以便檢查人員檢查窗框與玻璃的清潔度。
}

::OP-HK-15-Q05::關於浴室內馬桶的待檢狀態，下列哪一項操作最符合衛生檢查的標準？ {
~將馬桶蓋關上，並在上方放置「已消毒」封條。#關上蓋子會阻礙對馬桶內部與內蓋邊緣的清潔稽核。
=將馬桶的內蓋與外蓋全部掀起擺放。#全掀式的狀態能讓檢查人員直接觀察到所有接縫處與馬桶內部是否徹底清潔。
~僅掀起最外層的蓋子，保留座墊平放。#座墊平放會遮住座墊下方的區域，那裡往往是清潔的盲點。
~維持使用後的原始狀態，由檢查人員自行翻動。#待檢房的精神是減少檢查人員的物理動作，將重點集中在視覺確認。
}

::OP-HK-15-Q06::在洗手台區域準備待檢時，關於抽屜與浴巾籃的擺放，何者正確？ {
=洗手台下方的抽屜需拉出，浴巾籃也需拉出約三分之一。#拉出抽屜與浴巾籃能方便確認吹風機、備品與毛巾的整潔及摺疊狀況。
~抽屜需緊閉以防碰撞，僅將浴巾籃拉出至與洗手台齊平。#抽屜緊閉會讓檢查員需要額外開啟動作，降低檢查效率。
~抽屜拉出一半，但浴巾籃需完全推進櫃內。#浴巾籃若不拉出，檢查員難以看到後方是否有遺漏的髒毛巾。
~僅需將洗手台上的備品擺正，下方儲藏空間不需處理。#下方的儲藏空間是毛髮最容易累積的地方，必須拉出受檢。
}

::OP-HK-15-Q07::夥伴正在準備 Mini Bar 的待檢狀態，發現熱水壺與冷水壺都已清潔完畢，他應該如何擺放？ {
~將熱水壺與冷水壺的蓋子全部蓋緊，保持美觀。#蓋緊蓋子會讓檢查員無法確認內部是否有水垢或殘水。
=將蓋子打開，讓檢查人員一眼就能看見內部清潔狀況。#開啟蓋子除了方便檢查清潔度，也能讓殘餘水氣通風散發。
~熱水壺蓋子打開，冷水壺因為較小則不需打開。#標準要求兩者皆須打開，以確保容器內部均達到標準。
~將蓋子側放於壺旁，並將水壺反過來倒扣。#反過來倒扣反而會遮蔽壺口，不便於視覺檢查。
}

::OP-HK-15-Q08::關於浴缸的待檢細節，當浴缸塞子清潔完成後，最合適的操作做法為何？ {
~將塞子緊壓在排水孔上，測試其止水功能。#這會導致排水孔內部被遮蔽，檢查員無法確認孔洞周邊是否有汙垢。
~將塞子垂直立在排水孔正上方。#立在正上方不夠穩固，且遮擋視線效果與塞住類似。
=塞子轉開後放在旁邊，方便檢查排水孔清潔度。#將塞子移開能確保排水孔內外的毛髮與汙垢已被徹底清除，且塞子本身也易於被檢查。
~將塞子直接收進洗手台抽屜內。#塞子屬於浴缸零件，應留在原位以便檢查完整性。
}
