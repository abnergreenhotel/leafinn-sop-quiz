$CATEGORY: OP-HK-10 查房作業流程

::OP-HK-10-Q01::夥伴進入客房準備執行查房作業時，第一步最合適的操作是什麼？ {
~由外向內檢查，先巡視地板是否有明顯垃圾。#查房應遵循由內到外的原則，且應先確認房況更新而非直接清潔。
=確認群組房況是否有延退或續住等更新，並開啟全室電源。#掌握最新房況可避免作業衝突，開啟電源則能確保檢查環境光線充足並同步測試電器。
~先到浴室檢查沐浴用品是否充足。#未確認房況與開啟電源前就進入特定區域，容易忽略整體動線效率與電器測試。
~直接測試電視網路與頻道是否正常。#雖然需要測試電視，但在確認房況與開啟全室電源之前不應跳過前置準備步驟。
}

::OP-HK-10-Q02::在檢查洗手台上的三小瓶備品（洗衣精、乳液、洗手乳）時，關於擺放與補充的標準，下列何者正確？ {
=擺放順序應為：洗衣精、乳液、洗手乳，並補充至八分滿。#統一的擺放順序能確保服務的一致性，補充至八分滿則可避免溢出並維持視覺美感。
~只要三瓶都有在洗手台上即可，順序並不重要。#隨機擺放會降低客房的專業感，不符合標準作業指導書要求的固定順序。
~為了環保，內容物低於三分之一時才需要補充。#這可能導致下一位房客使用不足，標準規定應補充至八分滿以確保備品充足。
~擺放順序應為：洗手乳、乳液、洗衣精，並補滿至瓶口。#此順序不符合 SOP 規範，且補得太滿容易在房客按壓時造成液體滲漏。
}

::OP-HK-10-Q03::整理床鋪區域時，為了維持視覺上的整齊與專業感，被子與枕頭之間應保持多少距離？ {
~被子應緊貼枕頭，不留任何縫隙。#緊貼會顯得過於侷促，且不符合 SOP 定義的標準美感比例。
=距離約 15 公分，大約是一支遙控器的長度。#使用具體參照物（如遙控器）測量距離，能確保不同夥伴整理出的床鋪視覺效果一致。
~距離約 30 公分，方便房客放置靠枕。#此距離過長，會讓床鋪看起來不夠連貫，且背離了 15 公分的標準要求。
~距離約 5 公分，只要看得到床單即可。#過小的距離無法展現被子反摺後的層次感。
}

::OP-HK-10-Q04::檢查浴室馬桶區域時，關於捲紙架的維護細節，下列哪一項是常被遺漏但必須達標的？ {
=確認捲紙有摺成三角形，且捲紙架上方已完成除塵。#摺三角形代表房務已巡檢過，而架子上方是房客視線易及處，除塵能展現對細節的重視。
~只要紙筒內還有紙，就不需要調整外觀。#忽略外觀調整會讓房客覺得房間未經專業整理。
~捲紙剩餘用量只要超過一半就不需檢查。#SOP 要求需檢查剩餘用量，以確保房客使用過程中不會突然斷紙。
~只需要注意馬桶內部是否刷乾淨，捲紙架不屬於檢查範圍。#捲紙架與馬桶按鍵、馬桶邊縫同屬浴室檢查的關鍵項目。
}

::OP-HK-10-Q05::在執行 Mini Bar 與冰箱的查房項目時，下列哪項做法最能展現對衛生標準的堅持？ {
~只要確認冰箱電源有開，且裡面沒有垃圾即可。#冰箱還需檢查是否有前房客留下的異味或食物，以確保清新。
=檢查玻璃杯是否有指紋或棉絮，並確認冰箱無異味與遺留物。#透明器皿上的指紋最易引起衛生疑慮，確保冰箱空置且無味是基本要求。
~為了節省時間，玻璃杯若看起來沒人用過就不用再次檢查。#即使未被使用，玻璃杯仍可能積聚棉絮或在清掃過程中沾染指紋。
~只需確認茶包和餅乾的數量是否正確。#數量正確僅是基本，層架與托盤的除塵同樣是查房指導書中的重點。
}

::OP-HK-10-Q06::查房時檢查床區細節，下列哪一個位置是必須特別留意是否有毛髮、垃圾或食物殘渣的重點區域？ {
~床頭板的頂端。#床頭板雖然也要除塵，但最容易積聚食物殘渣與垃圾的地方是床底。
~被子的外表面。#床底是更容易隱藏垃圾與食物殘渣的視覺死角。
=床底及其周邊地板。#房客常會遺落物品或在此處殘留零食碎屑，若不檢查會導致嚴重的衛生問題。
~床墊的側邊縫隙。#雖然也是清潔點，但指導書特別強調床底需留意毛髮、垃圾、食物屑及拖鞋。
}

::OP-HK-10-Q07::在洗手台區域檢查時，關於排水孔蓋（小圓蓋）的標準擺放角度，以及毛巾的整理要求為何？ {
~排水蓋完全蓋平，毛巾疊放在抽屜內。#排水蓋蓋平不利於排水，且毛巾需挺立呈現以示專業。
=排水蓋呈 45 度角，且大小毛巾的緹花要對齊並挺立。#45 度角是排水與美觀的平衡點，毛巾緹花對齊則能展現極致的服務細節。
~排水蓋隨意放置，毛巾只要有摺整齊即可。#這缺乏標準化美感，無法滿足 SOP 對「挺立」與「緹花對齊」的要求。
~排水蓋垂直立起，毛巾緹花向內反摺隱藏。#垂直立起不穩定，且緹花應是視覺亮點而不應刻意隱藏。
}

::OP-HK-10-Q08::關於客房電視的測試標準，下列哪一項操作符合作業流程的要求？ {
=音量應設定在適當的 7 號，並測試網路與頻道功能。#固定音量可確保房客開啟時不會受到驚嚇或聽不到聲音，測試網路與頻道則是功能查驗。
~音量設定為 0 以避免干擾隔壁客房。#設定為 0 會讓房客誤以為電視沒聲音或故障，建議音量為 7。
~只要確認電視能開啟，不需要特地切換頻道。#SOP 要求包含頻道測試，以確保訊號傳輸完全正常。
~音量隨意設定，只要遙控器有電即可。#這忽略了對房客體驗的細節管理，不符合作業流程中對音量的具體規範。
}
