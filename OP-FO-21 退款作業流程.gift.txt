::OP-FO-21-Q03::在處理「電話訂房、已匯款的訂單要在德安取消」時，下列哪一項最符合帳務或紀錄要求？ {

::OP-FO-21-Q04::執行「官網訂單已付款完畢、顧客要取消」時，下列哪一項作法最能避免資料、帳務或後續追蹤風險？ {
~若顧客需求明確且現場可配合，可先彈性處理，事後再補充紀錄或向主管說明。#錯誤。錯在把明確限制改成現場彈性判斷；限制事項不能用事後紀錄取代。 正確作法是：原則上請顧客自行在官網取消，若無法操作可於 web 手動取消並記錄原因，再確認德安是否成功下傳。
~若系統資料暫時不完整，可先完成對客服務，再於交班前補登或修正資料。#錯誤。錯在把正式系統紀錄延後補做，可能造成訂單、房況或帳務資料錯誤。 正確作法是：原則上請顧客自行在官網取消，若無法操作可於 web 手動取消並記錄原因，再確認德安是否成功下傳。
