$CATEGORY: OP-GS-01 服裝儀容標準

::OP-GS-01-Q01::夥伴在準備上班前進行自我檢查，發現制服上有輕微但可見的摺痕，且名牌放在置物櫃忘記配戴。根據規範，下列哪項做法正確？ {
~因為摺痕不明顯，只要補戴名牌後即可直接上線服務#規範要求制服需整燙平整且不得有明顯皺摺，儀容不整者不得直接上線。
~名牌可放於口袋內，只要服裝整潔即可符合標準#名牌必須配戴於明顯且固定的位置，不可隨意放置或藏於口袋。
=應先將制服整燙平整並正確配戴名牌，確認整體外觀後再上線#上班前的自主檢查涵蓋了服裝平整度與名牌配戴，這是確保專業形象的基本要求。
~只要整體外觀乾淨無污漬，皺摺部分可於值勤空檔再處理#上線服務前的檢查應包含確保無明顯皺摺，而非上線後才處理。
~若趕著交班，可先口頭向主管報備後再省略自我檢查流程#標準作業程序強調上班前必須完成自我檢查，以維持統一的服務水準。
}

::OP-GS-01-Q02::在寒冷的天氣裡，某位夥伴想在規定制服外加穿一件自己購買的私人防寒外套。根據制服穿著標準，下列敘述何者正確？ {
~只要私人外套顏色與制服相近，即可自行搭配穿著#規範嚴禁私自混搭非規定款式的衣物，以維持團隊形象的統一。

~私人的保暖衣物只能穿在制服內層，不可露出來#雖然保暖重要，但標準中強調的是制服外觀的完整性與統一性。
~只要主管口頭同意，即可隨意搭配不同款式的配件#標準規定需依職務規定或指定服裝穿著，而非隨意變動。
~外套屬於個人防寒物品，不受到制服標準作業指導書的規範#制服標準明確涵蓋了外套與配件的穿戴規範。

::OP-GS-01-Q03::某女性夥伴擁有長髮，關於其髮型整理的原則，下列哪項做法最符合「綠宿行旅」的專業形象？ {
~為了展現個人風格，可隨意選擇誇張的染髮顏色與造型#過於突兀或誇張的造型需先與主管確認，以免影響專業形象。
=長髮應依規定綁束，且不遮擋臉部與視線，保持俐落感#整齊、俐落且不干擾視線是髮型整理的核心原則，長髮需妥善綁束。
~只要頭髮保持清潔，散亂或遮擋臉部並不會影響專業感#散亂的頭髮會影響專業形象，規範要求避免遮擋視線。
~值勤期間只需在開班前檢查一次髮型，之後不需再維持#規範要求值勤期間應隨時維持清潔與整齊。
~髮型規範僅針對女性夥伴，男性夥伴不受此限#髮型與個人清潔標準適用於全體夥伴，不分性別。
}

::OP-GS-01-Q04::夥伴在值勤過程中，因協助客人搬運大量行李導致汗流浹背、儀容略顯凌亂。此時該如何處置最適當？ {
~因為是辛勤工作的表現，即使儀容不整也應留在崗位繼續服務#值勤中若因特殊狀況造成儀容失整，應儘速整理而非維持不整的狀態。
~繼續服務直到下班前再進行整理即可#延遲整理會影響飯店的服務形象，應在發現失整後儘速處理。
=應儘速前往整理，確保儀容恢復標準後再回到崗位#規範明確要求因勞動導致儀容失整時應立即調整，以維持服務品質。
~直接請主管幫忙整理，避免離開服務崗位#個人儀容應由夥伴自行負責整理，主管職責是檢查與督導。
~不需要理會儀容，只要服務熱誠即可彌補外觀的不足#專業形象與服務態度相輔相成，規範強調必須維持標準儀容。
}

::OP-GS-01-Q05::關於值勤時的飾品配戴與妝容規範，下列哪一項敘述是錯誤的？ {
~飾品配戴應以不影響服務動作與安全為原則#這是規範中明確要求的原則，旨在確保工作效率與安全。
=香水味道若能讓客人遠遠聞到，更能展現飯店的高級感#規範特別註明香水味道不可過濃，過濃的氣味可能造成客人不適。
~妝容應以自然、乾淨、精神佳為基本訴求#自然乾淨的妝容是符合專業形象要求的標準。
~指甲應修剪整齊，且不可影響到工作的執行#指甲長度與整潔度是個人清潔與飾品規範中的重點。
~不可配戴體積過大或容易掉落的飾品#這是為了避免影響服務品質與排除安全疑慮的具體規定。
}

::OP-GS-01-Q06::當主管在巡檢時對夥伴說：「您的服裝需要再整理一下，麻煩您先調整完成後再回到櫃檯」。這表示夥伴應執行什麼後續動作？ {

=應立即調整儀容，待符合標準後再回到崗位，避免重複被提醒#規範要求未達標時應立即調整，若提醒後仍未改善將依內部管理規範處理。
~不需要做任何調整，因為這只是主管的個人主觀意見#主管的抽查是基於統一的標準指導書，夥伴必須遵循。
~只需要整理主管指出的部分，其他部位不需重新檢查#被提醒後應全面重新檢視，確保整體符合值勤標準。

::OP-GS-01-Q07::在制服維護方面，如果夥伴發現制服的拉鍊壞了或是扣子掉落，該如何處理？ {
~用別針暫時固定，繼續穿到報廢為止#別針固定不符合服裝整齊的要求，且制服需保持完整。
~若破損不明顯，可不需理會繼續穿著#制服扣子、拉鍊等細節必須保持完整，以維持專業感。
=應立即回報並申請更換，確保穿著完整性#規範明確規定制服若有破損（如拉鍊、扣子問題）需立即回報更換。
~自行去外面購買類似款式的衣物代替#私自更換非規定款式違反了制服統一穿著的規範。
~等到年度發放新制服時再進行更換#破損應「立即」回報，不可長期穿著損壞的制服。
}

::OP-GS-01-Q08::下列關於個人衛生與整體清潔的執行說明，哪一項符合標準？ {
~手部清潔僅需在處理食物前進行，一般櫃檯服務不需注意#規範要求保持口氣、手部與整體的全面清潔，不限特定職務內容。
~只要噴了重口味的香水，就不需要注意口氣清新#香水過濃與口氣不佳皆不符合衛生標準。
=應隨時保持口氣清新與手部清潔，維持良好的近距離服務品質#個人清潔是服務業專業形象的重要組成部分，直接影響客人感受。
~手部指甲只要塗上指甲油，即使長度過長也可以接受#指甲長度應以不影響工作為原則，過長並不符合規定。
~整體清潔僅指制服乾淨，不包含身體散發的異味#個人衛生標準涵蓋了整體清潔，包含口氣、手部與汗味等細節。
}

::OP-GS-01-Q09::夥伴在選擇上班配戴的飾品時，哪一項最符合規範中的「簡單、不影響服務」原則？ {

~配戴會發出清脆聲響的多個手鐲#發出聲響或過多的飾品不符合簡潔專業的原則。
=選擇設計簡潔、且在搬運物品時不會勾到衣物的飾品#飾品應以簡單、簡潔且不干擾服務動作為首要原則。
~為了展現時尚感，左右手各配戴三枚大型戒指#過多的飾品會顯得雜亂，且大型戒指可能影響手部作業。
~只要自己覺得美觀，任何形式的飾品皆可配戴#飾品選擇必須遵循館內的專業與簡潔標準，而非僅憑個人喜好。
}

~主管的檢查結果應由全體員工投票決定是否需要調整#標準指導書中賦予主管檢查與督導的權責。
