$CATEGORY: OP-FO-23 非住宿旅客參訪

::OP-FO-23-Q01::當一位訪客進入大廳，櫃檯夥伴的首要確認動作應為何？ {
~直接引導至沙發區稍坐並提供飲料#雖然提供服務很重要，但首要步驟應是釐清訪客身份與來意，以決定後續適用的作業流程。
=詢問該訪客是否為住宿旅客#根據流程1-1，第一步必須先確認來客是否為住宿旅客，以區分後續是執行入住手續或參訪流程。
~請訪客留下大名與電話以便登記#登記資料是針對非住宿旅客參訪的具體執行細節，並非對所有進店客人的首要確認動作。
~通知值班主管有客人來訪#通知主管通常是在確認訪客有特殊需求或背景（如熟人介紹）後的步驟，並非第一時間的反應。
~確認交接事項中有無今日預約#確認交接事項是針對「非住宿旅客」且「自稱為熟人介紹」時的下一步骤，需先確認身分後再進行。
}

::OP-FO-23-Q02::若訪客表示是由公司高層（如 SEVEN 或 KEVIN）介紹來參訪，櫃檯夥伴應優先執行哪項查核？ {
~立即致電該高層確認#直接致電高層可能干擾其工作，應先查閱內部已有的資訊紀錄。
~請訪客於一樓大廳稍坐，並直接讓其上樓#規範中明確規定切勿直接讓顧客上樓，必須經過身份確認與主管許可。
=先確認交接事項中是否有紀錄今日預約參訪#流程1-2-1規定應先確認交接事項有無預約，這是確保資訊對接與維持現場秩序的第一步。
~要求訪客出示與高層的通訊記錄#流程中並無要求查驗私人通訊紀錄，夥伴應透過內部交接與主管確認來查證。
~請主管帶領該顧客進行全館瀏覽介紹#全館瀏覽介紹的前提是主管確認認識該顧客且情況允許，並非在尚未核對資訊前就執行。
}

::OP-FO-23-Q03::若交接事項內無預約紀錄，且訪客自稱為熟人介紹時，下列哪項處置是錯誤的？ {
~請顧客於一樓大廳稍坐#這符合流程1-2-2的規定，即在確認身份期間應安頓客人在公共區域等待。
~向當班主管或 Kevin 告知有該組顧客欲參訪#當交接事項查無紀錄時，主動向上級呈報確認是標準的應變流程。
~向顧客留下大名及手機號碼#獲取訪客基本聯繫資料有助於身分核對與後續追蹤，是正確的執行步驟。
=為節省訪客時間，先帶領訪客上樓參觀#流程1-2-2特別註記「切勿直接讓顧客上樓」，避免在身份未明前造成安全或管理上的疑慮。
~若主管確認不認識該顧客，則轉為執行一般路過民眾的流程#根據1-2-4，若主管確認不認識，應參照1-3-2說明，即視為一般民眾處理。
}

::OP-FO-23-Q04::一位路過民眾在下午 2 點表示想要參觀旅館，櫃檯夥伴應如何安排登記？ {
~要求參訪的所有人員皆需留下姓名與電話#流程1-3-1規定若為兩人以上參觀，僅需留一位作為代表即可。
=請一位代表登記留下姓名與連絡電話#這是針對多數民眾參訪時，在效率與資料蒐集間取得平衡的標準作法。
~因為是路過民眾，不需登記即可參觀#非住宿旅客參訪皆需進行登記，以利管理並留意顧客動向。
~要求訪客提供身份證件以供影印留存#流程僅要求姓名與電話，過度蒐集個資不符合目前的作業指導書規定。
~請主管先過目確認是否允許參觀#一般民眾參訪若在規定時間內，依流程登記後即可參觀一樓或依指引進行，不需逐一由主管預審。
}

::OP-FO-23-Q05::關於「一般路過民眾」的參訪時間限制，下列敘述何者正確？ {
~全天候 24 小時皆可進入大廳與客房層參觀#為維護住客隱私與安全，參訪時間有明確限制，且非參訪時間禁止上樓。
~參訪時間為早上 9 點至下午 5 點#此時間設定不符合作業指導書 1-3-2 中規定的 11\:00 至 19\:00 時段。
=晚上 8 點來訪的民眾，僅能參觀一樓大廳#因為 20\:00 已超過參訪時間（11\:00-19\:00），依規定禁止引導上樓，僅限一樓區域。
~非參訪時間來訪者，需由主管特許方可參觀客房#流程明確規定非參訪時間請勿讓顧客上樓，這是維護管理標準的硬性規定。
~只要有登記，任何時間皆可參觀各樓層#登記僅是參訪的條件之一，時間限制與空間權限仍需依規定執行。
}

::OP-FO-23-Q06::當訪客明確表達「想要參觀房間並且想要訂房」時，正確的處理流程是？ {
~請櫃檯夥伴自行帶領顧客前往參觀空房#櫃檯夥伴需留守崗位，且流程規定應由主管負責帶領房間參觀。
~請訪客自行搭電梯前往標示為展示間的房號#這違反了「切勿直接讓顧客上樓」的安全原則，且無法提供專業的訂房說明。
=通知值班主管，請值班主管帶領顧客參觀空房#根據流程1-4，涉及參觀房間欲訂房的情況，應交由值班主管引導，以確保服務品質。
~請顧客先在訂房系統下單後再安排參觀#實務上通常是先參觀滿意後才下單，流程並無限制須先訂房才能參觀空房。
~告知目前無空房，無法提供參觀#即使無空房，也應依流程通報主管處理，而非直接拒絕潛在客戶的參觀需求。
}

::OP-FO-23-Q07::對於「參觀展覽」的民眾，作業項目中特別強調的動作是什麼？ {
~需檢查其行李有無危險物品#雖然安全重要，但目前的作業指導書中並未規定要進行行李檢查。
~必須支付參觀清潔費#資料中並未提及收費標準，應以流程中的簽到與引導為準。
=按照展覽簽到方法請參觀者確實簽到#依據作業項目 2「參觀展覽」，確實簽到是掌握參觀人數與對象的關鍵管控點。
~強制要求訪客在社交媒體打卡#打卡屬於行銷建議，並非作業指導書中規定的必要作業流程。
~引導至二樓以上的客房走廊參觀#展覽路線應依設計規劃，且需符合參訪空間的管控，不一定包含客房區域。
}

::OP-FO-23-Q08::若公司高層或熟人介紹的顧客到訪，且身份經主管確認後，在什麼樣的條件下可以進行「全館瀏覽介紹」？ {
