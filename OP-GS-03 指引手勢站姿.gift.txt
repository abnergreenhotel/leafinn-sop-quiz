$CATEGORY: OP-GS-03 指引手勢站姿

::OP-GS-03-Q01::在進行「基本站姿」服務時，下列哪一項符合綠宿行旅的標準作業規範？ {
~雙手自然交握於身後，以展現專業的背影。#規範要求雙手應自然交握於身前，而非身後。
~為了節省體力，在沒有顧客時可微靠櫃台站立。#標準規定雙腳自然站穩，不可倚靠櫃台或牆面。
=站立時抬頭、挺胸、雙肩放鬆，雙腳自然站穩。#這是規範中明確定義的基本站姿執行說明。
~為表示親近，站立時可將單手插入側邊口袋。#插口袋屬於禁止行為，會給人不夠專業的印象。
~雙手抱胸站立，以利隨時觀察場域狀況。#雙手抱胸屬於不禮貌的負面站姿，是被明令禁止的。
}

::OP-GS-03-Q02::當夥伴需要遞交房卡、發票或名片給顧客時，下列操作流程何者正確？ {
~為了效率，可以直接將物品放在櫃台上推向顧客。#規範禁止將物品直接推向顧客，這會顯得缺乏尊重。
=以右手遞交物品，左手需在旁輔助並將文字面朝向對方。#雙手遞物是核心標準，且應先調整物品方向以利顧客閱讀。
~單手快速拋遞，同時口頭告知顧客內容。#單手拋遞是禁忌動作，會嚴重影響服務品質。
~由下方遞出物品，不需口頭說明以維持環境安靜。#遞交時應同時口頭說明內容，以確保資訊傳遞準確。
~雙手交叉遞物，以表示熱情款待。#交叉遞物並非標準動作，應採取自然的雙手或主副手遞物。
}

::OP-GS-03-Q03::顧客詢問餐廳方向時，夥伴應如何正確執行「指引手勢」？ {
~單指指向遠方，明確標示餐廳的位置。#不可以單指指人、指路或指物，這是基本的服務禮儀規範。
=五指併攏、掌心自然打開，以整個手掌進行引導。#使用整個手掌且五指併攏是正確的指引方式，代表尊重的引導。
~僅以眼神示意方向，避免大幅度肢體動作。#標準流程要求使用手掌引導，眼神示意不足以構成完整指引。
~右手手背朝上，平攤指向目標方位。#手背朝上通常帶有壓迫感，應以掌心自然打開的方式引導。
~使用文件夾或筆作為輔助指標進行路徑指引。#應使用手掌引導，不應使用額外物品指向路徑。
}

::OP-GS-03-Q04::在帶領顧客前往客房或設施的「同行與帶看」過程中，哪一項執行說明是錯誤的？ {
~行進速度應配合顧客，不可走得太快。#配合顧客步調是正確的標準作業流程。
~應走在顧客的正後方，確保顧客視野開闊。#應走在顧客的「前側方」而非正後方，以便引導路徑。
~到達定位後，應以手勢搭配口頭完成最後指引。#這是標準的結尾動作，確保引導完整到位。
~遇到轉彎、門口或高低差處時，需提前提醒顧客。#提前提醒是為了確保顧客安全，屬於正確的流程細節。
=為了節省時間，帶位時應快步走在顧客前方 5 公尺處。#應走在顧客的前側方並保持適當距離，且速度需配合顧客，而非過快。
}

::OP-GS-03-Q05::若您正在櫃台內操作電腦，當看見顧客主動接近時，第一時間應如何反應？ {
~繼續完成手邊工作，待結束後再向顧客問好。#不可忽視顧客，應立即調整姿態以示歡迎。
~維持坐姿或側向姿勢，以免影響電腦作業進度。#應起身或調整為正面服務姿態，這是對顧客的基本尊重。
=立即起身或調整為正面服務姿態。#根據櫃台服務標準，顧客接近時應即刻切換為正向迎客模式。
~一邊操作電腦一邊口頭回應「您好」。#僅口頭回應而不調整姿勢，會給人冷淡、不專業的感受。
~直接離位去領取可能需要的資料。#不應在未與顧客互動前就擅自離位，且離位需先說明。
}

::OP-GS-03-Q06::關於櫃台服務中的「離位」標準，下列敘述何者最符合規範？ {
=若需離開拿取資料，應先向顧客說明：「請您稍等，我這邊幫您拿取資料。」#離位前先說明理由並告知顧客等待，是標準話術的一部分。
~只要顧客正在填寫資料，夥伴可以不經說明直接離位處理其他事務。#即使顧客在忙，離位仍應告知，以免顧客抬頭找不到人。
~返回櫃台後，只要能接續之前的工作即可，不需特別注意站姿。#返回後應重新以正面站姿服務顧客。
~離位時間若不超過 10 秒，則不需要對顧客做任何說明。#無論時間長短，為了避免顧客疑惑，離位前皆應說明。
~為了保持禮貌，離位時應背對顧客緩慢行走。#離位應講求效率且先與顧客說明，並非強調背對顧客的方式。
}

::OP-GS-03-Q07::情境題：當顧客帶著年幼的孩子並推著推車進入大廳，夥伴準備帶位至電梯口，哪種做法最適當？ {
~先行快速走到電梯按鈕處等待，展現高效率。#行進速度需配合顧客，特別是攜帶幼兒與推車的顧客。
=走在推車的前側方，並在進入電梯時提醒：「前方有門檻，請您留意腳下。」#此做法兼顧了正確位置、速度配合以及危險處的提前提醒。
~在後方指引電梯方向，由顧客自行推車進入。#帶位應走在顧客的前側方，主動引導而非僅在後方指路。
~伸出手指指向電梯，並說：「電梯在那邊，請自便。」#這違反了不可單指指路以及應主動帶位的服務標準。
~保持 3 公尺以上的長距離，以免干擾顧客推車。#應保持「適當距離」而非過遠，否則無法有效引導與溝通。
}

::OP-GS-03-Q08::在「雙手遞物」的服務標準中，哪一個細節能讓顧客感受到「易讀性」的貼心？ {
~遞交時保持沉默，讓顧客專心看物。#應同時口頭說明遞交物品內容，以增加資訊準確度。
~先將房卡、發票等物品的文字方向朝向自己檢查清楚。#遞交前應先將文字方向朝向「對方」，而非朝向自己。
=遞交前先將文字方向朝向對方，方便對方直接閱讀。#這是為了便利顧客操作與確認，是遞物流程的重要細節。
~使用單手將物品轉向後，再遞給顧客。#應以雙手或右手遞、左手輔助，且不應在遞交過程中才草率轉向。
~將物品重疊在一起，以減少佔用的空間。#物品應清楚展示，重疊可能導致文字被遮蔽或遺漏。
}

::OP-GS-03-Q09::下列哪一句話最不符合「指引手勢」與「同行帶位」的標準話術？ {
~「您好，這邊為您服務。」#這是基本站姿服務時的正確開場話術。
~「這是您的房卡與發票，跟您一併交付，請您收好。」#這是遞物時標準的內容說明話術。
=「餐廳就在那邊，你自己走過去就看到了。」#此話術缺乏服務熱忱，且長距離指引應主動帶位或搭配手勢引導。
~「電梯在這個方向，我帶您過去。」#這是指引方向並主動帶位的正確標準話術。
~「前方有階梯，請您留意腳下。」#這是帶位過程中遇到地面落差時的標準提示話術。
}

::OP-GS-03-Q10::在櫃台服務中，關於「背對顧客」的行為，規範的態度是： {
~只要在處理公務，背對顧客是可被接受的專業展現。#即使處理公務，也不應無故或長時間背對顧客。
=不可背對顧客長時間操作而無任何說明。#這是櫃台服務站姿與離位標準中的重要注意事項。
~背對顧客操作電腦比正面操作更有效率。#效率並非藉口，顧客感受到的服務態度更為重要。
~背對顧客時應保持微笑，以維持氣氛。#背對時顧客無法看見微笑，此舉沒有實質服務意義。
~為了隱私安全，操作電腦時應一律背對顧客。#應在兼顧作業與服務姿態的前提下進行，不可長時間無說明背對。
}
