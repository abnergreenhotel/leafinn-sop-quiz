$CATEGORY: OP-FO-10 禁菸標準作業流程

::OP-FO-10-Q01::情境：當夥伴在進行訂房流程與客⼈確認訂單時，根據 SOP 的規定，最核心的告知內容為何？ {
~提醒房客入住時需攜帶身分證件以利核對#雖然這是入住的常規流程，但並非禁菸 SOP 中特別強調的訂房增加告知內容。
=主動說明本旅館為禁菸旅館，且全館範圍內皆禁止吸菸#SOP 明確規定在訂房流程中應增加告知為禁菸旅館，以提前避免後續的消費糾紛。
~告知房客客房內提供免費的除臭設備供其使用#提供除臭設備可能被誤解為默許吸菸，且 SOP 的重點在於禁止吸菸而非事後處理。
~僅需在房客詢問時，再行解釋飯店的禁菸規範#SOP 要求主動在訂單確認時告知，而非被動回應，以落實風險預控。
~告知房客飯店周邊可以合法吸菸的區域位置#SOP 的首要任務是確保房客了解館內禁菸規定，周邊環境告知並非此流程的重點。
}

::OP-FO-10-Q02::為什麼要求房客在入住登記卡（R卡）上簽名，對於後續收取清潔費具有關鍵的法律意義？ {
~僅為了符合政府對旅館業管理規則的行政登記要求#行政登記是基本要求，但在禁菸 SOP 中，簽名的重點在於確立雙方的民事契約關係。
=表示房客已同意入住規範，使清潔費的收取具備契約成立的法律效力#SOP 註記只要客人在 R 卡簽名即表示契約成立，收取損害賠償費在法律上是站得住腳的。
~簽名代表房客承諾若在房內抽菸，願意接受政府開立的罰鍰#政府罰鍰是由公家機關依法執行，不需透過飯店契約授權，飯店契約是針對「損害賠償費」。
~是為了確認房客身分，以便在發現抽菸時能向衛生局舉報個人資料#舉報是後續手段，但簽名的核心目的是保障飯店收取清潔費的權利，而非僅為了舉報。
~簽名僅是服務流程的一部分，不具備任何實質的法律约束力#這是錯誤的觀念，簽名具有法律上的合意性，是保障飯店權益的重要依據。
}

::OP-FO-10-Q03::當服務專員發現房客在禁菸區域吸菸，在向其說明費用時，下列哪一種用詞是 SOP 規定正確且具備法律正當性的？ {
~根據規定，我們必須對您在房內抽菸的行為「罰款」3000元#SOP 特別提醒不能說「罰錢」，因為罰錢是公家單位才能做的權力。
=您需要支付本館因除味所需的「損害賠償清潔費」3000元#SOP 明確指示必須說是收取「損害賠償費」或「清潔費」，這屬於飯店與客人間的民事契約範疇。
~我們會將此筆費用作為交給衛生局的「違規規費」#飯店無權代收政府規費，且清潔費是留給飯店處理後續房況的成本。
~這筆費用是您違反菸害防制法而產生的「行政處分金」#行政處分金是由政府機關開立，飯店自行收取的費用不能以此名義命名。
~我們將向您收取等同於房價的「違約罰金」#雖然具備違約性質，但 SOP 建議使用「損害賠償費」或「清潔費」更符合實際作業名義。
}

::OP-FO-10-Q04::根據《菸害防制法》之相關規定，若旅客在飯店客房內吸菸，除了飯店收取的清潔費外，政府最高可處以多少額度的罰鍰？ {
~新臺幣 3,000 元#3,000 元是本館規定的清潔費金額，非政府罰鍰的法定上限。
~新臺幣 6,000 元#6,000 元是參考案例中五星級飯店收取的清潔費，並非政府法定罰鍰額度。
=新臺幣 10,000 元#根據法規與公告內容，違反菸害防制法最高可處新臺幣壹萬元（10,000元）罰鍰。
~新臺幣 20,000 元#此金額高於法規規範的上限。
~新臺幣 5,000 元#雖然此金額在罰鍰區間內，但非法律規定的上限額度。
}

::OP-FO-10-Q05::情境：夥伴在巡檢樓層時聞到客房散發出明顯煙味。根據 SOP，此時應採取的正確行動程序為何？ {
~為了不打擾客人休息，等到客人退房時再一併處理#延遲處理可能導致客房損害擴大且證據消失，應及時遏止。
=立即致電該房告知本館禁菸，並提醒可能產生的清潔費與政府罰鍰#SOP 要求入住時及時發現煙味應致電告知，以遏止客人繼續抽菸並為收費做預告。
~直接通報衛生局，請公務人員到場直接開罰#通常應先由飯店內部處理，情節重大或房客拒不配合時才考慮致電衛生局。
~在該房門口張貼禁菸警告標語，不與房客直接接觸#這無法達到及時溝通與遏止的效果，且不符合 SOP 規定的溝通流程。
~直接斷掉該客房的空調系統以防止菸味擴散#這可能引發更大的消費糾紛且非 SOP 規定的標準處置方式。
}

::OP-FO-10-Q06::情境：房務人員在房客退房後進入打掃，發現房內有菸味及菸灰。此時房務人員應優先執行的動作是什麼？ {
~先將垃圾倒掉，並迅速將床單撤除送洗以消除異味#這會破壞現場證據，增加後續向房客求償的困難度。
=保持現場原狀，拍照存證並立即告知櫃台聯繫客人#SOP 明確規定房務須及時告知櫃台並拍照存證，以避免客人後續不認帳。
~直接在客人資料備註為黑名單，不需要特別留存證據#缺乏證據的備註容易產生爭議，且無法正當收取清潔費。
~立即噴灑大量化學香水覆蓋菸味，以免影響下一組房客#這不是標準的除味程序，且掩蓋菸味並非當務之急，留證與標準除味才是重點。
~等到櫃台與客人完成收費確認後，再開始檢查是否有菸味#應是房務先發現並回報，櫃台才能據此與客人溝通收費。
}

::OP-FO-10-Q07::針對確認有抽菸情事的客房，SOP 規定房務人員在什麼情況下才可進入整房？ {
~噴灑除臭噴霧後立即進入#單純除臭噴霧無法徹底處理殘留菸害，且非規定的等待時間。
~房務主管親自視察确认沒有味道後#雖然需要檢查，但 SOP 有明確規定的儀器處理與等待時數。
=房務打臭氧後 30 分鐘才進去整房#SOP 流程第 4 點明確指出：房務打臭氧後 30 分才進去整房，以確保空氣品質與作業安全。
~開啟窗戶通風 2 小時後即可整房#通風僅是輔助，標準流程強調的是使用臭氧機處理。
~只要客人付清清潔費後，即可立即進入整房#整房流程是基於環境清潔與健康的專業判斷，與付費時間點無直接技術關聯。
}

::OP-FO-10-Q08::若房客在房內抽菸不慎導致地毯燒焦或設備毀損，除了基本的清潔費外，根據 SOP 應如何處理？ {
~因已收取清潔費，設備毀損由飯店自行吸收成本#清潔費僅針對空氣與織品的除味清理，不涵蓋硬體毀損。
~將清潔費金額調高為兩倍即可，不再另行求償#賠償金額應依實際受損價值計算，而非隨意翻倍清潔費。
=除收取清潔費之外，並須按造價賠償損毀之設備#公告中明確註明：若造成設備毀損，除收清潔費之外並須造價賠償。
~要求房客自行購買相同的設備更換即可#飯店設備通常有統一採購規格，且求償程序應由飯店主導。
~直接沒收房客的所有住宿押金作為補償#補償應具備具體的損害對價關係，不能無條件沒收押金。
}

::OP-FO-10-Q09::當服務人員需要與房客溝通抽菸違規收費時，SOP 對於「轉告態度」有何具體要求？ {
~必須語氣強硬，展現捍衛禁菸政策的決心以嚇阻房客#過於強硬的語氣容易引發不必要的肢體或言語衝突。
~採取公事公辦的冷淡態度，僅告知收費金額#這不符合飯店業的服務精神。
=口氣需婉轉注意禮貌，避免引起更嚴重的糾紛#SOP 控制點提醒：轉告時，服務員口氣需婉轉注意禮貌，以確保溝通效力同時維持服務品質。
~應先向房客致歉，感謝其入住後再委婉地請求其捐款#收費是契約義務而非請求捐款，語氣婉轉不代表需要放低姿態。
~無需特別注意口氣，直接依照規定收費即可#SOP 特別強調口氣，就是為了在執行規範時仍能兼顧顧客關係管理。
}

::OP-FO-10-Q10::針對情節重大的抽菸房客，除了收取清潔費外，SOP 提到的最高處理手段為何？ {
~沒收房客的行李直到其繳納政府罰鍰#飯店無權留置房客私人物品以代收政府罰款。
=由主管聯繫房客收取清潔費並致電衛生局舉發#SOP 的後續處理步驟包括主管聯繫收費，若情節重大則致電衛生局舉發。
~直接報警將房客以公共危險罪移送法辦#抽菸通常涉及菸害防制法，除非引發火災，否則不直接涉及公共危險罪。
~在社群媒體上公告該房客的個人資料以示懲戒#這會違反個人資料保護法並引發嚴重的法律風險。
~永久禁止該房客及其家屬入住所有連鎖飯店#SOP 僅提到列入資料備註提醒，而非涉及親屬的永久禁令。
}
