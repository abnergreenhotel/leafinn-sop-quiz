$CATEGORY: OP-OB-03 規範認同

::OP-OB-03-Q01::在準備新進夥伴的報到文件時，若發現公司制度剛進行更新，下列哪一項做法符合標準作業程序？ {
~為求效率，先使用舊版文件簽署，待日後再補簽新版#作業規範明確禁止使用舊版文件，以避免資訊落差或法律風險。
~僅需口頭告知制度已有更新，文件則沿用現有版本#口頭說明不能取代正式文件的更新，必須提供最新版本的書面資料。
=提供最新版本的員工手冊，並一併檢附最新版的公告說明#當制度更新時，確保文件版本為最新並提供相關公告是準備工作的核心重點。
~請新進夥伴自行至內部網路下載最新版本，不需準備紙本#SOP 要求應先備妥相關簽署文件與手冊，而非要求夥伴自行準備。
~直接在舊版文件上用手寫修改更新內容並請夥伴在旁蓋章#規範要求確認文件版本為最新，手寫修改舊版並非正式且合規的處理方式。
}

::OP-OB-03-Q02::在進行「規範內容說明」階段時，下列哪一項作業邏輯是正確的？ {
~為節省時間，應直接翻到最後一頁請夥伴先簽名再開始說明#規範明確要求不可僅要求簽名，必須先完成重點口頭說明。
=說明時需涵蓋出勤、請假、服儀、工作紀律與獎懲制度等重點#這是規範中明確列出的說明重點，確保夥伴了解其基本權利與義務。
~只需針對員工手冊內容進行說明，勞動條件應由夥伴自行閱讀#規範要求需說明勞動條件、試用期、排班與保密義務，不可省略。
~說明過程中若夥伴有疑問，應要求其自行閱讀文件尋找答案#規範要求對疑問應當場回覆或記錄後轉主管補充，以確保雙方認知一致。
~若負責人員不清楚排班細節，可憑經驗先口頭承諾夥伴#涉及排班、薪資等敏感疑義，應由主管或人資補充說明，不可自行承諾。

::OP-OB-03-Q03::當新進夥伴對「薪資計算」或「法規疑義」提出深度詢問，而負責報到的夥伴不確定正確答案時，應如何處理？ {
~請夥伴先完成簽署，等正式上線後再詢問同仁#規範要求確認清楚後再簽署，不可在有疑慮的情況下要求簽名。
~根據過去的經驗給予一個概括性的解釋，以免延誤進度#不確定的解釋可能造成後續爭議，應由專業窗口進行說明。
~告知夥伴這些內容不影響報到，可以略過不談#這違反了規範中「確認清楚再簽署」的原則。
=立即請主管或人資窗口協助補充說明，或記錄後回覆#涉及薪資與法規等專業範疇，由主管或人資介入是確保資訊準確的標準程序。
~要求夥伴自行撥打人資部電話或寫郵件詢問#負責報到的執行人員應主動協助轉介或記錄，而非將責任推回給新進人員。
}

::OP-OB-03-Q04::關於「文件閱讀與簽署」的執行細節，下列敘述何者錯誤？ {
~必須請新進夥伴逐份閱讀文件內容#這是正確的做法，規範要求夥伴必須逐份閱讀以確保知悉內容。
~應引導夥伴於簽名欄、日期欄與必要欄位完成填寫#這是正確的做法，確保文件的法律效力與完整性。
=若夥伴因故無法到場，可由其家人或代理人代為簽署#規範嚴禁代簽，報到文件必須由新進人員本人親自簽署。
~需確認附件頁及同意書均已完整填寫#這是正確的做法，避免文件缺漏造成報到不完整。
~若發現漏簽日期，應立即請夥伴現場補正#這是正確的做法，規範要求發現漏項需立即補正。
}

::OP-OB-03-Q05::在「簽署後複核」階段，負責人員應執行的關鍵動作為何？ {
~確認文件齊備後，將所有人的文件混合放置於收件盒#規範要求應將文件依「人員姓名」分類整理，而非隨機混合。
=檢查是否有漏簽或漏附文件，確保知悉與同意書均齊備#逐頁檢查是複核的核心任務，確保文件流程的完整度。
~先將文件歸檔至檔案室，等到有空時再進行逐頁檢查#規範要求文件複核完成前不得直接歸檔。
~只要確認有簽名即可，日期不完整可以事後補填#漏日期屬於不完整文件，需立即補正，不可留待事後。
~複核時若發現缺件，可於一週內通知夥伴補件即可#規範要求若有缺件，需當日完成補件。
}

::OP-OB-03-Q06::完成所有簽署與複核後，關於「建檔與交接」的規範，下列何者正確？ {
~簽署完成的文件應暫放在櫃檯，以便其他同仁隨時查閱#個人文件需妥善保管，嚴禁任意放置於公共區域以保護個資。
~只要文件收齊，不論有無簽名，均視為報到流程已完成#規範明定若新進夥伴尚未完成簽署，不可視為報到流程完成。
=如有待補件項目，應於交接紀錄上註明追蹤期限#這是確保後續進度能被追蹤且如期完成的重要步驟。
~建檔完成後，不需在任何名單上做註記，僅需存放文件#應於報到紀錄或人員名單上註記完成日期，以便行政控管。
~文件應交由清潔人員協助送往行政辦公室#文件應交由「主管或指定窗口」建檔，且過程需妥善保管。
}

::OP-OB-03-Q07::為了讓新進夥伴更快速找到需要簽名的位置，報到準備時建議採取什麼輔助動作？ {
~在每一頁需要簽名的位置先蓋好主管的印章#這不符合規範，且可能誤導夥伴，且規範並未要求蓋主管章於簽名處。
=將需要簽署的頁面以便條紙或標籤標示清楚#這是 SOP 1.0 階段明確建議的動作，旨在提高簽署效率並減少遺漏。
~先用螢光筆在簽名欄塗滿顏色，增加醒目度#雖然能增加醒目度，但可能破壞正式文件的整潔與正式度，規範建議使用標籤。
~將所有需簽名的頁面摺角標示#摺角並非正式的文件處理方式，且規範建議使用便條或標籤。
~直接將簽署文件單獨拆出，其餘手冊內容不需提供#報到必須包含手冊說明，不可僅提供簽名頁。
}

::OP-OB-03-Q08::在對新進夥伴進行話術溝通時，下列哪一段對話內容最符合 3.0「文件閱讀與簽署」階段的精神？ {
~「這幾份請直接幫我簽名，內容之後再慢慢看。」#這違反了應先請夥伴確認內容再簽署的要求。
=「這邊再請您確認文件內容，若沒有問題，請於簽名欄與日期欄完成簽署，謝謝您。」#這段話術正確引導夥伴進行內容確認及完整簽署（包含簽名與日期）。
~「如果您很趕時間，我可以幫您代簽，您只要確認一下就好。」#嚴禁代簽是文件的基本安全規範。
~「簽名就好，日期我們會統一人資幫您補上。」#日期欄位亦屬於需完成簽署的一部分，不可由他人代勞。
~「只要簽這份就好，其他的手冊您可以帶回家慢慢參考不急著簽。」#規範要求確認相關規範（如獎懲制度知悉）均已齊備，不可漏簽。

::OP-OB-03-Q09::執行人員在完成複核後，若發現夥伴遺漏了「獎懲制度知悉」文件的簽署，應如何處理？ {
~因為只是獎懲制度，不影響報到，可以略過#所有規範文件均需齊備，不可選擇性忽略。
~直接在交接紀錄上記錄「未完成」，並讓流程繼續往下走#發現缺漏應立即補正，且未完成簽署不可視為報到完成。
=必須於發現的當日通知夥伴完成補件#規範要求若有缺件，需當日完成補件。
~等到試用期滿前再一併補簽即可#報到文件的完整性是到職的第一步，不可無限期拖延。
~請同組的其他夥伴幫忙代簽以補齊文件#代簽是嚴格禁止的行為。
}
