$CATEGORY: OP-FO-31 基礎房務實習

::OP-FO-31-Q01::當房務員抵達客房門口準備執行清掃作業時，正確的敲門與告知身分流程應為何？ {
=連續敲門 3 下，每次間隔約 2 秒，並在敲門時同步口頭告知「Housekeeping」。#此做法符合 SOP 標準流程，確保給予客房內人員足夠的反應時間並清楚辨識來者身分。
~快速連續敲門 5 下以確保客人聽見，隨即按鈴 1 次後直接開門。#快速敲門可能造成客人壓力，且未依規定間隔與告知身分即進門會侵犯客人隱私。
~先按門鈴 3 次，若無回應則直接使用工作卡進門告知身分。#SOP 規定門鈴應按 2 次且每次間隔 2 秒，且進門前須經過多次確認而非直接進入。
~敲門一次並大聲喊叫房號，等待 5 秒後若無動靜即開門。#僅敲門一次且等待時間不足 10 秒，可能導致正在休息或洗澡的客人來不及回應。
}

::OP-FO-31-Q02::在敲門並告知身分後，房務員發現房內沒有回應，此時最正確的下一步操作是什麼？ {
~立即使用工作卡（Master Key）開門進房清掃以節省時間。#直接進入可能會驚擾到反應較慢或在聽不到聲音處的客人。
=必須在門口等待約 10 至 20 秒，確保客人有充足時間應門。#這是為了預防房客正在洗澡、睡覺或因故無法立即回應的情境，體現對隱私的尊重。
~轉往下一間房清掃，半小時後再回來重新開始流程。#雖然可彈性調整順序，但在等待不足規定的時間前不應直接判斷為無人房。
~直接推開房門並大喊房號，確認裡面是否真的沒人。#未經二次確認且未依標準等待即推門是嚴重違反進房禮儀的行為。
}

::OP-FO-31-Q03::若客房門口顯示「請勿打擾（DND）」燈號，但該房是預計今日退房的房間（退掃房），房務員應如何處理？ {
~既然是退房日，可直接敲門詢問客人何時離開。#只要 DND 燈號亮起，嚴禁任何敲門或按鈴動作，以免干擾客人權益。
=先與櫃檯確認該房是否已辦理退房手續，而非自行敲門或按鈴。#這是處理 DND 房況與退房衝突時的標準查核程序，可避免誤闖仍有客人在內的房間。
~無視燈號，照常執行敲門 3 下與按鈴 2 次的進房流程。#無視 DND 燈號會導致客訴，且違反 SOP 中「切記勿去按門鈴或敲門」的規定。
~在門口大聲喊叫 Housekeeping，確認客人是否已經離開。#口頭喊話同樣會干擾到室內需要安靜的客人，應透過系統或櫃檯查詢房況。
}

::OP-FO-31-Q04::房務員在進入「續住房」放置補充備品時，若客人目前不在房間內，下列哪項做法符合 SOP 安全與禮儀規範？ {
~穿著工作鞋快速進入，將備品放下後關門離開並回報。#進入已入住的客房必須脫鞋以保持室內清潔，這是對客人的基本尊重。
=進入房間前必須脫鞋，且在房內放置備品期間應保持房門敞開。#脫鞋是為了維持客房地板乾淨，敞開門則是為了保護房務員並避免空間封閉引起的誤會。
~為了安全起見，應進入房間後反鎖房門，避免其他客人誤入。#房務員單獨在客房內作業時不應關門，以確保作業透明度與安全性。
~直接將備品掛在門外把手上，不進入房間以保障客人隱私。#SOP 要求的是進入放置備品，直接掛門口不符合該項作業的執行說明。
}

::OP-FO-31-Q05::關於布巾數量的紀錄與登記，下列哪一種操作方式最能降低數量出錯的風險？ {
~將整層樓所有房間拆下的布巾集中後一次清點並填寫表格。#集中清點容易導致房號與數量對不上的混亂，違反 SOP 逐間紀錄的原則。
~憑記憶在休息時間一次補填所有清掃過房間的布巾數量。#憑記憶填寫極易發生錯誤，導致後續布巾盤點不準確。
=每完成一間房的拆床與清點，立即在工作表上對應房號欄位確實填寫。#SOP 強調「拆床一間寫一間」，這能確保數據的即時性與準確性。
~只記錄缺少的布巾數量，數量正確的房間則不需填寫以加快速度。#每一間房都應確實記錄，不論數量是否異常，以便後續追蹤控管。
}

::OP-FO-31-Q06::在鋪床作業中，關於枕頭套的擺放與處理，下列敘述何者正確？ {
~枕頭套開口應朝向床外側，方便客人拉取更換。#開口朝外會露出枕芯，影響視覺美觀且不符合標準。
=將枕頭套開口朝向內側（兩枕互相面對），並立式擺放對靠床頭板。#這能隱藏枕套開口，維持床鋪整體視覺的整齊美觀。
~枕頭應平放在床墊上，開口一律朝向房門口方向。#開口方向應對向內側而非門口，且應以立式對靠床頭板擺放。
~為了塞枕頭方便，枕頭套的開口應保持上下對齊而非左右對齊。#SOP 未提及上下對齊，重點在於「開口朝內」與「互相面對」的視覺標準。
}

::OP-FO-31-Q07::房務員在進行「鋪床 sheet（床單）」作業時，為了提高效率並符合標準，應如何操作？ {
~先將床單揉成團後再慢慢拉平，以確保能覆蓋整個床墊。#將床單揉成團會產生大量皺褶，且拉平過程非常費時。
=先將床單攤開成一長條並對準中線，接著再向左右兩側攤開。#這是確保床單鋪設平衡且定位準確的最快方式，能有效對準床墊中心。
~直接將床單四個角塞入床墊下，最後再調整中間的平整度。#先塞角會導致中間無法對準中線，造成兩側長度不均。
~站在床腳處將床單整張甩開，讓它自然落在床墊上後再整理。#這種方式很難精準控制中線位置，且容易讓床單滑落到地面。
}

::OP-FO-31-Q08::在鋪被套（Duvet cover）時，有一個關鍵步驟是為了「防止客人使用時被芯與被套分離」，該動作為何？ {
~使用大頭針將四個角落固定在床墊上。#使用大頭針有安全疑慮，並非正確的旅館房務做法。
~在被子四個角用力下壓，利用摩擦力固定位置。#僅靠下壓無法在客人拉扯被子時維持固定，被芯仍會位移。
=將被角與被套內部的繩子打上活結固定。#打活結是為了固定被芯與被套，且選擇「活結」方便下次換洗時拆卸。
~將被套開口處用強力膠帶黏貼封口。#這種做法會損壞布巾且不美觀，不符合房務專業流程。
}

::OP-FO-31-Q09::若在進房檢查時，房務員發現房內竟有客人在（疑似櫃檯排房錯誤），第一時間該如何反應？ {
~若客人在睡覺，應悄悄關上門離開，當作沒發生過。#不論客人是否醒著，都必須回報主管做後續處理，否則會影響房況正確性。
=立即向客人致歉並離開，隨後回報給櫃檯與值班主管處理。#這是處理「誤開房門」的標準作業流程，確保客人情緒得到緩解且問題被紀錄。
~向客人索取身分證明文件，確認對方是否為登記房客。#查驗身分是櫃檯的權責，房務員第一時間應致歉並退出，避免與客人發生爭執。
~要求客人立即離開房間，告知此房為待掃狀態。#這對客人極度不禮貌，且排房錯誤可能是飯店方的疏失。
}

::OP-FO-31-Q10::關於鋪床作業的注意事項，SOP 提到可以利用哪種原理來減輕推動床墊的體力負擔？ {
~利用滑輪原理，在床墊下方塗抹潤滑劑以便推動。#這是不切實際的做法，且會弄髒地毯與床墊。
=利用槓桿原理，藉由床頭板將床推出以便後續鋪床。#這是 SOP 建議的技巧，能讓房務員在狹窄空間更有效率地進行鋪床作業。
~利用慣性原理，快速撞擊床墊使其位移。#暴力撞擊會導致家具損壞，並非專業的作業技巧。
~利用大氣壓力原理，將床墊稍微抬高後即可輕鬆移動。#房務作業中無法實際應用大氣壓力來移動重物，此說法不成立。
}
