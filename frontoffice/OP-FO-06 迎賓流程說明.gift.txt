$CATEGORY: OP-FO-06 迎賓流程說明

::OP-FO-06-Q01::當客人在櫃檯辦理完業務準備離開時，根據迎賓流程，櫃檯人員應該在什麼時機點才可以轉身入座？ {
~確認客人已經轉身往門口走去時#即使客人已經轉身，若尚未完全離開，立即入座會給人服務終止太快的感覺。
=待客人完全離開現場後#這能確保服務的完整性，展現對客人的尊重直到最後一刻。
~完成電腦系統輸入並存檔後#行政作業應排在客人感受之後，不應為了存檔而先行轉身入座。
~下一組客人抵達櫃檯前#入座的時機是取決於前一位客人的離開，而非後一位客人的抵達。
}

::OP-FO-06-Q02::看到門口有車輛停靠時，夥伴上前協助開關車門後，應如何正確與貴賓進行初步溝通？ {
=「您好，請問是有訂房的貴賓嗎？」並接續詢問是否要放行李至櫃檯#這是標準的確認語法，能立即確認需求並主動提供行李協助。
~「您好，請先去停車，行李我幫您拿」#直接要求客人停車可能顯得突兀，應先確認身分並提供協助。
~「歡迎光臨，行李請交給我處理即可」#雖然禮貌但缺少了確認訂房身分的關鍵步驟。
~先觀察客人是否需要幫忙，若客人招手再上前詢問#流程要求「主動上前」，而非被動等待客人的示意。
}

::OP-FO-06-Q03::在雨天接待情境下，關於「放置小心地滑告示牌」的執行時機，下列敘述何者正確？ {
~等門口累積一定積水後再放置#為了安全考量，防護措施應具備預防性，而非等積水後才處理。
=與「開關車門」動作同時或緊接進行#這能確保客人在踏入館內的第一時間就受到安全提醒與保護。
~待客人進入大廳入座後再補放#這會導致客人在進門最濕滑的瞬間缺乏警示，增加跌倒風險。
~雨勢變大到需要撐傘時才需要放置#只要是雨天就應留意地面濕滑，不論雨勢大小。
}

::OP-FO-06-Q04::當飯店門口出現大型團體旅客（團客）或人潮眾多時，為了流程順暢，應如何操作自動門？ {
~請領隊站在門邊持續按住開門鈕#這會造成領隊的不便，且不符合飯店主動服務的原則。
~調整感應器靈敏度讓門保持開啟#調整機器設定過於繁瑣，並非 SOP 建議的標準做法。
=先按開自動門，隨即關閉電源讓門維持常開，待客進入後重啟#這是確保大量人群通行順暢且避免門頻繁開關夾傷客人的標準做法。
~手動將玻璃門推開並用滅火器頂住#使用非專業物品頂門會損壞設備且有礙觀瞻。
}

::OP-FO-06-Q05::關於行李服務，在何種情況下，夥伴應主動說出：「您好，需要幫您將行李放置房內嗎？」 {
~只要客人行李超過兩件時#此服務並非單純看件數，而是需評估現場人力狀況。
=當現場夥伴有空閒時間時#在不影響櫃檯主流程且人力允許下，提供延伸的入房服務能提升滿意度。
~只有在客人主動開口要求協助時#SOP 強調主動服務，而非等待客人提出要求。
~僅限於雨天客人不便自行拿行李時#此項服務不限於天候，而是根據現場人力餘裕度來決定。
}

::OP-FO-06-Q06::在執行「主動接待」流程時，SOP 對於溝通禮儀有哪項明確的要求？ {
=與客人談話時，眼神一定要注視著對方#眼神交流是建立信任感與展現服務熱忱的最基本要求。
~必須在距離客人三公尺處大聲問候#問候應自然，而非單純追求聲音大小或固定距離。
~溝通時手部必須隨時引導方向#雖然引導是好事，但 SOP 特別強調的是眼神注視而非手勢。
~為了展現效率，說話速度應加快#過快的語速可能導致客人聽不清楚，並非優質服務的表現。
}
