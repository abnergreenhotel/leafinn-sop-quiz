$CATEGORY: OP-FO-21 退款作業流程

::OP-FO-21-Q01::一位透過電話訂房並已匯款的客人在入住前 2 天打來要求取消訂房，根據目前的取消規定，您應如何告知退款金額？ {
~告知將會全額退還匯款金額。#全額退款僅適用於入住前 3 天取消的情況。
=告知將會退還 50% 的訂金金額。#針對入住前 1 至 2 天取消的預約，規定僅退還 50% 訂金。
~告知因已進入入住前 3 天，訂金將全額沒收。#全額沒收僅適用於入住當天取消的情況。
~告知可以全額退款，但需收取 10% 的違約金。#現行規範並無 10% 違約金之比例，應依據天數對應固定比例退還。
}

::OP-FO-21-Q02::當您在處理透過訂房網站（如 Agoda 或 Booking.com）預訂的客人要求取消時，下列操作何者正確？ {
~直接在德安系統幫客人取消並手動輸入退款資訊。#平台訂單涉及第三方合約，飯店端不應直接操作退款。
=請客人自行聯絡訂房平台進行取消作業。#外站平台訂單需由客人與平台端對接，以符合平台之取消規範。
~告知客人當天取消可以全額退款，並於一週內使用完畢。#平台當天取消通常沒收全額，且權益保留期限為三個月而非一週。
~幫客人聯繫平台，並在 UOF 系統填寫退款單。#應由客人主動聯繫平台，且平台訂單退款不由飯店端填寫 UOF 退款單處理。
}

::OP-FO-21-Q03::在處理「電話訂房」的退款申請時，夥伴必須向客人索取哪一項資訊才能完成後續流程？ {
~客人的身分證正反面影本。#退款作業主要需要銀行端資訊，而非身分證件影本。
~原匯款帳號的臨櫃匯款收據。#收據並非後續填寫退款單與匯款作業的必要憑證。
=客人的銀行存摺封面影本或加入lLINE傳送照片也可以。#電話訂房或 ATM 轉帳退款需確認收款帳戶正確性，必須提供存摺封面。
~客人的原始訂房確認郵件截圖。#訂房資訊可從系統查詢，退款作業最關鍵的是收款人的金融帳戶資訊。
}

::OP-FO-21-Q04::當客人詢問退款大約多久會入帳，以及是否會有額外費用時，下列回覆何者最符合作業規範？ {
~「退款會在 3 天內完成，不會扣除任何費用。」#作業時間不足，且忽略了轉帳手續費的可能性。
=「退款需 7-14 個工作天，非信用卡退款會產生 15元手續費。」#此回覆正確涵蓋了 SOP 規定的作業天數及銀行轉帳產生的手續費金額。
~「退款需 1 個月時間，手續費會由飯店全額吸收。」#作業時間過長，且非信用卡退款之手續費應由退款中扣除。
~「只要主管簽核完成就會入帳，通常不收費。」#回覆內容模糊，未給出明確的工作天範圍與費用標準。
}

::OP-FO-21-Q05::在德安系統中執行「電話訂房」取消流程時，進入顧客訂單後，下一個正確的關鍵點選動作為何？ {
~點選「訂房管理」後直接點選「刪除訂單」。#不應直接刪除，應依照流程進入 Profile 進行特定操作。
=點選「取消單筆」並存檔。#這是 SOP 規定的標準動作，能確保單筆訂單正確變更為取消狀態。
~點選「訂單備註」打上取消原因即可。#僅打備註無法改變訂單狀態，必須執行取消功能。
~進入「入帳作業」將金額歸零。#金額歸零前應先完成訂單狀態的行政取消流程。
}

::OP-FO-21-Q06::一位官網訂房的客人因為操作問題無法自行在網頁取消，您協助其在 web 手動取消後，如何確認該訂單已成功取消？ {
=訂單的底色會變為反藍狀態。#在 web 操作介面中，底色反藍是確認該訂單取消成功的視覺標示。
~訂單號碼會自動消失。#訂單不會消失，必須保留紀錄以供查核。
~系統會自動彈出退款成功的視窗。#取消訂單與退款作業是不同步驟，取消後仍需後續行政流程。
~訂單狀態會變為綠色。#根據文件，正確的成功標示顏色應為藍色而非綠色。
}

::OP-FO-21-Q07::在填寫 UOF 「訂房退款申請單」時，針對「是否酌收工本費」欄位，若客人生前是以信用卡刷卡支付，該如何勾選？ {
~勾選「是」，因為所有退款都要收費。#信用卡退刷流程與銀行轉帳不同，不應收取額外手續費。
=勾選「否」，因為信用卡退款一律不收取工本費。#SOP 明確規定信用卡退款一律不收取工本費（手續費）。
~勾選「是」，並手動輸入金額為 30 元。#手續費是針對銀行帳號退款產生的，信用卡退款不適用。
~不需勾選，該欄位僅限銀行轉帳退款填寫。#表單上有此必填選項，必須依據退款類別做出正確判斷與選擇。
}

::OP-FO-21-Q08::完成 UOF 退款申請單的填寫後，在德安系統端最後必須執行哪一個動作以利後續查核？ {
~直接將訂單狀態改為「結案」。#退款作業尚未完成入帳前，不應直接將狀態改為結案。
=在德安備註欄位打上填退日期。#這是退款申請流程的最後一步，目的是留下申請紀錄的軌跡。
~刪除該筆訂房卡的所有資訊。#所有操作軌跡應保留，不可隨意刪除資訊。
~列印出退款單據並夾在交接簿。#現行作業採 UOF 電子簽核，應在系統內完成備註而非單純依靠紙本交接。
}
