$CATEGORY: OP-FO-30 公區維護作業

::OP-FO-30-Q01::尖峰時段即將到來，櫃檯人員正在忙碌中，關於「大廳巡檢」最正確的執行方式為何？ {
~站在櫃檯內環視大廳與門口狀況，確認無異狀後記錄。#僅憑目測櫃檯周邊無法掌握視覺死角，標準作業要求必須實際走動查看。
=實際走動巡視大廳、電梯、洗手間與外圍動線，確保環境整潔與照明正常。#實際走動能確認氣味、細節清潔及安全狀況，是確保公共空間品質的關鍵。
~考量現場客人較多，為了不干擾動線，先暫停該班次的巡檢作業。#尖峰前巡檢正是為了確保環境處於最佳狀態，不能隨意取消，應以不打擾客人為前提進行。
~通知房務人員代為巡視，櫃檯人員專心處理現場入住作業。#公區維護是該班次例行作業，應由負責同仁依標準流程親自執行。
}

::OP-FO-30-Q02::在大廳執行地面整理作業時，發現地板因雨水濕滑，此時最優先的動作是什麼？ {
~立即拿取乾拖把將水跡擦乾，不需設置告示以維持美觀。#雖然擦拭很重要，但在未排除濕滑風險前，缺乏告示會增加客人滑倒的安全隱憂。
~先回櫃檯撥打電話告知主管，等待進一步指示再處理。#現場安全風險應優先排除或警示，而非先進行行政回報。
=立即設置「地面濕滑」告示牌，再進行後續的地面清理。#及時設置告示是防止意外發生的首要安全步驟，符合 SOP 的注意事項。
~通知門口警衛協助提醒客人，待手邊工作告一段落再處理。#異常清潔狀況應即時處理，不能延誤，以免造成不可挽回的安全意外。
}

::OP-FO-30-Q03::小明發現大廳迎賓區提供的餅乾快要拿光了，但他正準備去休息，他應該如何處理？ {
~等休息結束回來後，如果發現餅乾完全沒了再一併補充。#SOP 明確規定不可等備品完全用完才補充，應隨時注意數量。
=評估剩餘數量，在完全用完前立即進行補整，確保擺放整齊。#隨時維持備品充足與陳列標準，是提升顧客體驗與維持館內服務水平的關鍵。
~通知下一班同仁接班時再補充，避免影響自己的休息時間。#本班次發現的不足應立即處理，不能將已見的缺漏留給下一班處置。
~將剩餘的餅乾集中在一起，看起來比較多，就不用現在補充。#這只是視覺上的遮掩，並未解決量足與陈列標準的問題。
}

::OP-FO-30-Q04::關於「區域專案維護」（如植栽區、展覽區），下列執行考量何者正確？ {
~為了徹底清理，應選擇在人潮最多的時段封區作業以展現效率。#應避開人潮尖峰，與現場營運動線協調，以避免影響顧客體驗。
~專案維護只需由主管安排，同仁不需自行確認安全狀況。#即使是主管安排，執行前仍需確認封區的安全與執行時段的適切性。
=若需使用工具或封閉特定區域，需先確認作業時段與環境安全。#專案維護通常涉及特定範圍，必須確保作業過程不對客人造成危險或過大不便。
~專案維護可以替代每日的例行巡檢，節省人力作業成本。#例行巡檢與專案維護的目的不同，兩者不可互相取代。
}

::OP-FO-30-Q05::巡檢時發現公共廁所有異味且馬桶漏水，但現場沒有工務人員，正確的處理邏輯為何？ {
=先在現場設置提醒或安全警示，並立即回報主管或相關部門。#對於無法立即排除的異常，應先採取防護/警示措施，並確保異常被追蹤處理。
~因為無法修理，只需在交接本上寫下來，不需要現在處理。#漏水與異味會嚴重影響現場環境，應立即通報並視情況聯繫房務或主管處理。
~嘗試自行拆解修理，直到修好為止，不需特別回報。#同仁應視專業權責處理，若非專業工務背景不應強行修理，以免損害擴大。
~直接關閉該洗手間，不放告示牌，讓客人自行選擇其他區域。#無預警關閉且缺乏告示，會造成顧客困擾與服務上的負面印象。
}

::OP-FO-30-Q06::關於下班前的「確認與交接」作業，下列哪一項做法符合 SOP 規範？ {
~只要本班次工作都有做完，就不需要再進行下班前的巡視。#下班前需再巡視一次重點公區，以確保交給下一班的狀態是符合標準的。
=將尚未補充完成的耗材或異常案件詳細記錄於交班，並確保接班者能承接。#交接的目的是確保資訊不漏失，且讓公區維護結果能被下一班無縫承接。
~口頭告訴接班同仁即可，不需特別寫入交班紀錄中。#SOP 要求需寫入交班，以利後續追蹤並作為正式的紀錄憑證。
~未完成的事項如果覺得不重要，可以不寫入交班以免被主管檢討。#誠實記錄待辦事項與異常，才能確保旅館整體營運的品質與安全。
}

::OP-FO-30-Q07::執行例行清潔工作中，若看到有客人在大廳休息區交談，下列應對方式何者較佳？ {
=以不打擾顧客動線為原則，避開客人周邊區域進行清潔。#旅館服務應以顧客體驗為優先，執行作業時應將干擾降至最低。
~請客人先移駕到其他區域，以便能一次性完成大廳的所有清潔作業。#主動要求客人移動會降低服務品質，應彈性調整清潔順序。
~繼續照常作業，哪怕掃地機器人或抹布會碰到客人也沒有關係。#這嚴重違反了「不打擾顧客」的服務原則，容易引發客訴。
~為了避免尷尬，這班次的清潔工作直接取消，等客人離開再做。#例行工作仍須執行，只是需根據現場狀況調整做法與優先順序。
}
