$CATEGORY: OP-FO-27 顧客遺留物作業

::OP-FO-27-Q01::房務同仁在 302 號房發現房客遺留的一件夾克。櫃檯人員聯繫上客人後，對方表示隔天會親自回來領取，此時櫃檯人員應優先執行的後續操作為何？ {
~將物品直接放入行李房，等客人明天來再登記。#未即時標示與登記容易造成交接遺漏或物品混淆。
=在物品上黏貼寫有取物人全名與手機的便條紙，並放置於行李房上方置物籃。#明確的資訊標示與固定存放位置是確保失物能準確歸還的關鍵步驟。
~將夾克暫時放在櫃檯下方，方便隨時交給客人。#櫃檯並非指定存放區域，可能造成桌面凌亂或物品受損。
~為了環保，不使用便條紙，僅在 PMS 系統備註即可。#實體物品上必須有清楚標識，以便現場人員快速核對身分。
}

::OP-FO-27-Q02::一位房客退房後致電飯店，表示可能將無線耳機遺落在房間內。房務人員初步回報「未看到」。身為櫃檯人員，下列哪種回覆最符合服務流程？ {
~告知客人「房務已經找過了，確定沒有，請您再回憶一下放在哪裡」。#這種說法容易讓顧客感到被質疑，且未展現持續協助的態度。
=詢問客人最後一次使用後可能放置的位置，告知會持續尋找並再次回電給客人。#確認細節並承諾後續追蹤能展現積極負責的態度，也是 SOP 指定的話術邏輯。
~直接告訴客人「如果是貴重物品，沒看到通常就是找不到了」。#此回應過於消極且帶有負面暗示，違反服務專業。
~請客人自行向警方報案，飯店不介入尋找流程。#飯店有義務協助客人確認房內狀況，而非直接推諉給警方。
}

::OP-FO-27-Q03::客人要求將遺落的相機以「黑貓貨到付款」方式寄回。櫃檯人員在與客人溝通時，下列哪項資訊是必須提醒客人的？ {
~飯店將代墊運費，請客人事後匯款。#SOP 明定採用「貨到付款」模式，由客人收貨時支付運費。
=運費大約 120-150 元，且寄送需要 3-5 天的作業時間。#提前告知預估費用與時程能避免客人對物流成本或速度產生誤解。
~物品寄出後若有損壞，飯店將負全責。#SOP 未提及飯店須承擔物流過程的毀損責任，此為不當承諾。
~運費一律固定為 100 元。#運費會依大小變動，SOP 提供的是 120-150 元的參考區間。
}

::OP-FO-27-Q04::大夜班同仁在執行巡檢時，發現員休冰箱內有一份「已開封」且客人未約定領取日期的甜點。根據保管期限規定，應如何處理？ {
=直接將該食品丟棄，因為易腐敗物品不留過夜。#對於易腐敗或衛生用品，SOP 規定不留過夜，通常於大夜巡檢時處理。
~將其轉移到行李房置物籃保留 6 個月。#食物會腐敗，不能與一般物品一樣長期存放。
~等早班同仁聯絡客人後，再決定是否丟棄。#除非客人有指定領取日期，否則易腐敗物品不應保留至隔日。
~讓同仁自行領用，避免浪費食物。#未經核准的遺留物不可隨意領用，且衛生用品具安全疑慮。
}

::OP-FO-27-Q05::房客反映在房內遺失了一只名貴鑽戒。針對此類「貴重物品」的搜尋流程，下列做法何者最正確？ {
~房務同仁自行進入房間翻找，找到後再回報。#單獨作業缺乏公信力，且可能衍生更多紛爭。
=必須由主管及房務同仁一同進入房間搜尋，且過程最好錄影。#多人見證與錄影存證能保護員工並確保搜尋過程透明。
~因為是名貴物品，應立即請警察進入客房搜尋。#應先由飯店內部進行複查流程，若無尋獲再依規回報主管處理。
~為了效率，請下一班交接人員有空再去尋找。#遺失貴重物品應立即處理，並回報主管。
}

::OP-FO-27-Q06::在使用 PMS 系統進行「失物明細」登記時，關於 A 區的「遺失日期」與「資料連結」，下列操作何者正確？ {
~日期應填寫遺失物被丟掉的那一天。#遺失日期通常設定為登記當日或發現日。
=點選右方「查詢住客」，透過 C/O 日期與房號找到客人資料並帶入。#透過系統連結退房資料，能確保客人身分與聯繫方式正確無誤。
~只需填寫房號，不需要帶入客人個人資料。#缺少聯繫資訊將導致後續無法與客人聯繫取回。
~日期應手動輸入客人的入住日期 (C/I)。#應以遺失物登記當日為基準，並透過退房日 (C/O) 查詢住客。
}

::OP-FO-27-Q07::飯店內存放的一般遺留物（如書籍、衣物）已超過 6 個月的保管期限。根據 SOP，下列哪項不是正確的後續處理方式？ {
=直接將物品移交警察局處理。#移交警方主要是針對「貴重物品」，一般物品通常由內部領用、轉贈或捐贈。
~經主管核准後，由拾獲的員工領回。#這是符合 SOP 的一般物品處理方案之一。
~統一彙整後，捐贈給公益團體。#捐贈給公益團體是 SOP 中建議的處理方式之一。
~員工領用或轉贈處理。#這屬於一般物品逾期後的合法處理範圍。
}

::OP-FO-27-Q08::若遇連假期間，黑貓宅急便暫停收件，但客人急需寄回失物，櫃檯人員應如何變通處理？ {
~告知客人連假後才能寄送，請客人耐心等候。#雖然是一種做法，但 SOP 有提供更具主動性的替代方案。
=改前往鄰近的 7-11 進行寄件作業。#當貨運收件暫停時，超商寄件是 SOP 明確指定的應變措施。
~請客人自行聯繫其他貨運公司到飯店收貨。#這會增加客人的負擔，非最佳服務做法。
~將物品暫時存放，直到連假結束自動忘記這件事。#這屬於嚴重的服務疏失。
}

::OP-FO-27-Q09::在 PMS 系統中，當客人已透過黑貓宅配領取物品後，最後必須完成哪個步驟才算流程結束？ {
~直接按離開放棄操作。#未完成狀態更新會導致該失物在系統中一直處於未領取狀態。
=勾選該筆資料，點選「遺失者領回」，並在領回備註填寫取回方式後存檔。#完整的銷案動作必須包含選擇領回模式、備註說明並按下存檔鍵。
~刪除該筆資料，以免佔用系統空間。#應保留紀錄而非刪除，以備後續查證。
~點選「新增報失」，重新輸入一次資料。#這是重複輸入，並非結案動作。
}

::OP-FO-27-Q10::當客人詢問是否有物品遺失時，若現場搜尋後並「無尋獲」，櫃檯人員下列哪項行為是必須的？ {
~因為沒找到，所以不需要回電給客人。#SOP 明定「有無尋獲皆須聯繫顧客」，這是基本的服務禮儀。
=無論結果如何，皆必須主動聯繫顧客回報狀況，並落實交接。#主動回報能確保客人得知進度，且交接能讓下一班同仁了解已處理完畢。
~直接告訴客人「飯店沒有監視器，無法確定是誰拿的」。#此說法帶有卸責嫌疑，且非 SOP 規範之應對。
~先將遺失物表填好，但不做任何後續動作。#填表僅是過程，後續的聯繫與交接才是確保服務延續的重點。
}
