$CATEGORY: OP-GS-02 服務禮儀常用語

::OP-GS-02-Q01::當您在飯店大廳看見一位剛進門的顧客時，下列哪一項做法最符合服務標準？ {
=保持微笑並與顧客有眼神接觸，在三秒內主動說出「午安，您好」。#服務標準要求在見到顧客三秒內主動問候，並搭配眼神接觸與微笑，以展現親切感。
~為了不打擾顧客，先等顧客主動開口詢問後再回應。#SOP 明確禁止低頭不語或等待顧客先開口，這會顯得服務主動性不足。
~點頭示意即可，等顧客走到櫃檯前再進行口頭問候。#標準流程強調應在見到顧客「三秒內」主動問候，而非等到近距離接觸才開始。
~一邊處理手邊的電腦文書，一邊抬頭說「您好」。#問候時應專注並搭配眼神接觸，分心處理其他事務會讓顧客感到不被重視。
}

::OP-GS-02-Q03::如果某項服務需要操作電腦查詢較久，您該如何與顧客溝通？ {
=先向顧客說：「好的，請您稍等，我這邊先為您查詢。」#回應應以清楚、正向為原則，且在執行查詢動作前必須先告知顧客稍候。
~沉默地快速操作電腦，等查到結果後再告知顧客。#長時間的沉默會讓顧客感到不安，應先口頭說明下一步動作。
~告訴顧客：「現在系統有點慢，您可能要等很久喔。」#回應應維持正向原則，而非傳遞負面或帶有抱怨性質的資訊。
~跟顧客說：「我查一下喔，大概可以吧。」#「大概可以」屬於含糊回應，容易造成服務糾紛或顧客誤解。
}

::OP-GS-02-Q04::當顧客因為進房時間稍微延誤而向您反映時，最合適的應對順序為何？ {
=先向顧客致歉，說明處理方式並立即協助確認。#面對異常或延遲，應先致歉再說明處理方式，語氣需穩定且不推卸責任。
~先解釋是因為房務人員太少，所以才會延誤。#這屬於推諉責任的表現，SOP 要求語氣需穩定且不找藉口。
~只道歉不處理，重複說：「不好意思，造成您的不便。」#SOP 明確規定致歉後需接續具體的處理動作，不可只道歉不處理。
~直接請主管出來處理，不需進行初步對話。#現場夥伴應先進行初步致歉並告知會如何處理，而非第一時間就放棄溝通。
}

::OP-GS-02-Q05::服務完成，顧客準備離開飯店前往景點時，哪一種服務行為是錯誤的？ {
~主動說：「謝謝您，祝您今天一切順心。」#這是標準的感謝與送客語，有助於提升品牌形象。
~除了口頭送客，同時要有目送顧客離開的動作。#送客不僅是語言，視覺上的目送也是禮儀的一部分。
=說完「謝謝您」後，趁顧客還在門口時立刻蹲下整理櫃檯內的垃圾桶。#SOP 規定不可在顧客尚未離開前就轉身做自己的事，這會讓服務顯得虎頭蛇尾。
~親切地提醒顧客：「若還有需要歡迎再跟我們說。」#這展現了後續服務的熱忱，符合感謝語的精神。
}

::OP-GS-02-Q06::顧客詢問某個他館的活動資訊，但您並不清楚細節，下列哪種說法最合適？ {
~「這不是我負責的，您可以打電話問那間旅館。」#這屬於禁用語中的推諉，會讓顧客覺得服務生硬、不被協助。
=「我這邊先幫您確認正確窗口，請您稍等一下。」#即使無法立即回答，也應說明原因、提供替代方案或主動協助查詢。
~「不知道耶，飯店官網應該有寫，您自己看。」#「不知道」與「你自己看」皆為禁用語，是不負責任的應對方式。
~「不好意思，我們這邊不行查他館的資訊。」#應避免直接生硬地說「不行」，而應轉化為積極的協助態度。
}

::OP-GS-02-Q07::新進夥伴在教育訓練期間，應如何使用「七大常用語」？ {
~只需在被主管盯場時使用，平常可以隨意說話。#常用語應融入日常執勤中，成為自然的習慣。
=應於教育訓練期間熟悉，並在值勤期間優先且自然地使用。#SOP 規定新進夥伴需熟悉這些語言，並於執勤期間優先使用，以確保服務標準化。
~只要背下文字即可，不需要考慮語氣或情境。#教育訓練的目標是「自然使用」，包含語氣、態度與情境的配合。
~盡量使用過度口語化的方式來拉近與客人的距離。#SOP 提醒需避免過度口語化，應維持專業的服務禮儀常用語。
}

::OP-GS-02-Q08::在處理顧客投訴飯店設施故障時，下列哪一項「致歉用語」的原則是錯誤的？ {
~說話語氣需保持穩定。#穩定的語氣能安撫顧客情緒，是致歉的基本要求。
=如果真的是工程部的疏失，可以向顧客說明是哪位同事的錯。#SOP 要求致歉時「不推責」，將內部疏失歸咎於特定人員是不專業的行為。
~先致歉，再說明後續的處理方式。#這是標準的處理順序，先表達歉意再展現解決問題的誠意。
~致歉後應接續具體的處理動作（如立即派人維修）。#具體動作才能真正解決顧客問題，僅有口頭道歉是不夠的。
}

::OP-GS-02-Q09::關於「禁用語與替代說法」，當顧客提出一個違反飯店規範的要求時，該如何應對？ {
~直接說「不行」，讓顧客知道飯店的底線。#「不行」是直接生硬的語句，應避免使用。
=先說明原因，並嘗試提供替代方案或協助查詢折衷辦法。#即使結果是拒絕，也應透過說明原因或提供替代方案來降低顧客的不適感。
~告訴顧客：「規定就是這樣，我也沒辦法。」#這屬於消極的回應方式，沒有提供顧客下一個步驟或解釋。
~不說話，直接把飯店規章推給顧客看。#這符合「你自己看」的生硬態度，缺乏服務溫度與專業溝通。
}

::OP-GS-02-Q10::在「七大常用語」中，哪一句話最適合用來結束一段服務，讓顧客感受到被尊重？ {
=「謝謝您，祝您今天一切順心。」#這句話結合了感謝與祝福，是送客與結束服務時非常標準且暖心的常用語。
~「好啦，就這樣。」#這屬於過度口語化的用語，不符合專業服務標準。
~「您慢走。」#雖然也是送客，但相較於 SOP 推薦的「祝您一切順心」，顯得較為簡短且例行公事。
~「再見。」#過於日常且單薄，應使用更具感謝意味與祝願的常用語。
}
