$CATEGORY: OP-FO-15 訪客接待流程

::OP-FO-15-Q01::當一位訪客來到櫃檯並表示要找住客時，作為櫃檯人員，第一時間最正確的服務儀態與開場白為何？ {
=起身並微笑親切地詢問：「您好，有什麼可以協助您的部分呢？」#良好的首要印象應包含主動起身與親切微笑，並使用標準化話術進行詢問。
~坐在位置上禮貌詢問：「您好，請問您要找哪一位住客？」#維持坐姿可能會讓訪客感受到被冷落或不夠受重視，應起身迎接以展現服務專業。
~先確認對方的身分證件，並詢問：「您好，請問有什麼事嗎？」#第一時間應先進行親切的問候與協助意願表達，而非直接進入質疑或查驗身分的程序。
~直接詢問：「您好，請問您有與住客約好嗎？」#此問法較容易讓訪客感受到被質疑或拒絕，不符合親切有禮的標準作業規範。
}

::OP-FO-15-Q02::若住客已事先告知櫃檯有朋友來訪，當訪客到達櫃檯並正確報出住客全名後，下一步正確的操作流程是什麼？ {
~直接告知訪客房號，並請其自行搭乘電梯上樓#為了住客隱私與安全，不應直接口頭告知房號，應由人員協助感應電梯。
=致電住客房間確認，得到同意後協助感應電梯讓訪客上樓#致電確認能確保住客當下的方便性，協助感應電梯則是落實門禁控管的必要步驟。
~請訪客在沙發區稍坐，告知住客後請住客親自下來帶訪客#若住客已表示同意，櫃檯應主動提供協助感應電梯的服務，而非強制住客下樓接待。
~查詢訂單確認住客在房內後，給予訪客臨時感應卡上樓#給予訪客感應卡並非 SOP 中的流程，標準作法是由櫃檯人員協助感應電梯。
}

::OP-FO-15-Q03::一位訪客表示要找某位住客，但該住客之前已交代櫃檯「請勿打擾」，此時你應該如何得體地回應訪客？ {
~「不好意思，該住客現在交代我們請勿打擾。」#直接透露住客交代「請勿打擾」會間接證實住客在房內，未能有效保護住客隱私。
=「您好，目前沒有查詢到訂單耶。」#這是保護住客隱私的標準話術，能優雅地婉拒訪客而不透露住客的實際留宿狀態。
~「目前住客不方便接見朋友，請您晚點再聯繫。」#此回應語氣較生硬且透露了住客的存在，不符合 SOP 對於隱私權的保護要求。
~「幫您聯繫一下住客，請您稍等。」#住客已明確告知請勿打擾，若此時再致電確認會干擾到住客的休息。
}

::OP-FO-15-Q04::在處理訪客接待時，關於電話的使用，下列哪一個操作符合 SOP 的注意事項？ {
=若訪客沒有手機，可以引導其使用櫃檯電話聯繫住客#提供櫃檯電話作為輔助通訊工具，是 SOP 中對於無手機訪客的友善配套措施。
~為了安全起見，禁止任何人使用櫃檯內部的電話機#雖然有安全考量，但 SOP 明確指出在特定情況下可協助顧客使用電話。
~請訪客到大廳公用電話區使用投幣式電話#主動提供櫃檯電話協助是更具備服務熱忱的做法，且能掌握聯繫的狀況。
~請同仁用私人手機幫訪客撥打給住客#使用員工私人設備不符合專業作業規範，應以飯店櫃檯設備為主。
}

::OP-FO-15-Q05::一位訪客因「查無訂單」的情境被婉拒後離開了飯店，接下來櫃檯人員最關鍵的後續動作為何？ {
~立即在交接簿註記，防止該訪客再次進入飯店#除非访客有違規行為，否則單純的註記並非首要任務，應先通知受影響的住客。
=致電住客告知剛才有人來訪，但因隱私考量未告知房號#此舉能讓住客掌握外界訪蹤，同時體現飯店對隱私的重視，若客有需要可自行聯繫。
~不需做任何動作，以免再次干擾到住客#主動告知有人來訪是服務的一環，能讓住客決定是否要回撥或與對方取得聯繫。
~致電給主管詢問是否需要調閱監視器紀錄#若只是正常詢問且無異常狀況，並不需要立即升級到調閱監視器的層級。
}
