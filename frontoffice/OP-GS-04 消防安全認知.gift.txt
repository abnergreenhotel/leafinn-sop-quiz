$CATEGORY: OP-GS-04 消防安全認知

::OP-GS-04-Q01::新進夥伴阿強剛到職，他認為只要背熟辦公室內的「消防平面圖」就足以掌握逃生動線。根據飯店安全規範，阿強還需要進行哪項必要的動作？ {
~只需在每半年的消防演習時參與即可。#雖然定期演習很重要，但新進夥伴的基本要求是在到職初期即掌握環境，而非等待半年一次的訓練。
=必須親自到現場進行實地走點，確認設備與路徑。#實地確認能確保人員在緊急狀況下具備直覺性的方位感，這是平面圖無法提供的感官記憶。
~請資深夥伴口頭描述逃生出口的詳細特徵。#口頭描述容易產生認知偏差，親自確認才是確保安全的最高準則。
~將平面圖拍在手機裡隨時查看即可。#在火警發生、濃煙密佈或電力中斷時，依賴手機查看圖紙會大幅延誤逃生與反應時間。
}

::OP-GS-04-Q02::當你在客房走廊發現垃圾桶起火，煙霧剛開始散發時，第一步應該如何判斷是否執行滅火？ {
~無論火勢大小，先拿滅火器噴灑再說。#未經評估就行動可能讓自己陷入無法控制的火場風險中。
=先確認自身安全，並評估火勢是否在可控制的初期階段。#確保自身安全是處理火災的前提，只有在初期且可控的情況下才建議嘗試滅火。
~立即尋找水源灌救，不論是否有化學滅火器。#不同的起火類型需對應正確的滅火工具，隨意使用水源可能造成意外擴散。
~等到警報器自動響起後，再根據廣播指示行動。#發現初期火勢時應主動評估並通報，消極等待可能導致火災迅速擴張。
}

::OP-GS-04-Q03::在通報火警、煙霧或異味時，為了讓相關編組能精準支援，通報內容應包含哪些關鍵資訊？ {
~只需要大聲呼喊「失火了」讓整層樓聽到。#模糊的呼喊會造成恐慌且無法提供有效的救援資訊。
~詳細描述起火的原因以及可能的賠償責任。#緊急時刻應專注於現場現狀，事故原因與責任應在事後復原階段再進行紀錄。
~告知發現者的員工編號以及到職天數。#這些個人資訊與當前的滅火救援行動無關，會浪費寶貴的通報時間。
=明確回報所在的樓層、區域、火勢狀況及是否有受困人員。#精確的空間位置與受困資訊，能讓消防編組立即判斷資源配置與救援優先順序。
}

::OP-GS-04-Q04::疏散引導過程中，夥伴在走廊引導房客往安全出口移動時，必須不斷提醒哪一項重要事項？ {
~提醒房客記得帶上房間內的貴重物品。#疏散時應以生命安全為重，尋找財物會延誤逃生黃金時間。
=強調「不要搭乘電梯」，請依指示往樓梯移動。#火災時電梯可能斷電或產生煙囪效應，引導人員必須確保住客使用正確的安全動線。
~詢問房客對於飯店服務的滿意度。#緊急疏散情境下，任何與逃生無關的對話都會造成不必要的干擾。
~要求房客在樓梯間保持安靜並排成一列橫隊前進。#雖然秩序重要，但「橫隊」會阻塞逃生路徑，且提醒不要搭乘電梯才是安全核心。
}

::OP-GS-04-Q05::抵達集合點後，為了確保沒有房客受困於館內，引導夥伴應隨身攜帶並使用哪一項工具進行確認？ {
~飯店的年度業績報表。#管理數據與緊急人員清查完全無關。
=當日住客名單（包含預計退房與續住名單）。#持有準確的名單才能進行精確點名，判斷是否仍有住客留在房內或受困。
~空白的筆記本與筆，現場請大家簽名。#讓房客現場簽名太過緩慢，且無法得知誰「應該在場但沒出現」。
~飯店的所有客房鑰匙。#鑰匙對於點名作業沒有直接幫助，應由指揮人員或專業消防員使用。
}

::OP-GS-04-Q06::關於館內緩降機與特殊設備的使用規範，下列敘述何者正確？ {
~只要看過設備旁的教學貼紙，任何人都可以指導住客操作。#緩降機具有危險性，未經正式訓練者誤導操作可能導致墜落意外。
=僅限受過訓練的人員操作，且需依現場指揮使用。#緩降機是最後的避難手段，必須由受訓人員在正確時機操作以確保安全。
~為了節省時間，可以多人同時使用一組緩降機下滑。#設備有載重限制與操作邏輯，多人同時使用會引發嚴重事故。
~平時為了方便練習，夥伴可以自行解開緩降機試用。#非緊急或非訓練期間隨意動用消防設備會造成設備損壞或緊急時失效。
}

::OP-GS-04-Q07::消防事件處理完畢並確認人員安全後，夥伴在「復原與回報」階段應採取哪項行動？ {
~只要口頭跟主管說「沒事了」即可下班。#重大安全事件必須留下文字紀錄，不可僅憑口頭交接。
~將使用過的滅火器放回原位，假裝沒有發生過。#使用過的滅火器壓力已失，若未即時報修補充，下次發生意外時將無效。
=完成點名、異常回報與紀錄，並針對使用過的設備報請檢修。#完整的紀錄與設備檢修，是確保飯店防禦體系在事件後能重新恢復運作的必要步驟。
~立即在社交媒體上發布現場火警照片增加曝光。#這違反職業操守且與 SOP 的復原流程無關。
}
