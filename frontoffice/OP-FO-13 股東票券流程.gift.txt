$CATEGORY: OP-FO-13 股東票券流程

::OP-FO-13-Q01::有一位客人在電話中表示想預約「中秋節連假」期間使用股東票券入住，身為訂房人員，你應該如何處理？ {
~直接為客人查詢剩餘房況並協助訂房#忽視了特定節日的使用限制，可能導致後續核銷失敗。
=委婉告知國定假日及連續假期不開放使用該票券#股東票券在國定假日及連假期間是不適用的，應在初步諮詢時就明確告知。
~請客人先傳送票券照片，確認代碼後再決定是否開放#即便票券代碼正確，日期若屬於連假，規則上仍無法受理。
~告知連假需以原價入住，票券僅能折抵 $500$ 元#隨意更改票券優惠內容並不符合操作流程，應直接依照日期規範執行。
}

::OP-FO-13-Q02::在電話預約階段，為了確保票券內容核對準確並避免聽覺誤差，下列哪一項是建議的操作方式？ {
~請客人在電話中逐字唸出票券後方的所有使用須知#口頭核對繁瑣且容易發生聽錯或誤解，效率較低。
=請客人將票券正、反面拍攝後傳送至官方 LINE 帳號#透過影像確認券碼、須知與印章最為清晰，能有效降低人工輸入錯誤。
~要求客人將票券正本先行掛號郵寄至旅館辦公室#這會增加客人的麻煩與寄送風險，並非初審階段的必要流程。
~僅記錄客人提供的姓名與電話，待現場入住再核對票券#未在訂房階段初審票券，若現場發現票券有問題將導致爭議。
}

::OP-FO-13-Q03::當客人預約在「週六」使用股東票券入住時，訂房人員在報價時必須特別注意什麼？ {
~告知假日房價需以原價計算，不提供任何折扣#這會讓客人損失應有的權益，股東票券在假日仍有其優惠折數。
=主動告知假日房價以 $6$ 折計算，並說明折後應付金額#針對假日入住，應主動說明折扣標準（$6$ 折）並讓客人清楚最終需支付的房費。
~告知假日的折扣與平日相同，均不需額外補貼#忽略了平日與假日的計價差異，可能造成現場收費時的糾紛。
~請客人自行查閱票券後方的假日收費標準，不另行口頭告知#訂房人員有義務在最後確認階段主動核對並重複折後金額。
}

::OP-FO-13-Q04::在資料蒐集與備註階段，除了基本資料外，哪一項資訊是必須詢問並記錄的細節？ {
=客人的交通方式（是否開車）及預計抵達時間#掌握交通與抵達時間有助於安排停車位與接待人力，是 SOP 規定的蒐集項目。
~客人的職業背景與過往入住其他飯店的經驗#這些屬於過度蒐集的私人資料，並非標準訂房作業的核心項目。
~確認客人是否願意支付額外費用升等房型#雖然是銷售機會，但並非 SOP 流程中針對票券預約指定的必填資訊。
~詢問客人的票券購買來源或購買價格#重點應在於核對票券本身的真偽與規範，而非其取得管道。
}

::OP-FO-13-Q05::針對「假日的票券預約」，為了確保訂單有效性，訂房人員應提醒客人完成哪項程序？ {
~假日入住只需口頭確認即可，不需支付任何款項#未進行擔保可能導致客人爽約，且不符合假日的特殊作業要求。
~提醒客人入住當日必須先支付全額原價現金作為押金#收費金額與方式應依折扣後房費為準，而非要求全額原價現金。
=要求客人必須先行授權或匯款剩餘的房費金額#對於假日入住，預先收取或授權餘額房費是保障營收的重要步驟。
~請客人在 LINE 上面回覆「同意入住」即可取代匯款#文字回覆不具備實質的金流保障功能。
}

::OP-FO-13-Q06::在系統中辦理「股東票券」的入住核銷作業時，關於訂金的錄入方式何者正確？ {
~必須先在系統手動錄入與房費相等的虛擬訂金#這會導致系統帳務邏輯錯誤，SOP 針對此類票券有簡化處理。
=無須錄入訂金，直接選擇入住即可#股東票券的特性在系統操作上不需錄入訂金，簡化流程以符合票券核銷邏輯。
~將票券代碼輸入在訂金備註欄，並錄入 $1$ 元作為記號#不正確的錄入金額會影響財務報表的準確性。
~必須等待主管授權後，方可進入系統錄入訂金#SOP 已明確規定此票券的操作流程，不需逐筆申請授權錄入訂金。
}

::OP-FO-13-Q07::辦理入住時，若發現客人的實體住宿券上有明顯的塗改痕跡，應如何處理？ {
~只要券碼清晰，塗改處不影響辨識即可接受#塗改可能涉及非法變造或效期爭議，不能輕易允收。
=視為票券瑕疵，應根據異常狀況處理規範進行處置#塗改痕跡是票券瑕疵的徵兆，必須依 SOP 異常流程判斷其合法性或拒絕使用。
~請客人在塗改處簽名確認，即可視同有效票券#簽名並不能修復票券本身的效力，仍存在核銷失敗的風險。
~將塗改處拍照上傳，等退房後再請後勤單位判斷#核銷應在入住辦理階段完成，若退房後才發現無效會難以追回房費。
}

::OP-FO-13-Q08::若客人抵達現場辦理入住時表示「忘記帶票券正本」，櫃檯人員應給予什麼提醒？ {
~告知只要有 LINE 的照片截圖，一樣可以享有優惠#照片僅供初審，現場必須回收實體券才能完成核銷流程。
=告知必須攜帶票券正本，否則將影響優惠核銷#實體票券是優惠核銷的唯一憑證，未帶正本會導致無法依票券優惠計價。
~請客人現場填寫切結書，承諾一週內寄回即可#SOP 中並未提及切結書替代方案，未帶正本通常需改按一般方式收費。
~直接請客人返家領取，否則櫃檯不能為其辦理入住#應告知對優惠核銷的影響，而非直接拒絕入住，通常會涉及計價方式的變更。
}
