$CATEGORY: OP-FO-10 禁菸標準作業流程

::OP-FO-10-Q01::在處理訂房流程時，為了避免後續產生消費糾紛，下列哪一項做法是服務人員必須優先執行的？ {
=在訂單確認時主動告知本館為全館禁菸旅館#主動告知能確保客人在入住前就理解規範，是預防後續糾紛的最前線步驟。
~等到客人入住後再由櫃檯說明禁菸規定#若到現場才說明，客人可能因行程已定而感到被迫接受，較容易產生爭議。
~僅在官網標示禁菸，不需在訂單確認中重複說明#僅依賴官網標示無法確保客人已閱讀並知曉，主動告知是 SOP 要求的控制點。
~直接在客人信用卡預刷一筆清潔費作為押金#SOP 並未規定預收清潔押金，且此舉可能造成不佳的客戶體驗。
}

::OP-FO-10-Q02::當客人在入住辦理手續（Check-in）時，櫃檯人員最重要的合約確保動作為何？ {
~向客人詳細解釋菸害防制法的法律條文內容#雖然需要提醒法規，但法律教育並非合約成立的核心法律依據。
=確認客人在登記卡（R卡）上簽名以表示同意入住規範#登記卡簽名象徵契約成立，是飯店後續若需收取損害賠償費的重要法律依據。
~口頭確認客人是否有抽菸習慣並記錄在系統中#口頭確認缺乏白紙黑字的證明，在產生消費糾紛時難以作為依據。
~要求客人複誦一遍全館禁菸的規定#此做法過於繁瑣且不具備正式的契約效力。
}

::OP-FO-10-Q03::若客人在房間內抽菸被發現，飯店人員在向客人說明收費時，應使用下列哪一種名稱較為精確且具法律正當性？ {
~政府違規罰鍰#政府罰鍰是由公家單位開立的處分，飯店不具備行政裁罰權。
=客房損害賠償清潔費#飯店收取的是針對客房恢復原狀所需的清潔費用或損害賠償，這在私權契約中是合理的。
~抽菸違規罰款#「罰款」二字通常屬於公權力行使，飯店應避免使用這類字眼以防法律爭議。
~環境空氣維護稅#飯店並非稅務機關，無法隨意創設稅名收費。
}

::OP-FO-10-Q04::服務人員在巡檢樓層時發現走廊有菸味，且判斷是由特定客房內飄出時，下一步應如何處理？ {
~直接進入房內制止並要求客人立刻搬離#在未經溝通前強行進入或驅逐客人可能引發更大的服務衝突。
=立即致電該房客，以婉轉禮貌的口氣告知本館禁菸規定#及時致電能立即遏止行為，且婉轉語氣是 SOP 要求的服務專業。
~在該客房門口貼上警告標語並拍照後離去#這無法即時停止抽菸行為，且會嚴重影響客人的居住感受。
~等客人退房後再一併處理，避免與客人正面對決#SOP 要求應及時發現並遏止，以減少煙味擴散及後續清理難度。
}

::OP-FO-10-Q05::房務人員在客人退房後進房巡查，發現房間內有菸味及菸灰，首先應進行哪項標準作業？ {
~立即將垃圾倒掉，避免味道殘留#過早清除跡證會導致後續收費時缺乏證據，造成爭議。
=拍照存證並立即告知櫃檯聯繫客人#拍照是保存證據的必要行為，及時告知櫃檯才能在客人離開前處理收費事宜。
~自行聯繫客人要求回來清理#聯繫客人的職責通常由櫃檯或主管負責，房務應專注於現場採證。
~先灑香水蓋過菸味，照常整房提供給下一組客人#僅掩蓋味道並非真正的清潔，且違反了後續需要打臭氧的流程。
}

::OP-FO-10-Q06::根據 SOP，對於曾發生在客房內抽菸行為的房號，在進行整房前應執行什麼特別步驟？ {
~全面更換房內的地毯與壁紙#這屬於過度維修，除非有具體損毀，否則非標準清潔流程。
~使用芳香劑噴灑後關閉空調 1 小時#噴灑芳香劑無法消除菸味，且 SOP 並未提及關閉空調的規範。
=由房務人員打臭氧後，靜置 30 分鐘再進去整房#臭氧能有效去除異味，靜置 30 分鐘是確保空氣品質與作業安全的必要程序。
~將窗戶打開通風 5 分鐘後即可整房#僅通風 5 分鐘難以去除深層菸味，不足以達到清潔標準。
}

::OP-FO-10-Q07::在與抽菸客人的後續聯繫與處理中，若客人的行為屬於「情節重大」，SOP 建議採取哪種進階行動？ {
~將客人的個人資料公佈在飯店官方網站上#這違反個資法且屬於不當的公審行為。
=致電衛生局進行檢舉舉發#針對嚴重違規者，尋求公權力介入檢舉是 SOP 中列出的處置選項之一。
~聯絡該客人所屬的公司要求連帶賠償#法律上客人為行為主體，通常不涉及其公司，且 SOP 並無此規範。
~強制沒收客人留在房內的所有私人物品#這屬於非法侵占，並非合法的損害賠償處理方式。
}

::OP-FO-10-Q08::退房後發現吸菸情事，櫃檯人員與客人聯繫並確認收費後，最後在系統操作上必須完成哪項紀錄？ {
=在客人資料中備註列入提醒（黑名單備註）#記錄在系統中能讓後續接待人員有所防範，是完整的客戶關係管理。
~刪除該客人的所有入住歷史紀錄#刪除紀錄會導致飯店失去後續查證的依據。
~在系統中修改客人的姓名加上「吸菸者」後後置#直接修改姓名不專業，應使用正式的備註欄位記錄細節。
~發送一封感謝信祝賀客人完成賠償#此情境不適合發送感謝信，重點應放在違規紀錄與後續防範。
}
