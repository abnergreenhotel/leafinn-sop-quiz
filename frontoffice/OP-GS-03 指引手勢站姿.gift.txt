$CATEGORY: OP-GS-03 指引手勢站姿

::OP-GS-03-Q01::當櫃檯目前沒有客人、處於待機狀態時，哪一種站姿最符合旅館服務標準？ {
=雙手自然交握於身前，不倚靠櫃檯，並保持抬頭挺胸。#保持挺胸、雙肩放鬆並自然交握雙手，能呈現專業且隨時準備服務的姿態，避免因倚靠顯得懶散。
~為了保持長期間站立的體力，可以輕微側靠牆面或櫃檯邊緣。#倚靠櫃檯或牆面會給人專業度不足或精神渙散的負面服務印象。
~雙手插入口袋或自然垂放兩側，雙腳分開站穩。#插口袋是嚴格禁止的禁忌行為，這會顯得對顧客缺乏基本尊重與禮貌。
~雙手抱胸注視前方，展示嚴謹、專業的防禦性站姿。#雙手抱胸屬於封閉性動作，容易讓顧客感到被排斥或具有距離感，非適當的服務姿態。
}

::OP-GS-03-Q02::當您準備將房卡與發票遞交給顧客時，下列哪一項操作符合 SOP 的「雙手遞物」原則？ {
~直接將房卡放置在櫃檯桌面上，由顧客自行拿取。#將物品直接推向或放置於桌面由顧客自取，缺乏互動感且不符合雙手遞交的溫暖服務標準。
~單手快速遞交，另一手同時操作電腦系統以節省時間。#單手遞物容易顯得輕率，且無法同時兼顧口頭說明遞交內容的必要步驟。
=以雙手遞交，並在交給對方前將房卡文字方向轉向顧客。#確保文字方向朝向對方，能方便顧客閱讀內容，體現細心的服務品質。
~以右手單手遞交，只要同時口頭說明內容即可。#SOP 要求應以雙手或右手遞、左手輔助，單手操作不符合正式的遞物禮儀。
}

::OP-GS-03-Q03::若顧客詢問電梯的位置，但電梯位於大廳轉角較隱蔽處，應如何正確指引？ {
~伸出食指明確指向電梯的方向。#單指指路是服務中的禁忌行為，會讓人感到被指使或不禮貌。
=五指併攏、掌心打開，以整個手掌引導方向並視情況主動帶位。#使用全手掌引導能展現大方的儀態，且長距離指引應搭配主動帶位以確保顧客順利抵達。
~僅以口頭詳細描述路徑，手部維持自然交握避免干擾。#缺乏手勢指引會降低溝通效率，指引時應搭配手掌引導與口頭說明。
~用下巴或眼神示意該方向，並告知顧客「就在那邊」。#使用非手部動作指引是非常不專業且不尊重的行為，無法達成引導效果。
}

::OP-GS-03-Q04::在帶領顧客前往客房或餐廳的過程中，關於「行走細節」的敘述何者正確？ {
~為了展現效率，應走在顧客正前方並快速引導至目的地。#行進速度需配合顧客，不可走得太快導致顧客跟不上或感到壓力。
=走在顧客前側方，遇到樓梯或門口處時需提前口頭提醒顧客。#提前提醒高低差或轉彎能確保顧客安全，且走在前側方方便隨時關照顧客需求。
~行走時不需要說話，等到達定位後再進行最後一次指引即可。#過程中若遇到特殊路況（如階梯）必須提前溝通，而非僅在到達終點時才說話。
~為了讓客人能清楚看到路徑，帶位人員應走在顧客後方引導。#帶位者應走在顧客前側方發揮領航作用，走在後方無法有效指引方向。
}
