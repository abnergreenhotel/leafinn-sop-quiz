$CATEGORY: OP-FO-23 非住宿旅客參訪

::OP-FO-23-Q01::當一位訪客步入大廳，且並非飯店住客時，櫃檯夥伴的第一步行動應為何？ {
~請對方先在沙發區稍坐，並遞上茶水招待#尚未確認來客身份與意圖前，過早安排座位可能導致後續流程不順或影響大廳秩序。
=主動詢問來客身份以及進來飯店的具體目的#優先確認身份與來意是標準作業流程的第一步，能協助後續判斷應適用的處理規範。
~直接帶領訪客前往電梯口介紹環境#未經身份確認與登記便讓非住宿旅客進入客房層，會產生嚴重的安全管理漏洞。
~要求訪客立即出示身分證件進行登記#在確認來意前直接要求證件可能顯得唐突，應先透過詢問引導出客人的參訪需求。
}

::OP-FO-23-Q02::某位訪客表示自己是主管 Kevin 的熟人想參觀，但你查閱交接事項後發現今日並無相關預約，應如何處理？ {
=請顧客留下姓名及手機，聯繫當班主管或 Kevin 確認，並請客人在一樓稍候#當交接事項未記載時，必須先取得連繫資訊並向上呈報，且嚴禁在確認前讓客人才上樓。
~既然是主管的熟人，可先行帶領其參觀以示禮貌，事後再補報備#這違反了「切勿直接讓顧客上樓」的禁令，且可能造成主管行程安排上的困擾。
~告知目前無預約故無法接待，請顧客自行撥打電話聯繫主管#這種做法缺乏服務熱誠，應由櫃檯協助聯繫主管，而非讓客人自行處理。
~請顧客在大廳稍坐，不需登記資料，直接等主管下樓巡視時再處理#未留下基本資料將導致無法追蹤來訪紀錄，且被動等待會導致客人感受不佳。
}

::OP-FO-23-Q03::若聯繫主管後確認該訪客確實為公司高層熟識，且當下營運情況允許，後續應如何安排？ {
~由櫃檯夥伴親自離崗帶領訪客進行全館介紹#櫃檯人員需留守崗位，且流程規定確認熟人身份後應由主管進行專業介紹。
~請訪客自行搭電梯上樓，並請其參觀完後回櫃檯領取紀念品#非住宿旅客的參觀應由人員陪同導覽，而非由其獨自在館內活動。
=請值班主管下樓接待，並進行全館瀏覽介紹#高層熟人參訪在確認身份後，由主管親自帶領介紹能確保服務品質與專業度。
~僅能指引其在 1 樓展覽區活動，不可前往客房層#針對確認過的熟識貴賓，在情況允許下是可以進行全館導覽的。
}

::OP-FO-23-Q04::一對情侶路過飯店想參觀環境，此時已是晚上 7 點半，櫃檯夥伴應如何指引？ {
~請其登記資料後，開放 11 點前的時段供其上樓快速參觀#晚間 7 點後已超過參訪時間，不得開放上樓，應嚴格遵守時段限制。
=告知目前的參訪時間已過，僅能參觀一樓大廳，並需留意其動向#超過參訪時限（11\:00-19\:00）的民眾僅限於 1 樓活動，且人員需維持現場監控。
~為了顧客滿意度，破例開放並請其於 10 分鐘內離開客房區#非預約民眾的參訪時間限制是為了維護住宿旅客的隱私與安全，不應隨意破例。
~請其先至櫃檯登記，再告知可由其自行上樓參觀公設#不論何時，一般路過民眾均不應在無人員引導下自行前往客房樓層。
}

::OP-FO-23-Q05::一組由 4 位朋友組成的參訪團體欲參觀飯店，關於登記作業的正確流程為何？ {
~為了安全起見，要求 4 位參訪者皆必須留下姓名與手機#針對多人參訪，SOP 規定僅需留一位代表即可，以兼顧行政效率。
=請其中一位代表留下姓名及聯絡電話進行登記即可#這是符合多人參訪作業說明的簡化登記方式，只要有一位負責人資訊即可追蹤。
~不需登記資料，直接由櫃檯口頭引導參觀路線即可#遺漏登記手續將導致安全管理斷點，且無法符合非住宿旅客參訪的基本規範。
~要求其出示身分證件並拍照存檔後方可進入#SOP 僅要求留下大名與電話，過度的個資索取可能引發客訴。
}

::OP-FO-23-Q06::當客人明確表達「我想要預訂下週的房間，但想先看過實際房型」，櫃檯夥伴應如何安排？ {
~直接給予客人空房鑰匙卡，請其自行上樓查看#嚴禁讓非住宿者單獨持有鑰匙卡或獨自進入房間，必須有同仁陪同。
~告知現在是參訪時間以外，請其明天中午再來參觀#欲訂房客人的看房需求應積極處理，而非僵化地套用一般參訪的時間限制。
=通知值班主管，請主管親自帶領顧客參觀空出的房間#針對有訂房意願的參觀者，應由主管引導參觀，能提高轉單機會並確保管理安全。
~請客人至官方網站查看照片即可，現場不提供參觀服務#這會導致潛在客戶流失，現場參觀是飯店銷售的重要環節。
}

::OP-FO-23-Q07::多位民眾專程前來參觀飯店內舉辦的藝術展覽，夥伴在引導時應注意什麼？ {
~不論展覽性質，一律禁止參觀者拍照以維護版權#SOP 指出應「依照展覽設計」指引，而非強制設定統一禁令。
=按照展覽簽到方法請參觀者確實簽到，並指引規定的參觀路線#確實簽到能掌握展覽人流，指引特定路線則可避免參觀民眾誤入房務工作區。
~請參觀者自行在大廳活動，不需額外簽到或引導#漏掉簽到步驟會導致活動數據缺失，且無引導可能造成人潮混亂。
~僅限已訂房旅客才能參觀，拒絕路過民眾進入展區#展覽參觀通常對外開放，這與非住宿旅客參訪的初衷相符。
}
