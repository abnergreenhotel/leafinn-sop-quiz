$CATEGORY: OP-FO-09 入住介紹說明

::OP-FO-09-Q01::當客人預訂的房型獲得免費升等（UG）時，櫃檯夥伴在介紹房號後，最適合如何引導客人提供反饋？ {
=主動告知升等訊息，並請客人在住得舒服的前提下，於網路留下滿星好評。#SOP明確指出升等時應主動告知驚喜，並適時邀請客人分享正向評論以提升飯店口碑。
~只提醒房號，等客人退房時再詢問是否滿意，避免在入住初期給予壓力。#這會錯過在第一時間建立好感的機會，且SOP要求在介紹房號前就要先說明升等。
~直接將房卡遞給客人，並強調升等是為了維持住房率，不需要特別提評價。#提到住房率會削弱升等的尊榮感，且未能達成引導好評的作業目標。
~告訴客人因為房間客滿才幫他升等，若沒問題請他在入住單上簽名。#這將升等的價值歸因於飯店內部需求，而非對客人的特別禮遇，不符合服務話術建議。
}

::OP-FO-09-Q02::若客人在一樓大廳休息區詢問 Wi-Fi 帳號，櫃檯夥伴應請客人搜尋哪一個特定的帳號名稱？ {
~greenhotel99#這是 Wi-Fi 的密碼內容，而非大廳專用的搜尋帳號（SSID）。
=greenhotel1f01#SOP規定在大廳使用 Wi-Fi 時，應搜尋此特定帳號，其密碼則與房內相同。
~greenhotel_lobby#這是常見的命名方式，但並非本館 SOP 規定的標準名稱。
~直接搜尋房號即可連接#大廳與客房區的訊號來源不同，應引導客人使用專屬的大廳帳號。
}

::OP-FO-09-Q03::關於客房內的一次性備品提供，下列哪一項做法符合飯店的環保與作業規範？ {
~為了方便客人，入住時直接在房卡套內主動放入牙刷與刮鬍刀。#依照政府規定與環保政策，房內不主動提供一次性備品，應由客人視需求索取。
=在櫃檯指著 A4 立牌，詢問客人是否有備品需求或忘記攜帶。#透過引導式詢問（指著立牌），既能符合法規減少浪費，也能確保客人需求被滿足。
~告知客人房內抽屜已有完整備品，若用完可以再跟櫃檯索取。#這與 SOP 提到的「房內不提供牙刷、刮鬍刀等備品」規範相左。
~告訴客人備品需付費購買，以落實環保愛地球的精神。#SOP僅提到不主動放置於房內，若有需要仍可至櫃檯領取（或詢問有無需要），並未提及收費。
}

::OP-FO-09-Q04::一位客人想要體驗館內的「心之光牆」互動，請問他在什麼時段可以透過各樓層的平板電腦進行操作？ {
~早上 07\:00 至 09\:30#這是早餐供應的時間段，並非光牆互動的開放時間。
~下午 15\:00 至 晚上 22\:00#此時間段包含了入住尖峰期，但比 SOP 規定的互動開始時間早，且結束時間晚。
=下午 17\:00 至 晚上 21\:30#根據 SOP 介紹，光牆互動僅在此特定夜間時段開放客人使用平板放煙火或投射文字。
~全天候 24 小時皆可使用#互動設備有固定開放時間，並非 24 小時開放，以符合節能及夜間靜謐需求。
}

::OP-FO-09-Q05::客人向櫃檯反映希望換一個比較「貼合頸部」的乳膠枕頭，夥伴應優先建議哪一款枕頭？ {
~獨立枕#獨立枕特點是高度較高且支撐性較強，並非強調貼合頸部的乳膠材質。
~麵包枕#雖然是乳膠枕，但麵包枕的造型為圓弧狀，非特別設計來貼合頸部曲線。
~羽絨枕#這是房內的標準配置，客人正是因為想體驗特殊的枕頭才提出要求。
=曲線枕#SOP 註記中明確指出曲線枕為乳膠材質，且功能是「可以更貼合頸部」。
}

::OP-FO-09-Q06::關於「聞香區」的介紹，下列操作流程何者正確？ {
~引導客人將精油直接滴在房卡上，方便隨時聞香。#精油應滴在房卡套撕下來的書籤或專屬試香片上，直接滴在房卡可能損壞晶片或卡面。
~請客人在精油區選好味道後，由櫃檯夥伴幫忙滴在書籤上。#SOP 設計是讓客人自行試聞並操作滴精油，增加互動體驗感。
=提醒客人聞香區皆有聞香紙，到精油區滴上精油，吸收後即可帶回房內。#這是正確的互動流程，利用書籤作為擴香媒介讓客人隨時感受香味。
~告知客人擴香石可以滴上精油後帶回房間，退房時也能帶走當紀念品。#SOP 規定擴香石不可帶走，退房時必須留在房內。
}

::OP-FO-09-Q07::若遇到地下一樓餐廳公休，需要與「早安有喜」配合外帶餐盒時，午櫃夥伴應在何時完成客人的點餐作業？ {
~等隔天早上客人下樓取餐券時再詢問想吃的口味。#這會導致大夜班無法提前統計數量，造成送餐延誤。
=辦理入住（Check-in）時先請客人填寫菜單，若客人當下無法決定則請其最晚晚上 9 點前送回櫃檯。#這能確保大夜班夥伴有足夠時間在 9 點至 10 點間統計並準備後續作業。
~請客人在晚上 12 點前撥打電話到櫃檯告知餐點即可。#SOP 建議在 9 點前收集完畢以利作業，12 點太晚會影響統計效率。
~客人辦理入住時只需確認是否吃素，不需填寫具體菜單。#外帶餐盒需要詳細記錄餐點、飲品、送餐時間與方式，並需簽名確認。
}

::OP-FO-09-Q08::當客人在下午時段詢問迎賓茶水區提供的熱茶種類時，夥伴應如何回答？ {
~全天候皆提供咖啡機及大吉嶺紅茶。#晚上10點後無提供咖啡機，下午會更換部分種類。
=下午提供的是黑糖薑母茶（冬季限定）與歐式大吉嶺茶。#SOP 區分了早晚茶飲，下午包含季節限定的黑糖薑母茶。
~下午只有提供咖啡機，不提供熱泡茶飲。#咖啡機雖有開放，但茶水區仍有固定提供的下午茶飲。
~下午提供檸檬水與香草植物水，沒有熱茶。#檸檬水與香草水通常屬於迎賓區配置，與茶水區的配置不同。
}

::OP-FO-09-Q09::關於客房內飲用水的使用介紹，下列哪一種說法最符合館內的 SOP？ {
~房內小冰箱已有兩瓶礦泉水，喝完後可再向櫃檯免費領取。#館內響應環保不提供瓶裝水，此回答與現行環保政策不符。
~若需要飲用水，請至一樓櫃檯領取紙杯裝取。#每層樓皆設有飲水機，應引導客人就近使用樓層設備。
=因為響應環保不提供瓶裝水，每層樓都設有飲水機，可利用房內的冷水壺或熱水壺去盛裝。#這完全符合 SOP 的環保說明與設施指引流程。
~房間內的水龍頭打開即可生飲，不需另外盛裝。#台灣多數建築水龍頭水不可生飲，且 SOP 亦指示應使用飲水機。
}
