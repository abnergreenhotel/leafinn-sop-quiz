$CATEGORY: OP-OB-03 規範認同

::OP-OB-03-Q01::在準備新進夥伴的報到文件時，若發現公司最近剛更新了加班申請制度，但手邊還留有幾份舊版的員工手冊，此時最正確的做法為何？ {
~為了環保，先將舊版手冊發放完畢，再口頭補充新規定。#使用過時的規範會導致資訊落差，增加後續勞資爭議的風險。
=全面換成最新版文件，並主動提供制度更新的相關公告。#確保資訊準確性是報到準備的首要原則，且必須一併提供最新的異動說明。
~在舊版手冊上用紅筆修正更新處，請夥伴在修正處旁簽名。#官方文件應保持整潔且版本統一，手工塗改容易造成權益界定模糊。
~直接移除舊版手冊中過時的頁面，僅保留不變動的部分。#文件完整性非常重要，隨意拆分頁面會導致夥伴閱讀內容不連貫且失去規範效力。
}

::OP-OB-03-Q02::當你在向新進夥伴說明「規範內容」時，下列哪一種執行方式最符合服務標準作業指導書的要求？ {
~請夥伴先自行閱讀一小時，閱讀完後直接在末頁簽名即可。#SOP 明確規定不能僅要求簽名，必須先完成重點的口頭說明。
~將重點彙整成投影片，僅針對出勤與請假規則進行快速帶過。#除了出勤外，服儀、工作紀律與獎懲制度等皆是必須說明的重點項目。
=說明員工手冊重點（含紀律與獎懲），並針對疑問處即時回應。#這符合 SOP 中要求的依序說明重點項目，並包含即時回覆或記錄疑問的流程。
~為了加速流程，告知夥伴內容與一般業界相同，只需確認個人資料。#每家公司的勞動條件與保密義務各異，簡化說明可能導致夥伴違反特定規定。
}

::OP-OB-03-Q03::在說明勞動條件時，新進夥伴對於「試用期」的法律定義提出了較為專業的法規疑問，而你並不完全確定細節，此時應如何處理？ {
~根據自己的理解先行回覆，以免顯現出專業度不足。#錯誤的資訊回覆可能會造成後續法律風險或夥伴的誤解。
~告知夥伴試用期是業界常態，請其先簽署文件以利報到完成。#規避問題無法解決疑義，且 SOP 規定涉及法規疑義時有特定的處理對象。
=將疑問記錄下來，隨後轉請主管或人資單位進行更精準的補充。#當涉及薪資、排班或法規等專業疑義時，由相關專責窗口回覆是標準做法。
~請夥伴自行上網查詢相關勞基法條，確認後再決定是否簽署。#公司有義務對其制定的規範內容進行正確的說明，而非讓員工自行推測。
}

::OP-OB-03-Q04::在引導新進夥伴簽署文件的過程中，下列哪一個行為是明確禁止的？ {
~要求夥伴在每一份文件的日期欄都要填寫當天日期。#這是正確的要求，確保文件生效日期的正確性與完整性。
=因為夥伴正在接受急單教育訓練，由你代為在保密協議上簽名。#SOP 嚴格禁止代簽行為，文件必須由當事人親自閱讀並完成簽署。
~在需要簽名的頁面貼上便條紙提醒，避免夥伴漏簽。#這屬於預先準備的範疇，有助於提高作業效率與正確率。
~請夥伴逐份閱讀內容後，再開始進行簽署動作。#這是 SOP 要求的標準步驟，確保夥伴在知悉內容的前提下簽署。
}

::OP-OB-03-Q05::文件簽署完成後，負責人員正準備進行「複核」步驟。此時最重要的一項檢查內容是什麼？ {
~確認夥伴的字體是否端正清晰。#雖然清晰度重要，但字體端正與否並非 SOP 複核流程中的核心安全檢查點。
~立即掃描成電子檔存入雲端硬碟備份。#複核應先於歸檔，確保紙本文件完整後再進行後續的行政處理。
=逐頁檢查是否有漏簽名、漏填日期或頁面附件不齊全的情況。#逐頁檢查漏簽與漏附文件是確保報到法律文件具備完整效力的關鍵步驟。
~詢問夥伴是否已經記住所有手冊內容。#複核階段的重點在於「文件完整性」的檢查，而非記憶力測驗。
}

::OP-OB-03-Q06::如果在複核時發現夥伴漏簽了一份「勞資規範同意書」，但該夥伴已經前往其他部門工作。根據規範，這筆漏件應何時處理完畢？ {
~等到試用期滿前，一併檢查所有缺失並補正。#延遲處理會導致報到流程處於未完成狀態，增加管理上的風險。
=必須於發現漏件的「當日」聯繫夥伴完成補件。#SOP 明確規定若有缺件，需當日完成補件，且文件複核完成前不得直接歸檔。
~由負責人員自行補上夥伴姓名，並在交接紀錄註記。#這屬於代簽行為，嚴重違反作業規範且具法律風險。
~註記在待辦事項中，等到下一次會議時再找夥伴簽署。#SOP 要求的是即時性，且報到流程未完成前，人員的身分規範認同尚不完整。
}

::OP-OB-03-Q07::新進夥伴報到流程結束後，簽署好的個人文件暫時還沒交給主管。關於這些文件的放置位置，下列何者正確？ {
~暫放在櫃檯顯眼處，方便主管經過時直接拿走。#個人文件包含隱私與個資，不可任意放置於公共區域。
~夾在常用的交接簿中，讓下一班夥伴協助遞交。#這可能導致個資外洩給非相關人員，且公共交接簿並非妥善保管場所。
=妥善保管於指定位置，不可暴露於公共區域。#基於個資保護與作業規範，個人簽署文件必須妥善收納保管。
~放在夥伴休息室的桌上，讓夥伴自己隨時可以查看。#休息室亦屬於公共區域，且已簽署回傳的文件應由公司統一管理。
}

::OP-OB-03-Q08::關於報到流程的「完成判定」，下列敘述何者最符合 SOP 精神？ {
~只要夥伴領取了員工手冊，即可視為報到流程完成。#領取並不代表認同或簽署，規範認同的核心在於「簽署」。
=若新進夥伴尚未完成所有規範文件的簽署，不可視為報到流程完成。#簽署文件是報到法律流程的最後一道防線，未完成則流程不算正式結束。
~只要口頭說明完畢，就算夥伴拒絕簽名也視為已完成程序。#口頭說明與書面簽署缺一不可，拒絕簽署代表流程中斷。
~夥伴已在現場排班工作，系統會自動預設報到流程已完成。#報到流程包含實質的行政手續，排班工作並不等於行政文件已備齊。
}
