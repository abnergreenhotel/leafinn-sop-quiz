::OP-OB-01-Q01::當旅客在夜間抵達旅館，稱讚建築外觀非常有設計感時，身為服務人員，下列哪一項關於外觀的介紹最精確？ {
=這是以「樹木造型」為靈感設計的建築，不論白天或晚上觀看都具備獨特的藝術風格。#正確，因為此介紹符合外觀建築以「具設計感的樹木造型」為核心且晝夜皆具風格的特點。其他選項提及的回收材料外牆或玻璃帷幕並非本導覽手冊中對外觀的主要定義。
~整棟建築是由26000個回收寶特瓶砌成，所以晚上的燈光反射特別漂亮。#不適合，因為26000個回收寶特瓶是用於室內「心之光牆」，而非建築外部結構。這樣回答會誤導旅客對建築工法的認知。
~為了環保，外牆採用純自然木材搭建，旨在減少城市中的水泥感。#不適合，因為手冊中僅提到外觀為「樹木造型」，並未說明外牆是純木材搭建，應避免提供未經證實的建材資訊。
~建築造型象徵著「希望種子」，這也是整棟旅館唯一的設計主題。#不適合，因為「希望種子」僅是其中一個樓層的空間主題，並非整棟建築外觀的唯一象徵，且外觀主打的是設計感的樹木造型。
}

::OP-OB-01-Q02::一位商務旅客在辦理入住時顯得疲憊不堪，您想推薦他到一樓大廳的「聞香區」放鬆，下列哪項說法最符合導覽建議？ {
=我們櫃台前設有精油聞香區，提供10種味道的精油，可以幫助您舒緩旅途的疲憊。#正確，因為手冊明確指出聞香區提供10種味道的精油，且目的即是舒緩旅途疲憊。其他選項提及的味道數量或功能皆不符。
~大廳提供各種多肉植物，您可以採摘葉片聞香，感受自然的香氣。#不適合，因為聞香區是指「精油」而非多肉植物，且多肉植物是觀賞用的，不應引導旅客採摘。
~聞香區有提供免費的櫻桃咖啡試喝，香氣能夠提神醒腦。#不適合，因為櫻桃咖啡屬於展售區項目，與提供10種味道精油的聞香區功能不同。
~您可以到聞香區體驗，那裡有專業的芳療師隨時為您進行精油按摩。#不適合，因為手冊僅提到「聞香區」提供精油舒緩疲憊，並未提到有現場芳療師或按摩服務，應避免擴大承諾服務內容。
}

::OP-OB-01-Q03::當旅客在一樓大廳看到高達20公尺的藝術牆面時，驚訝地詢問其設計由來，您應該如何正確說明？ {
=這面牆是由26000個回收寶特瓶組成的，除了壯觀外，更象徵我們友愛大地的環保理念。#正確，因為此敘述完整包含了手冊提到的回收寶特瓶數量與環保核心價值。其他選項對於牆體材料的說明與實際導覽內容不符。
~這面牆完全是由多肉植物編織而成的「植生牆」，是為了增加室內含氧量。#不適合，因為「心之光牆」主要是由回收寶特瓶構成，植生牆只是其中的一部分特色，數量與主體不應混淆。
~牆面上的綠色葉子是請藝術家手工噴漆上去的，象徵城市森林的脈動。#不適合，因為牆面效果是透過回收材料與LED光影呈現，而非單純手工噴漆，且這樣解釋忽略了最重要的環保材料價值。
~這座牆是為了支撐中央天井的結構，避免旅館因老舊而倒塌。#不適合，因為這面牆是「藝術牆」，手冊中提到的中央天井是為了「自然採光」以減少用電，而非結構支撐，此說法會引發旅客安全疑慮。
}

::OP-OB-01-Q04::一位帶著小孩的旅客詢問，除了休息之外，館內是否有可以讓孩子參與的有趣活動，下列哪項引導最合適？ {
=您可以利用樓層平板操作互動APP，讓孩子在寶特瓶LED牆面上施放虛擬煙火。#正確，因為手冊提到的各樓層平板設置互動APP，能讓客人在LED牆施放煙火及祝福，是館內具體的互動體驗。其他選項的活動皆非手冊內容。
~我們有提供免費的DIY課程，教小朋友如何製作水草燈。#不適合，因為手冊僅提到「水草燈」是裝飾設施，並未提到有提供DIY教學活動，應避免自行承諾不存在的課程。
~在B1餐廳有專門的兒童遊樂區，裡面有許多環保玩具。#不適合，因為導覽中對於B1餐廳的描述為座位區與早餐區，並未提到兒童遊樂區或環保玩具設施。
~大廳展售區的所有商品，小朋友都可以隨意拿取體驗。#不適合，因為展售區商品是販售項目，應引導旅客參觀而非隨意拿取，且此回答未提供實質的活動內容。
}

::OP-OB-01-Q05::旅客辦理入住後詢問：「如果我想要去附近的審計新村逛逛，建議怎麼去？」，下列哪一種回應最準確？ {
=審計新村距離我們非常近，步行約10至15分鐘即可抵達。#正確，因為導覽手冊明確列出審計新村步行時間約為10\~15分鐘。其他時間長度皆不符合手冊標註的實際距離。
~走路大約要半小時，建議您直接搭乘我們的預約接駁車。#不適合，因為手冊標註步行僅需10\~15分鐘，且手冊提到的交通相關服務為「叫車服務(Taxi)」而非專屬接駁車。
~審計新村就在勤美誠品旁邊，走路5分鐘就到了。#不適合，因為雖然鄰近，但步行5分鐘過於樂觀，應以手冊標註的10\~15分鐘為準以避免旅客預期落差。
~那邊目前在施工，我們比較建議您去步行20分鐘的自然科學博物館。#不適合，因為不應在無確切資訊下告知旅客景點施工，且轉移目標不符合旅客當下的導覽需求。
}

::OP-OB-01-Q06::若旅客在辦理退房後，表示想在附近繼續觀光但行李太多，您可以如何利用館內服務協助他？ {
=主動告知旅館有提供貼心的行李寄放服務，讓他能輕鬆遊覽景點。#正確，因為行李寄放是手冊中明確列出的貼心服務之一。其他選項提到的寄送或託管方式非手冊內容。
~建議他將行李放在一樓大廳的展售區，我們有專人代為保管。#不適合，因為展售區是販售商品的地方，並非指定的行李存放空間，會造成大廳動線混亂。
~告知目前無寄放服務，但我們可以幫忙預約快遞將行李送至車站。#不適合，因為手冊明確有提供寄放服務，若回答無此服務將導致旅客不便且違反作業標準。
~行李必須隨身攜帶，但您可以租借我們的腳踏車，將行李掛在車頭。#不適合，因為雖然有腳踏車租借，但將大量行李掛在腳踏車上非常危險且非解決行李問題的正確做法。
}

::OP-OB-01-Q07::旅客問到：「你們旅館的早餐是在哪裡吃？名字有什麼特別意義嗎？」，下列說明何者正確？ {
=早餐在B1的「旭暮」餐廳，名字取自歌劇院『序暮』之意，希望帶給您美好一天的開始。#正確，因為此說明完全符合手冊中對於B1餐廳「旭暮 mu café」的命名涵義描述。其他選項的命名解釋皆為杜撰。
~早餐在一樓大廳的品茶區，名為「綠意」，象徵綠色生活的開端。#不適合，因為早餐區位於B1而非一樓，且餐廳名稱應為「旭暮」。
~餐廳名為「森呼吸」，是希望旅客在用餐時能感受到城市的氧氣。#不適合，因為「森呼吸」是旅館的品牌標語（Slogan），並非B1早餐餐廳的正式店名。
~早餐店名是「OhBear」，是因為我們與喔熊合作，希望能吸引親子客群。#不適合，雖然有喔熊聯名商品，但餐廳名稱為「旭暮」，混淆名稱會讓旅客找不到用餐地點。
}

::OP-OB-01-Q08::關於旅館的低碳環保措施，若旅客好奇我們如何具體「友愛大地」，下列哪一項敘述是錯誤的？ {
=我們所有客房都不提供冷氣，完全依靠中央天井的自然風循環來降溫。#正確(此敘述錯誤)，因為手冊提到中央天井是為了「自然採光」減少「照明用電」，而非完全不提供冷氣。其他選項皆為正確的環保作為。
~我們鼓勵旅客不使用一次性備品，並提供環保餐具的租借服務。#不適合(此為正確措施)，手冊確實提到鼓勵不使用一次性備品，並在服務清單列出環保餐具租借，這是落實理念的作為。
~大廳的藝術牆使用26000個回收寶特瓶製成，落實廢棄物再利用的價值。#不適合(此為正確措施)，這確實是館內最顯眼的環保硬體設計，符合友愛大地的核心理念。
~我們獲得了台中市低碳旅館認證，這代表在節能減碳上有達到政府標準。#不適合(此為正確事實)，手冊中明確列出了行政院環保旅店與台中市低碳旅館認證的殊榮。
}
