$CATEGORY: OP-FO-12 續住作業流程

::OP-FO-12-Q01::當住客表達想要續住時，櫃檯人員完成房號與房型空房確認後，下一步最優先的房務安排原則是什麼？ {
~不論房型是否相同，直接安排至當日最容易清理的空房#隨意更換房間會增加客人的行李搬運負擔，也會增加房務重新整備一間房的成本。
=優先安排入住原房號，以避免不必要的換房作業#維持同一房號入住能簡化行政流程，並提升住客的便利性，符合作業標準中避免換房的原則。
~請客人先退房並將行李寄放櫃檯，等下午三點後再重新入住#這屬於重新入住流程而非續住流程，會造成客人的極大不便。
~強制要求客人升等至較高等級的空房，以增加營收#續住作業應以滿足客人需求與穩定房況為主，而非在流程中強迫推銷升等。
}

::OP-FO-12-Q02::在確認住客要續住後，櫃檯人員應如何與房務單位進行後續的溝通協作？ {
~等當日所有住客退房後，再統一彙整名單給房務群組#延遲通報會導致房務人員無法及時排進工作排程，影響清潔效率。
~僅需告知房務部有房間續住即可，不需說明清潔需求#若未告知清潔需求與出門時間，房務人員將無法判斷何時進場作業。
=應立即通知房務，並註明客人是否需要清潔及預計出門的時間#即時通報清潔需求與客人動向，能讓房務團隊精確掌握工作節奏，確保服務品質。
~請客人自行撥打房內分機向房務部預約清潔時間#櫃檯應作為資訊轉達的中心，由櫃檯統一回報群組才是標準流程。
}

::OP-FO-12-Q03::住客詢問是否可以安排在下午三點以後才進行續住房間的清潔，根據服務標準，櫃檯人員應如何回覆較為適當？ {
~直接答應客人的要求，以客為尊#這違反了 SOP 中關於最晚清潔時間的規範，可能導致房務人員下班前無法完成工作。
=委婉告知客人，續住房間的最晚清潔受理時間為下午二點前#明確告知服務限制時間，有助於管理客人的期待並確保房務作業能在規定時間內完成。
~告訴客人下午三點房務已經下班，完全無法提供任何服務#這種回覆方式過於生硬，且未提供 SOP 中建議的替代時間點或標準說明。
~告知客人若要在下午三點清潔，必須額外加收清潔費用#SOP 中並未提到加收費用的規定，應以標準時間限制進行溝通。
}

::OP-FO-12-Q04::關於續住房費的收取流程，下列哪一項操作符合作業指導書的規範？ {
~要求客人必須在表達續住意願的當下立即付清房費，否則不予續住#過於強硬的態度可能損及服務體驗，SOP 建議的是約定一個合適的時間。
~告知客人房費可以等最後一天退房時再一併結清#為了降低欠費風險，SOP 規定應在續住當日的約定時間前完成支付。
=與客人約定一個時間，並請其在該時間前至櫃檯支付房費#透過事先約定付款時間，既能給予客人彈性，又能確保飯店能按時入帳。
~直接從客人的信用卡授權中扣款，不需另行通知客人#金流作業應與客人明確溝通付款時機，直接扣款若未經說明容易引發糾紛。
}

::OP-FO-12-Q05::在執行續住作業的話術中，當發現飯店「目前已無空房」時，正確的處理方式為何？ {
~建議客人直接在走廊等待，看有沒有人提早退房#這並非專業的飯店應對方式，且會造成公共空間的混亂。
=立即向客人致歉並明確告知目前已無空房，無法提供續住#透明且誠懇的溝通是處理房況不足的基礎，能讓客人及早做其他安排。
~私下聯繫其他同業，強行將客人轉介過去並收取出差費#SOP 僅規範了告知無房的話術，未授權現場人員進行此類複雜的外部轉介操作。
~請客人在櫃檯稍坐，由櫃檯人員不斷刷新系統直到有房為止#若當日已滿房，被動等待會浪費客人的時間，應依照標準話術直接回覆。
}

::OP-FO-12-Q06::在續住作業的注意事項中，關於房價的處理原則，下列敘述何者正確？ {
~因為是續住，所以必須以當日的最高門牌價（Rack Rate）計費#這不符合鼓勵客人續住的銷售邏輯，且偏離了 SOP 的建議。
=續住期間飯店可以視情況提供房價折扣#提供折扣是續住作業中常見的優惠措施，有助於提高住客滿意度與留客率。
~續住一律不給予折扣，且需加收續住手續費#SOP 明確提到「續住可給房價折扣」，而非加收額外費用。
~折扣額度由房務人員在清潔時根據房間整潔程度決定#房價與折扣屬於前台行政與業務範疇，並非由房務人員決定。
}
