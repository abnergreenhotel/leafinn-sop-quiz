$CATEGORY: OP-FO-11 退房作業流程

::OP-FO-11-Q01::當旅客報出的房號與系統顯示的登記人姓名不符時，櫃檯人員第一步應如何處理？ {
~請旅客出示身分證件，並直接在德安系統修改登記人姓名。#隨意修改原始登記資料會導致帳務與身分追蹤混亂，且未先確認原登記人身分。
=詢問旅客入住時登記人的名稱，並透過系統的「房間管理」功能核對資料。#這是標準的核對程序，透過詢問登記人姓名並在德安系統中進行交叉比對，以確保身分相符。
~直接告知旅客資料不符，請其確認正確房號後再重新辦理退房。#此做法過於消極，應主動詢問登記人名稱以協助旅客找出可能的紀錄偏差。
~先幫旅客辦理退房，待稍後房務檢查房間時再行確認。#在身分未確認前辦理退房會導致帳務錯誤風險，必須在結帳前完成核對。
}

::OP-FO-11-Q02::旅客辦理退房時，表示不小心弄丟了一張房卡，根據作業標準應如何收費？ {
=向旅客收取遺失費用一百元。#SOP 明確規定若遺失房卡，需依規定收取每張一百元的工本費。
~請旅客至大廳沙發稍候，待找到後再辦理，不另收費。#遺失應立即按價賠償，以免延誤旅客行程或導致資產流失。
~檢查房間設備是否完整，若無損壞則可免收房卡賠償費。#房卡賠償與客房設備損壞是兩項獨立的收費項目。
~收取兩百元作為房卡磁條重新設定的費用。#金額不符合 SOP 定價，SOP 規定一張房卡為一百元。
}

::OP-FO-11-Q03::旅客欲歸還借用的「酒杯」與「餐具組」，櫃檯人員應採取什麼行動？ {
~請旅客自行拿至餐廳交給洗碗人員清潔。#櫃檯應作為服務窗口處理物品回收，不應增加旅客負擔。
=詢問房務部是否已收到相關物品並確認清洗完畢。#針對需要清洗的借用物，需透過房務端確認物品現況與清潔進度。
~直接收下並放置於櫃檯後方，等有空再通知房務。#借用物若含有食物殘渣或髒污，未及時通知房務處理會影響環境衛生及後續備品使用。
~告知旅客餐具不需回收，可直接留在房間內。#退房時必須確認所有租借物品是否繳回，以利資產盤點。
}

::OP-FO-11-Q04::旅客在 11\:30 退房時間後提出希望「再多待三小時」，櫃檯應如何處理？ {
~告知每小時酌收兩百元，共收取六百元，並幫其延長時間。#雖然收費金額正確，但 SOP 規定加休最多只能延長兩小時。
=委婉告知加休上限為兩小時，並通知房務部調整後的退房時間。#符合 SOP 中關於加休上限兩小時的規定，且正確履行了通知房務的義務。
~請旅客先辦理退房，行李寄放後再到大廳休息，不提供加休服務。#飯店提供收費加休服務，不應在未溝通限制條件的情況下直接拒絕。
~直接同意加休需求，待旅客離開時再根據實際時間收費。#加休需事先確認時間、告知收費標準並立即通知房務部，不能事後才結算。
}

::OP-FO-11-Q05::一位旅客透過 Agoda 平台訂房，退房時要求櫃檯開立「含公司統編的紙本發票」，應如何應對？ {
~在德安系統中補入統編，直接列印電子發票交給旅客。#透過特定訂房平台（如 Agoda）訂位時，飯店端不開立發票給旅客。
=委婉說明訂房平台之發票需向原訂房平台申請，飯店無法重覆開立。#針對 Agoda, EXP, KLOOK 等平台，SOP 明確指出旅客需向原平台索取發票。
~請旅客提供電子信箱，告知退房後系統會自動補發電子發票。#飯店端並無發票開立紀錄可供補發，此回應會誤導旅客。
~開立一張住宿證明給旅客，並在房費欄位註明「已開立發票」。#這與發票申請問題無關，且未解決旅客對於報帳憑證的需求。
}

::OP-FO-11-Q06::若旅客是透過「旅行社付款」，在協助開立「住宿證明」時，下列哪項做法是正確的？ {
=住宿證明上不可顯示房費金額。#SOP 規定，若旅客為旅行社付款（飯店未開立發票予旅客），住宿證明不可顯示房費。
~照常填寫房費，但在下方蓋上「旅行社已付」章戳。#與 SOP 規定的「無發票者不可顯示房費」原則相違背。
~請旅客自行向旅行社索取住宿證明，飯店端不提供。#飯店仍可提供住宿證明（包含姓名、日期等資訊），只是房費欄位有顯示限制。
~僅在住宿證明上顯示稅額，不顯示總價。#只要沒有開立發票給旅客，任何與房費相關的具體金額都不應顯示。
}

::OP-FO-11-Q07::在德安系統進行「旅客帳處理」的結帳流程中，選擇付款方式後，最後的操作步驟為何？ {
~點選「入帳」並輸入金額。#入帳是結帳前的準備步驟，並非最後一步。
~點選「房間管理」並重新整理房態。#這只是房態確認，不代表完成結帳手續。
=點選「結帳」，並在彈跳視窗選擇「確定退房」。#這是系統操作的最後關鍵步驟，確保房態轉為已退房且帳務關閉。
~直接關閉視窗，旅客住宿卡會自動歸檔。#未點選確定退房會導致系統房態錯誤，且住宿卡需人工放置於櫃檯下方留存。
}

::OP-FO-11-Q08::詢問顧客住房滿意度時，旅客反映房間排水孔有異味導致睡眠不佳，應如何依流程處理？ {
~告知旅客我們會轉達給工務組，並請其下次入住再觀察。#回應過於官條，未展現即時關懷或致歉。
=先向旅客誠懇致歉、視情況送禮物賠罪，若無法處理應立即通知主管。#這是 SOP 規定的客怨處理原則：關心、道歉、補償、必要時上報。
~解釋是因為天氣氣壓影響導致異味，請旅客體諒。#找藉口會讓旅客感受不佳，應優先以客人的感受為重。
~直接給予房費折讓，不需要打擾主管。#SOP 規定若無法處理（如涉及金額折讓等授權）必須通知主管，不可擅自決定。
}

::OP-FO-11-Q09::關於退房後的「送客服務」，下列哪一項做法符合 SOP 的規範？ {
~在櫃檯內微笑點頭致意即可。#SOP 要求的送客包含更多主動服務，如開門或拿行李。
=幫客人開門、拿行李，若遇雨天需主動幫顧客撐傘。#SOP 明確列出開門、拿行李、雨天撐傘與目送離開等細節。
~等客人走到門口後，大聲喊出「期待下次入住」。#送客應包含肢體服務，而非僅有口頭道別。
~請行李員處理即可，櫃檯人員需留在座位處理行政工作。#SOP 指導書中並未區分職位，這是每位現場夥伴應執行的送客標準。
}

::OP-FO-11-Q10::退房手續完成後，旅客填寫的「住宿登記卡」應如何管理？ {
~直接放入碎紙機銷毀，以免資料外洩。#登記卡需保留存查，不可立即銷毀。
~放置於櫃檯桌面，方便下一班交接核對。#桌面屬於公共區域，隨意放置會導致個人個資外洩。
=統一放置於櫃檯下方留存，並嚴禁洩漏個資。#SOP 明確規定住宿登記卡應放置於櫃檯下方留存，並強調個資保護。
~交由房務人員帶回辦公室存檔。#房務人員不負責前台帳務與資料管理，應由櫃檯單位依規範留存。
}
