$CATEGORY: OP-FO-02 客務基本認知

::OP-FO-02-Q01::當客人想要租借館內較為貴重的物品時，櫃檯人員應該如何處理押金相關事宜？ {
~一律要求支付與物品價值相等的現金押金#雖然貴重物品可能需要押金，但 SOP 規定應先詢問主管後再做決定，而非一律自行決定金額。
~請客人在租借單上簽名確認即可，無需額外處理#這忽略了 SOP 中針對「貴重物品」的特殊處理原則，可能導致資產損失風險。
=先行詢問主管，再根據指示決定是否收取押金#根據客務認知 SOP，一般物品不收押金，但貴重物品必須經過主管授權後決定租借條件。
~請客人抵押身分證件，待歸還時退回#SOP 中並未提到抵押身分證件的做法，通常以詢問主管決定押金金額為主。
}

::OP-FO-02-Q02::準備將腳踏車交給客人租借前，哪一項是現場必須執行的實體檢查重點？ {
=確認輪胎是否有充足氣體#SOP 特別強調租借前需確認輪胎胎壓，以確保客人的騎乘安全與產品正常使用。
~先行幫客人將坐墊調整到最低高度#每位客人的高度需求不同，應視情況協助調整，而非一律調到最低。
~清潔整台車架並打蠟#雖然保持清潔很重要，但 SOP 中明確提到的安全檢查重點是輪胎狀況。
~確認車鈴聲響是否清脆#這屬於一般維護，但 SOP 提到的關鍵檢查點在於輪胎的胎壓。
}

::OP-FO-02-Q03::若客人欲租借摩托車並已確認有車，櫃檯在填寫合約書後的後續處理步驟為何？ {
~請客人退房時再與房費一併結帳#根據摩托車租借規範，確定有車後必須先行收款，而非離店時支付。
=必須先向客人收取現金費用#SOP 明確規定若確定有車可借，需要先向客人收取現金款項。
~引導客人直接前往車行，由車行自行收款#流程規定由櫃檯先確認數量、填寫合約與派車單，並由櫃檯端完成預收動作。
~請客人在 R 卡（登記卡）上簽名即可#摩托車租借涉及外部廠商合作，流程中強調需先收取現金並填寫專屬合約書。
}

::OP-FO-02-Q04::當房客在房間內打電話到櫃檯索取牙刷與梳子時，夥伴應如何根據環保與備品政策回應？ {
=告知房內不提供一次性備品，請房客自行至櫃檯領取#飯店政策規定房內不提供牙刷等一次性物品，且需引導客人至櫃檯領取。
~立即安排房務人員將備品送往房間#這違反了 SOP 中要求顧客至櫃檯領取一次性備品的規範。
~告知飯店完全不提供此類物品，請客人到附近超商購買#飯店仍有提供這些備品，只是領取地點在櫃檯而非預置於房內。
~詢問房客房號，紀錄在 R 卡後請房務補入#針對一次性備品，SOP 的流程是請房客到櫃檯領取，並非由房務補入房間。
}

::OP-FO-02-Q05::有一組客人（共 2 位大人）辦理入住，並表示會多加一位大人同行且需要含早餐，應如何告知收費？ {
~加收 300 元，提供一份備品但不含早餐#300 元是不含早餐的加人價格，不符合客人含早餐的需求。
~加收 500 元，並安排加床服務#雖然金額正確，但飯店明確規定「無加床服務」。
=加收 500 元，提供一份額外備品與早餐#根據 SOP，含早餐的加人費用為 500 元，包含一份備品，且無加床。
~告知因為房型限制，無法增加人數入住#SOP 有提供加人收費標準，表示在規範內是可以接受加人入住的。
}

::OP-FO-02-Q06::在處理行李寄放時，櫃檯夥伴使用「黃綠色」的行李吊牌，這代表該行李是什麼狀態？ {
~客人的行李需要冷藏或冷凍處理#冷藏或冷凍行李應使用灰色的行李吊牌。
~客人是剛抵達、尚未辦理入住的預進房客#預進客人的行李吊牌應使用綠色。
=客人已經辦理完退房，暫時寄放行李#SOP 規定行李牌分類中，黃綠色代表「退房」客人的寄放。
~該行李內容物包含貴重物品，需特別留意#行李吊牌的顏色是用來區分客人的入住狀態（預進/退房）或特殊儲存需求（冷藏）。
}

::OP-FO-02-Q07::現場有一組 4 位成人要前往機場，攜帶了 3 件 24 吋行李箱，櫃檯應如何協助叫車？ {
=直接撥打 55688 告知需要大型車輛#SOP 注意事項提到，4 位成人搭配 3 件 24 吋行李時，應協助叫大車以確保空間足夠。
~幫客人叫兩台一般的計程車分開坐#雖然分兩台可行，但 SOP 建議針對此人數與行李量優先協助叫大車，這對客人更經濟方便。
~請客人在飯店門口自行攔車即可#櫃檯應主動提供代叫車服務，且需考量行李件數提供正確的車種建議。
~僅幫客人叫一台一般 5 人座車輛，讓客人自行處理行李#一般 5 人座車輛在坐滿 4 位成人後，通常無法塞下 3 件 24 吋大行李。
}

::OP-FO-02-Q08::當客人確定要預約「包車行程」後，櫃檯填寫包車單並收費時，哪項敘述是正確的？ {
~可以直接請客人刷卡支付包車費用#SOP 載明包車付款方式只有現金，不能刷卡。
~櫃檯應按照司機給的 NET 價格直接報給客人#流程規定可將司機給的 NET 價格加報約 10% 給客人。
=包車費用僅限收現金，且需先收款#根據包車流程，付款方式僅限現金，且必須先完成收款動作。
~告訴客人行程結束後直接將錢交給司機即可#SOP 規定櫃檯需先跟客人收款，司機到場後再由櫃檯給予司機對應的款項。
}

::OP-FO-02-Q09::櫃檯夥伴準備進房為客人添加被子或枕頭時，下列哪一項操作符合安全與禮儀規範？ {
~為了保護房客隱私，進房後應立即將門鎖上#SOP 規定放備品時房門須保持「打開」狀態，以避免疑慮並保障自身權益。
~敲門三下後直接使用總卡刷卡進入，節省客人等待時間#應敲門三下並報上身分（Housekeeping/櫃檯），且重複兩次動作確認房內狀況。
=進房前需敲門三下並告知身分，且作業時保持房門開啟#正確流程包含標準敲門報身分動作，以及在房內作業時門需維持開啟以策安全。
~若客人在房內，可將被子拆封後直接鋪在客人的床上#SOP 提到送進房間時不宜待在房內過久，若是客人進房前則放置於靠近門的床上。
}

::OP-FO-02-Q10::有訪客要留下一份包裹轉交給下午才辦理入住的房客，櫃檯應優先確認哪些資訊？ {
~詢問包裹內部的具體價值以便索取押金#轉交物品重點在於資訊核對，而非物品價值或押金。
=留下寄件人與取件人的全名、手機以及預計取件時間#轉交物品流程要求確實記錄雙方的全名、手機，以及取件時間，以便核對身分與後續追蹤。
~請寄件人先行拍照存證，櫃檯不負責紀錄#櫃檯有責任確實交接物品數量、位置及顧客資訊，而非推卸紀錄責任。
~只需確認房號，不需留下聯絡電話#若逾期未取需主動致電顧客，因此手機資訊是必須留下的。
}
