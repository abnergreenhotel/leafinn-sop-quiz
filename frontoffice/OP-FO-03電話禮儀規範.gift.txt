$CATEGORY: OP-FO-03電話禮儀規範

::OP-FO-03-Q01::當一位外部訪客致電櫃台，表示要找住在館內的女性朋友，但無法聯繫上對方時，身為櫃台人員，您的第一個回應步驟應為何？ {
=請教對方要找的房客全名以及對方的貴姓。#這能確保在提供任何協助前，先確認雙方的身份資訊，是後續聯繫的基礎。
~直接詢問對方的房號，以便快速查詢系統。#房號屬於敏感資訊，不應由櫃台主動提及，應先確認基本姓名資訊。
~立即幫對方轉接至該性別房客可能入住的所有房間。#隨意轉接會嚴重侵犯其他房客隱私，應先鎖定明確的找尋對象。
~告知對方由於隱私規定，無法協助代為聯繫。#在確認身份前拒絕協助並不符合服務規範，SOP 規定應嘗試協助聯繫。
}

::OP-FO-03-Q02::若您嘗試聯繫房客但房間電話無人接聽，在向訪客詢問留言細節時，除了姓名與電話外，哪一項資訊也是 SOP 規範中必須詢問的？ {
~訪客的國民身分證字號。#除非有特殊安全需求，否則留傳留言通常不需收集身分證字號等過度私密的資訊。
=建議的回電時間。#掌握方便回電的時間能增加房客回訪的成功率，提供更體貼的服務。
~訪客目前所在的地點座標。#地理位置對留言內容通常沒有直接幫助，並非標準紀錄項目。
~房客與訪客認識多久了。#這屬於房客個人社交隱私，櫃台人員不應過度打聽雙方的私交細節。
}

::OP-FO-03-Q03::一位自稱是某房屋仲介的先生致電，強烈要求轉接給執行長談論資產配置。根據規範，您的處理方式應為何？ {
=先協助過濾資訊，若判斷為廣告類可回覆若有需求會再回電，不須另外通知。#針對執行長或總經理的電話，櫃台有責任過濾房仲或廣告等非重要訊息，避免干擾。
~為了展現效率，應立即轉接至執行長辦公室。#未經過濾即轉接廣告電話會造成高階主管的困擾，違反過濾資訊的職責。
~要求對方先傳真營業執照，確認身份後再轉接。#雖然需要確認身份，但要求傳真證照過於繁瑣，且廣告電話應直接過濾而非驗證。
~請對方留下留言內容後，務必立刻衝到執行長辦公室報告。#非重要訊息不需即時處理或特別通知，應依據重要性判斷。
}

::OP-FO-03-Q04::在記錄廠商要留給主管的訊息時，為了確保資訊傳達的精確度，在掛斷電話前應執行哪一個關鍵動作？ {
~詢問對方是否滿意剛才的通話服務。#雖然是禮儀，但對於確保「資訊正確性」的核心目的並無直接關聯。
=將記錄下的資訊複誦一遍向對方再確認。#複誦確認能及時更正聽錯的電話號碼或姓名，確保傳達內容無誤。
~要求對方在電話中等待五分鐘，讓您先輸入電腦。#讓通話者長時間在線等待是不禮貌的，且無助於核對資訊內容。
~直接承諾主管會在十分鐘內回電。#櫃台無法替主管決定具體回電時間，應僅詢問「對方的建議時間」。
}

::OP-FO-03-Q05::當來電者詢問：「請問王大明先生是住在你們 805 號房嗎？」而系統確實顯示王大明在住，您最符合規範的回應是？ {
~「是的，王先生目前在房間內，請稍等我幫您轉接。」#這直接證實了房客的存在並揭露了位置，違反了不主動證實房客是否在館的隱私規範。
~「沒錯，但他現在不在房內，要幫您留言嗎？」#即便告知不在房內，也已經證實了房客的存在，同樣違反隱私保護原則。
=「為您嘗試聯繫房客，請稍後。」（不提及房號或確認在住狀態）#不主動提供房號且不主動證實房客是否存在，能最大程度保護房客的隱私安全。
~「不好意思，我們 805 號房現在住的是林小姐喔。」#這不僅揭露了房客隱私，還涉及散播錯誤資訊或揭露另一位房客的個資。
}

::OP-FO-03-Q06::若您要找的同事正值「非當班時間」或「休假中」，您在處理外部找該同事的電話時應如何處理？ {
~直接給予該同事的私人手機號碼讓對方自行聯繫。#未經同意提供同事私人聯絡方式違反個資保護，且會干擾同事休假。
=告知對方該同事目前不在位置上，並詢問是否留下聯絡資訊請其回電。#這是專業且保護隱私的做法，既能受理需求，又不會揭露過多同事的私人作息。
~告訴對方公司規定休假期間不能處理業務，請他明天再來電。#這種回應過於生硬，應採取協助留言的方式以維持良好的溝通橋樑。
~代為接聽並隨口答應所有廠商的要求。#在不了解業務細節的情況下隨意承諾，會造成後續處理的困擾與責任問題。
}

::OP-FO-03-Q07::在記錄轉達留言時，下列哪一組資訊是 SOP 中明確要求記錄的「完整五要素」？ {
=姓名、手機、公司行號、轉達內容、留言時間與對象。#這五項資訊能讓受訊者清楚知道誰在何時找他、為了何事，以及如何回聯。
~姓名、性別、年齡、穿著打扮、說話語氣。#部分資訊如年齡或語氣過於主觀，並非留言必備的客觀聯繫要素。
~電話號碼、住家地址、電子郵件、通訊軟體 ID。#僅需基本的聯繫電話即可，索取過多聯繫方式會引起通話者反感。
~公司名稱、統編、負責人姓名、公司登記地址。#這屬於徵信資訊，而非一般留言轉達所需的溝通資訊。
}

::OP-FO-03-Q08::當同事正處於「業務忙碌中」，例如正在接聽另一線電話，櫃台協助轉接時應採取的正確態度為何？ {
~先幫對方切換至等待模式，讓對方聽音樂直到同事講完。#長時間等待會讓來電者焦慮，且不知道要等多久，並非最佳處理方式。
~直接掛斷電話，等同事講完再回撥回去。#掛斷電話會讓對方覺得被拒絕，應徵詢對方意願後再行動。
=告知對方目前正忙碌中，徵詢是否留下聯繫資訊請其晚點回電。#主動說明現況並提供替代方案（留言回電），是專業的電話應對表現。
~不斷插播提醒同事有另外一線電話，直到他掛斷舊電話。#干擾同事通話會影響其目前的業務處理，應保持禮貌並在後端協助。
}
