$CATEGORY: OP-FO-14 換房需求作業

::OP-FO-14-Q01::當客人反映房間內有明顯菸味且感到不適時，櫃檯同仁的第一步應如何處理最為恰當？ {
=先對造成客人的不便表示誠摯歉意，並展現共感。#第一時間道歉並展現同理心能有效緩和客人的不滿情緒，這是服務受理的首要標準。
~請客人自行開窗通風，觀察半小時後若仍有味道再處理。#要求客人自行處理異味會降低服務滿意度，且未展現積極解決問題的態度。
~立即告知客人換房需要補收清潔費，以維護客房品質。#若是因為異味（設備或環境問題）而非客人主動喜好要求換房，不應直接要求補收費用。
~要求房務部先去噴灑除臭劑，請客人在房內稍候。#僅噴灑除臭劑可能無法根除菸味，且讓客人在不適的環境中等待並非最佳的服務判斷。
}

::OP-FO-14-Q02::若客人因為單純偏好「高樓層」而要求換房（原房型設備皆正常），櫃檯人員應採取什麼行動？ {
~直接拒絕，告知飯店規定設備故障才能換房。#直接拒絕會損害服務體驗，應視房況提供彈性選擇。
~無條件幫客人升等至高樓層房型以示友好。#若是出於客人個人喜好的非故障換房，應維持公平原則並進行報價作業。
=確認房況後，向客人說明相關作業流程並報價收取價差。#針對因個人喜好產生的換房需求，應確認房況並依規定收取房價差額。
~請客人自行去電訂房網站修改訂單，櫃檯不予受理。#櫃檯應具備直接處理換房需求的能力，推諉給訂房網站會造成客人不便。
}

::OP-FO-14-Q03::在 PMS 系統中執行換房操作後，櫃檯人員必須完成哪項跨部門通知作業？ {
~通知工務部立即檢查新房間的設施是否完善。#新房間應在提供前就是乾淨房，除非有故障，否則不需每次換房都通知工務部。
=通知房務部將舊房改為髒房，並告知新房號以利掌握客蹤。#及時通知房務部能確保舊房間被列入清掃清單，並讓相關部門知道客人已移入新房。
~通知採購部準備補貼給客人的精美小禮物。#採購部負責物料購買，禮物發放通常由現場主管或櫃檯依權限決定，無需通知採購部。
~通知餐廳部延後該位客人的用餐預約時間。#換房流程主要涉及房務與櫃檯，除非客人特別要求，否則不涉及餐飲預約的變更。
}

::OP-FO-14-Q04::當員工帶領客人進行搬遷，抵達舊房間與客人會合時，下列哪項動作是「搬運時」最關鍵的提醒？ {
~提醒客人檢查新房號是否正確。#這是在搬遷後或櫃檯告知時的動作，並非搬運過程中最需檢查的細節。
~告知客人退房時間維持不變，請準時離場。#此時重點在於協助搬遷，過於強調退房時間會顯得缺乏人情味。
=請客人檢查保險箱、衣櫃、浴室、冰箱及充電器等易遺留物。#在搬離舊房現場時提醒檢查易遺忘區域，可大幅降低客人財產損失及後續處理成本。
~提醒客人在新房間不要使用太多插座以免跳電。#此提醒與搬遷流程無直接關聯，且可能讓客人對飯店設施產生負面疑慮。
}

::OP-FO-14-Q05::若客人因個人隱私考量，希望「獨自搬運」行李至新房間，同仁應如何應對？ {
~堅持由員工搬運，以符合飯店五星級服務標準。#過度堅持會侵犯客人隱私，服務應以客人的舒適與選擇為優先。
=尊重隱私，交予新房卡後請客人自行搬遷，並約定舊卡歸還時間。#在尊重客人的同時，明確約定舊卡歸還時間是確保房卡管理安全的重要步驟。
~告知客人若自行搬運造成行李損壞，飯店概不負責。#這種說法過於負面且具威脅感，並非良好的服務溝通方式。
~請客人在舊房間等待，直到主管核准其自行搬運的要求。#這會造成不必要的等待，且自行搬運行李並非需要特殊核准的異常行為。
}

::OP-FO-14-Q06::搬遷完成後，櫃檯人員應於何時致電客房進行關懷回訪？ {
~搬遷完畢後立即致電，以免忘記。#立即致電可能會打擾正在整理行李或熟悉新環境的客人。
=搬遷完成後約 15–30 分鐘，詢問滿意度與設備使用狀況。#此時段客人通常已大致安頓好，能具體回饋新房間的設備是否順手或是否仍有問題。
~等隔天早上客人下樓用餐時，再口頭詢問即可。#回訪時間過久會失去處理問題的時效性，無法體現即時關懷。
~除非客人再次主動致電櫃檯，否則不應打擾客人。#主動回訪是品質追蹤的標準步驟，能展現飯店對服務缺失的補救誠意。
}

::OP-FO-14-Q07::若換房原因是因為「冷氣故障」且無法立即修復，造成客人的極大不便，除了換房外還可視情況提供什麼補償？ {
~免費贈送下一晚的住宿。#這通常超出了標準作業中提及的「服務補償」範圍，成本過高且需更高層級授權。
=提供客訴小禮物或安排延遲退房作為慰問。#針對飯店疏失造成的故障不便，小禮物或延後退房是常見且有效的服務補償方式。
~告知客人這是意外情況，無法提供任何補償。#完全不提供補償會讓受設備故障困擾的客人感受到服務冷漠。
~將客人原本的房費全額退款，並請其免費續住。#過度補償不符合一般經營邏輯，應視嚴重程度提供適度且合理的慰問。
}

::OP-FO-14-Q08::在關懷回訪中，下列哪種話術最符合服務標準用語？ {
~「請問您搬好家了嗎？舊房卡什麼時候要拿下來？」#這聽起來像是在催促客人歸還物品，缺乏服務關懷感。
~「想確認新房間還有沒有菸味？我們已經換了最好的房間給您。」#過於強調房間是「最好的」可能給客人壓力，且不應引導客人回憶不愉快的異味細節。
=「您好，這邊是櫃檯，想詢問您對新房間是否還滿意呢？設備使用都順手嗎？」#此話術包含了身份識別、開放式滿意度詢問以及對設備功能的確認，標準且親切。
~「因為冷氣壞了才幫您換房，現在新房間的冷氣會冷了嗎？」#這會再次強調設備故障的負面印象，不夠委婉。
}
