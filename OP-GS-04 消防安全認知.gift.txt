$CATEGORY: OP-GS-04 消防安全認知

::OP-GS-04-Q01::新進夥伴在進行消防安全教育時，下列關於「消防設備辨識」的執行說明何者正確？ {
~為了節省時間，新進夥伴僅需透過研讀各樓層平面圖來確認設備位置#僅看平面圖無法讓夥伴完全熟悉實際環境與動線，規範要求必須實地走點。
=新進夥伴必須實地走點，確認滅火器、緩降機、警報設備與集合點位置#實際到現場確認（走點）是熟悉消防設備與逃生動線的必要程序，不可僅依賴圖面資訊。
~消防設備辨識屬於房務部專屬職責，客務部夥伴只需了解通報流程#消防安全涉及跨部門協作，所有夥伴皆應參與相關訓練並明確自身任務範圍。
~消防設備辨識只需針對大廳區域，客房樓層則交由值班主管處理#夥伴需要熟悉的是館內各類設備的位置，包含不同樓層的滅火與逃生設施。
~實地走點只需確認滅火器的位置，其他設備如緩降機可於發生狀況時再行摸索#緊急避難設備如緩降機與警報設備同樣屬於必須熟悉的範疇，不得事後摸索。
}

::OP-GS-04-Q02::當你在館內發現初期火勢時，根據標準作業流程，第一步應該怎麼做？ {
~立即搬運重型消防栓進行強力滅火#首要應確認安全與火勢大小，盲目移動重型設備可能造成不必要的延誤或危險。
~在不確認火勢大小的情況下，優先嘗試使用滅火器撲滅#規範要求須先確認自身安全與火勢大小，若無法安全處理應立即撤離。
=先確認自身安全與火勢大小，再決定後續處理方式#在處理任何初期火警前，評估現場危險程度與是否可控是保護人員安全的首要準則。
~先自行撤離至集合點，不需通報他人以免造成恐慌#若無法安全處理，除了撤離外還必須立即通報主管與相關編組。
~尋找水源嘗試澆熄火勢，而不使用滅火器#執行滅火應依館內訓練使用正確的滅火器基本操作。
}

::OP-GS-04-Q03::在通報火警、煙霧或異味時，下列哪一組資訊是「通報話術」中必須包含的關鍵內容？ {
~發現者的員工編號、到職日期與目前心情#這些個人資訊與現場應變無關，會浪費黃金通報時間。
~起火區域最近的販賣機位置與客房房價#通報內容應聚焦於災害現況，而非營運或不相關設施。
=正確樓層、具體區域、火勢狀況以及是否有受困人員#清晰回報樓層、區域、火勢狀況與受困情形，能讓指揮官迅速做出正確決策。
~目前當班的所有人員名單與排班表#通報首重現場火勢與受困者的即時訊息，而非靜態的排班資訊。
~建議的維修廠商與後續賠償方案#在初期火警通報階段，重點在於現場確認與支援，而非事後補償規劃。
}

::OP-GS-04-Q04::有關「疏散引導」的執行細節，下列敘述何者「錯誤」？ {
~應持續提醒住客與同仁不要搭乘電梯#這是正確的執行重點，因為電梯在火災時具有極高風險。
~若遇到行動不便的住客，應優先通報主管協助處理#這是標準程序，特殊住客需要優先通報以便安排必要支援。
~抵達集合點後，應協助確認人員狀況#這是抵達集合點後的關鍵後續任務，以確保全體安全。
=為了讓住客快速離開，應優先指引搭乘員工電梯#規範明文規定疏散引導時應提醒不要搭乘電梯，無論是客梯或員工電梯。
~引導住客撤離時應依館內指定的安全出口與路線#依據指定路線疏散是確保人員不會進入死胡同或受困火場的正確方式。
}

::OP-GS-04-Q05::在疏散至集合點後，負責夥伴需攜帶哪一種文件以確保點名的準確性並確認無房客受困？ {
~飯店所有房型價目表#價目表無法提供目前住宿人員的真實資訊，對點名無幫助。
=當日住客名單，包含預計退房與續住名單#完整的當日住客名單能幫助夥伴準確確認哪些房客應在場，哪些可能仍受困室內。
~客房維修紀錄表與清潔備品清單#維修與清掃紀錄不包含房客點名所需的人員資訊。
~過去一個月內的已退房房客歷史紀錄#歷史紀錄對當下的緊急事故處理與點名作業沒有直接參考價值。
~當日值班夥伴的考核表#人員名單雖然重要，但此處特指確保「房客」安全的點名工具。
}

::OP-GS-04-Q06::關於館內「緩降機」的使用規範，下列何者符合標準作業指導書？ {
~若現場沒有主管，新進夥伴應指導住客自行操作#未受訓者不得自行指導他人操作，以免造成二次意外。
~緩降機只要熟悉存放位置，任何人皆可直接進行實操指導#熟悉位置與「受過訓練」不同，指導他人需經過專業訓練。
=緩降機屬緊急避難設備，僅於受過訓練並依現場指示下使用#由於緩降機操作具有專業性與風險，必須配合訓練內容與現場指揮才能啟動。
~發生火警時，不論樓層高低應優先選擇緩降機而非樓梯#緩降機是特殊避難設備，通常在無法使用樓梯撤離時才依指示使用。
~緩降機存放箱不可上鎖，但可以堆放雜物以掩蓋其位置#設備周邊應保持通暢，且夥伴需熟悉位置，不可遮蔽或推放物品。
}

::OP-GS-04-Q07::事件處理結束後，下列哪一項不是「事件後回報與復原」階段的必要動作？ {
~配合主管完成點名#點名是確認人員安全是否完全無虞的關鍵步驟。
~若發現滅火器已噴灑，應立即回報安排補充#確保消防設備隨時處於備用狀態是復原工作的重要環節。
~針對異常狀況完成紀錄，不可僅口頭帶過#書面紀錄有利於後續調查與改進，是規範中的明確要求。
=向住客推銷飯店的消防設備品牌以獲得額外獎金#復原階段應專注於人員與設備的確認回報，不涉及與事故處理無關的行銷行為。
~若警報系統故障，需立即回報安排檢修#檢查並維修故障設備是恢復館內消防安全等級的必要行動。
}

::OP-GS-04-Q08::在火警通報的原則中，關於「確認火勢」的說法何者最為精確？ {
~必須等待火勢撲滅後才能向上通報，以免被責難#延誤通報會造成不可挽回的後果，應發現異狀即刻通報。
=不得因未完全確認而延誤通報#消防規範強調通報的即時性，發現煙霧或異味即應啟動通報流程。
~通報前應先與其他部門開會討論火災的可能性#緊急通報應迅速明確，不應透過跨部門討論來浪費應變時間。
~只有在看到明火時才需要通報，煙霧則視為房客抽菸#煙霧與異味皆可能是火災前兆，規範要求發現異狀即刻通報。
~優先通報 119 消防局，再通知館內主管#依規範應先啟動館內通報流程並通知相關編組，以利現場即時確認與處置。
}

::OP-GS-04-Q09::飯店的消防自衛編組每隔多久會進行一次訓練？其班別劃分不包含下列哪一項？ {
=每半年月進行訓練；劃分為通報班、滅火班、避難引導班#根據資料內容，編組訓練頻率為「每半年月」，明確區分通報、滅火與避難引導三個主要任務範圍。
~每一年進行一次訓練；劃分為維修班、採購班、行銷班#訓練頻率過低且班別不符合消防應變職責。
~每三個月進行訓練；劃分為前台班、房務班、餐飲班#消防編組是根據「消防任務」而非「部門」來劃分專業應變班別。
~每個月進行訓練；劃分為檢查班、紀錄班、攝影班#班別劃分應著重於通報、滅火與避難引導等實際應變功能。
~每兩年進行訓練；僅劃分為一人全能班#訓練週期太長且缺乏多部門協作與任務分工的原則。
}

::OP-GS-04-Q10::在引導疏散時，夥伴應掌握當日住客名單，此名單必須包含下列哪些房客類型以確保點名無誤？ {
~僅需包含目前在房內的房客名單#無法預期哪些房客目前在房內，應以完整入住名單為準。
~僅包含預計退房的名單，不含新入住的名單#所有住宿者皆需清點，排除特定類型會造成疏漏。
=包含預計退房與續住名單#根據指導書第 4.0 條，名單應涵蓋當日所有可能的住客狀態（預計退房及續住），以確保清點的準確性。
~包含未來一週內預約訂房的名單#未來預約名單與當下火場點名無關，會混淆清點作業。
~僅包含購買早餐券的房客名單#早餐名單並非完整的入住名單，會遺漏不含早餐的住客。
}
