$CATEGORY: OP-FO-02 客務基本認知

::OP-FO-02-Q01::當房客提出欲租借腳踏車時，下列哪一項是櫃檯夥伴在將鑰匙交給客人「之前」必須執行的標準作業步驟？ {
~先向客人收取新台幣 500 元的租賃押金#根據規範，租借腳踏車原則上無須付押金，除非是較為珍貴的物品才需詢問主管。
~請客人至 8 樓洗衣房確認輪胎是否有氣#洗衣房並非存放腳踏車的地點，且確認輪胎狀況應由夥伴在租借前先行完成。
=請客人填寫租借表格並確認目前可租借的車輛數量#標準流程要求先確認庫存數量，並引導客人完成二樓抽屜內的租借單填寫，隨後才拿鑰匙說明使用方式。
~直接請客人至館外自行尋找喜歡的車輛再回報#流程規範需由夥伴拿鑰匙到外面跟客人說明使用方式，而非讓客人自行處理。
~先詢問房客是否具備維修輪胎的基本工具操作能力#調整坐墊雖可能需要工具輔助，但並非租借前的必要確認事項或作業流程。
}

::OP-FO-02-Q02::若一位房客在凌晨 2 點來到櫃檯，反應房內沒有牙刷與刮鬍刀，根據「客房備品提供」規範，夥伴應如何正確應對？ {
~告知房客館內不提供一次性備品，請其自行至鄰近超商購買#飯店雖不主動在房內提供，但規範明確指出顧客可至櫃檯領取。
=告知房客房內不提供一次性備品，若有需要可至櫃檯領取，並主動詢問是否需要其他備品#此做法符合「告知領取地點」以及「主動詢問其他備品需求」的雙重服務標準。
~直接進入房客房間補足所有備品，以展現貼心服務#規範要求須先詢問客人的領取意願，且一次性備品通常是由客人在櫃檯領取。
~請房客先支付 300 元的一次性備品清潔費用#300元是針對「加人」且不含早餐的收費標準，並非領取一次性備品的費用。
~告知房客備品僅能在辦理入住時領取，目前已逾領取時間#資料中並未限制一次性備品的領取時段。
}

::OP-FO-02-Q03::關於行李寄放的分類管理，若房客寄放的是需要「冷藏或冷凍」的伴手禮，應使用哪一種顏色的行李吊牌？ {
~綠色吊牌#綠色吊牌專門用於「預進（入住前）」的行李寄放。
~黃綠色吊牌#黃綠色吊牌是用於「退房後」的行李寄放。
=灰色吊牌#根據行李牌分類規定，灰色吊牌專指冷藏或冷凍的物品。
~紅色吊牌#資料中並未提到紅色吊牌，這是流程外的錯誤干擾選項。
~藍色吊牌#資料中僅定義了綠色、黃綠色與灰色三種吊牌用途。
}

::OP-FO-02-Q04::有一組房客共 4 位成人，帶著 3 件 24 吋的行李箱要求代叫計程車。根據作業指導書，櫃檯夥伴應如何處理最為妥當？ {
~直接撥打 55688 叫兩台普通計程車，確保空間充裕#雖然人數多可分兩台，但規範建議此類人數與行李量可先嘗試叫大車輛。
~告知房客行李過多，請其自行連絡外面的車輛#這違反了櫃檯代叫計程車的服務流程，夥伴應主動協助。
=撥打 55688 並特別要求協助派送「大車輛」以容納人員與行李#規範明確指出若 4 位成人搭配 3 件 24 吋行李，應撥打 55688 協助叫大車。
~請房客將行李寄放櫃檯，改搭大眾運輸工具#此做法並未解決房客代叫計程車的原始需求。
~請房客先支付計程車預約金，以免車輛抵達後取消#流程中僅要求注意取消時間，並無收取預約金的規定。
}

::OP-FO-02-Q05::在處理房客「加備品（如枕頭、被子）」送入房間的流程中，下列哪項行為符合隱私與安全規範？ {
~為了保持神祕感，將備品放在門口後按門鈴即離開#規範要求若客人在房內應詢問送入時機，若房客不在則有特定的放置位置，而非按門鈴後立即消失。
~敲門三下後直接推門進入，並大聲喊叫 Housekeeping#正確敲門流程是「敲門三下 + Housekeeping」，且需重複兩次動作，非直接闖入。
=放置備品時房門須保持打開的狀態，且不宜在房內待過久#保持房門開啟可避免顧客疑慮並保護員工權益，是標準的安全作業要求。
~若客人在房內，直接將備品交給客人，不須進入房間#雖然可以遞交，但規範提到需詢問是否現在送進房內，且若送進房則須遵守房門開啟原則。
~加備品時若房客不在，應將備品放置在浴室地板上以保持整潔#規範建議若房客進房前加備品，應放置在靠近門的「床上」。
}

::OP-FO-02-Q06::房客希望租借摩托車，櫃檯夥伴告知目前「廣群老闆」回覆尚有車輛可借，後續的正確收費程序為何？ {
~請客人在還車時，直接將租車費交給摩托車廠商#規範要求若確定有車可以借，需要先收錢。
=先向客人收取費用，且付款方式僅限現金#根據流程注意事項，摩托車租借若確定有車，需要先收現金。
~請客人在 R 卡上簽名，費用將併入退房時的總帳單扣款#摩托車租借需事先收取現金，不可掛房帳至退房。
~確認車輛品牌後，先向客人收取一半的金額作為訂金#資料並未提到收取「訂金」或比例收費，而是要求直接收錢。
~告知房客本館僅提供諮詢，費用部分需自行與老闆刷卡結算#流程指引由櫃檯填寫合約與派車單並收費（現金）。
}

::OP-FO-02-Q07::關於「包車流程」中的價格報法，櫃檯夥伴應該如何向客人報價？ {
~直接報司機提供的 NET 價格，以展現飯店優待#司機給予的是內部價，夥伴應根據此價格進行調幅後報給客人。
~將司機給的 NET 價格減去 10% 作為促銷優惠報給客人#這樣會導致飯店虧損或利潤損失，與規範方向相反。
=將司機給的 NET 價格多報 10% 給客人#作業標準規定，可依據司機給的 NET 價格多報 10% 給客人作為飯店服務費用。
~請司機直接向客人報價，櫃檯不干涉價格#流程中規定由櫃檯給予客人包車景點資訊及價格，而非讓司機自行報價。
~一律報新台幣 5,000 元，不論行程遠近或人數#價格應根據司機提供的 NET 價格進行計算，而非統一報價。
}

::OP-FO-02-Q08::若有房客需要幫忙轉交物品給另一位即將抵達的友人，櫃檯夥伴應記錄哪些關鍵資訊？ {
~僅需留下物品內容與放置位置即可#這不足以進行後續聯繫或核對領取人身分。
=留下寄物人的姓名電話，以及領取人的姓名電話、預計取件時間#確實交接雙方姓名、手機及取件時間是避免遺失或錯領的重要步驟。
~請寄物人將物品拍照上傳至飯店官方 LINE#流程規定是留下雙方聯絡資訊，並未要求上傳照片。
~要求寄物人先支付 100 元的物品轉交保管費#資料中並無提到轉交物品需收取費用。
~告知物品若存放超過 1 小時，將一律以遺失物處理#規範僅提到逾期未取須「主動致電顧客」，而非直接作為遺失物處理。
}

::OP-FO-02-Q09::關於館內洗衣服務的說明，下列哪一項敘述是「錯誤」的？ {
~8 樓洗衣房提供兩烘兩洗的設備#這是正確的，館內確實提供此項設備。
~洗衣房免費開放時間為上午 8 點至晚上 10 點#這是正確的，資料明確記載了開放時段。
=若客人需要送洗服務，需告知大約需 10 個工作天#錯誤。資料規定送洗的大約工作天為「兩天」，而非十天。
~送洗服務會產生額外的收費#這是正確的，與洗衣房自理免費不同，送洗是會收費的。
~洗衣房內有提供免費的洗衣粉供房客使用#這是正確的，資料註明有提供洗衣粉。
}

::OP-FO-02-Q10::房客詢問是否可以加床，櫃檯夥伴應如何依照規範回覆？ {
~告知加床費用為一晚 500 元（含早餐）#這是「加人」的費用，並非加床服務。
~直接幫房客將兩張單人床併在一起#這與加床服務無關，且資料中未提及此作業。
=禮貌告知本館「無提供加床服務」#規範注意事項明確標註「無加床服務」。
~請房客升等至更大坪數的房型即可提供加床#資料中並未提供加床的替代方案，僅註明無此服務。
~告知房客需自行至 8 樓領取摺疊床墊#館內無加床服務，亦無提供領取床墊的選項。
}
