$CATEGORY: OP-FO-06 迎賓流程說明

::OP-FO-06-Q01::當您在櫃檯與貴賓交談時，根據標準作業規範，下列關於眼神與肢體動作的敘述何者正確？ {
=與客人談話時，眼神必須始終注視著對方，待客人離開後櫃檯人員方可轉身入座。#注視對方展現專業與尊重，而待客人離開後再入座則是維持服務禮儀的完整性。
~為了準備下一位客人的資料，與當下客人交談時可以適時低頭查看電腦。#未保持眼神交流會讓顧客感到不被重視，違反了主動接待的基本原則。
~若櫃檯作業繁忙，在向客人道謝並告知其可以入房後，人員即可先行入座處理公文。#規範要求必須等到客人完全離開視線或離開櫃檯區後，人員才可轉身入座。
~眼神應適度環顧四周以留意其他進店客人，不需要一直注視著正在談話的對象。#規範強調與當前談話對象眼神注視的重要性，以確保服務品質。
~談話時保持微笑即可，眼神注視的時間長短並沒有硬性規定。#SOP明確規定眼神一定要注視著對方，這是建立服務連結的關鍵。
}

::OP-FO-06-Q02::當您看到門口有車輛停靠時，下列哪一項是首要且正確的執行動作？ {
=主動上前關心，並協助貴賓開關車門。#主動上前與協助開關門是迎賓流程中對於車輛抵達的第一步標準動作。
~先站在自動門口觀察是否有行李，再決定是否上前協助。#SOP強調應「主動」上前，而非在遠處觀察或等待。
~請客人自行將車輛停妥後，再前往車後協助搬運行李。#流程中要求應主動上前關心並先協助開門，而非要求客人先自行停妥。
~為了安全起見，應等待司機先行下車後，服務人員再上前問候。#服務標準是看到車輛停靠即應主動上前，提供即時的協助。
~先回櫃檯確認是否有當日預約名單，再決定是否接待該車輛。#迎賓的主動性要求在確認身分前就應先進行初步的關心與協助。
}

::OP-FO-06-Q03::在協助剛抵達的貴賓處理行李時，下列哪一個話術最符合公司規定的標準流程？ {
=「您好，先幫您將行李放到櫃檯。」#此話術明確告知客人接下來的處理方式，並將行李集中於櫃檯方便辦理入住。
~「您好，請問需要幫您把行李拿進去嗎？」#SOP建議使用更為主動的肯定語句來協助客人處理行李，而非詢問式句型。
~「您的行李很多，我幫您放在門口旁邊。」#行李應放置於櫃檯而非門口，以確保財物安全與流程順暢。
~「請您先辦理登記，行李的部分等一下再說。」#這會延誤行李處理時機，標準流程是先協助下行李並引導至櫃檯。
~「請問您的車內還有其他貴重物品嗎？」#雖然是出於關心，但非迎賓流程中規定的第一優先標準話術。
}

::OP-FO-06-Q04::遇到雨天時，關於撐傘與安全維護的作業順序，下列敘述何者最符合規範？ {
=放置小心地滑告示牌的動作，必須與「開關車門」同時或緊接進行。#確保雨天安全是首要任務，告示牌的放置與開關門動作有嚴格的時效關聯。
~應先幫客人撐傘入內，等客人入座後再出來放置小心地滑告示牌。#告示牌的放置應與開門同時或緊接進行，不可延誤至客人入座後。
~若只有一名工作人員，則以撐傘為主，告示牌待雨勢變小後再放。#安全警示在雨天是強制執行的動作，不能因人力或雨勢大小而略過。
~先將告示牌放在櫃檯顯眼處，再拿傘去門口接應客人。#告示牌應放置於受雨水影響的地點，而非僅放在櫃檯。
~撐傘時只需注意不要讓客人淋雨，不需要特別提醒地板濕滑。#規範要求除了撐傘，還必須口頭提醒貴賓「下雨小心地滑」。
}

::OP-FO-06-Q05::雨天接應賓客時，標準的提醒話術應包含下列哪一項內容？ {
=「您好，請讓我為您撐傘，下雨小心地滑喔！」#此話術同時包含了主動服務的意願以及關鍵的安全警示提醒。
~「歡迎光臨，今天雨很大，請快點進來。」#雖然親切，但未依照SOP要求提及撐傘服務與地滑警示。
~「您好，這把傘給您用，請您進門後放在傘架。」#SOP要求應「為顧客撐傘」，而非僅提供雨傘讓顧客自理。
~「不好意思下雨了，請先在車上稍候，我幫您拿行李。」#這會讓顧客在車內等待，正確流程是主動撐傘協助進出。
~「早安，請問是預約的貴賓嗎？外面雨大請小心。」#未提及「撐傘」動作的說明，不符合雨天服務的專屬話術。
}

::OP-FO-06-Q06::當飯店門口出現團體賓客或人數眾多的情況時，自動門的標準操作方式為何？ {
=將自動門常開，先開門後再關閉自動門電源，待客進入後再重新打開電源。#這能避免自動門頻繁開關導致阻礙或夾傷大量進出的賓客，是提升效率與安全的做法。
~指派一名人員站在感應區，以維持門始終處於開啟狀態。#這會浪費人力，且不如直接關閉電源來得穩定安全。
~調整自動門感應靈敏度，讓門開啟的時間延長。#調整設定並非現場執行的標準程序，正確做法是操作電源開關。
~維持正常感應模式即可，不需要特別更改操作方式。#針對團體客，SOP有特別規定以電源控制常開模式以利通行。
~為了節能，應請團體賓客分批進入，避免自動門開啟過久。#服務標準應以顧客通行的便利性與安全為優先，而非強求分批進入。
}

::OP-FO-06-Q07::關於「拿行李」至客房的服務，下列哪一項執行條件與標準是正確的？ {
=在人員有空閒時，應詢問：「您好，需要幫您將行李放置房內嗎？」#此項服務屬於延伸協助，在確保櫃檯運作正常（有空閒）的情況下應主動提議。
~不論忙碌與否，行李一律必須由工作人員送至房內。#SOP中註明此項服務是在「有空閒」的前提下執行。
~為了行李安全，應要求客人自己提領行李至房內。#這違反了主動協助行李的服務標準。
~人員應直接將行李提走，不需要徵詢客人同意。#進入房內前或處理私人財物前，應使用標準話術詢問客人的意願。
~行李服務僅限於雨天提供，平時則以引導路徑為主。#行李服務與天氣無關，是迎賓流程中的常規項目。
}

::OP-FO-06-Q08::當賓客進入大廳，人員進行「主動接待」的第一個動作應該是？ {
=關心客人需求，並說出「早安，午安，晚安您好」。#親切的問候與主動關心需求是建立良好第一印象的首要步驟。
~直接要求客人出示身分證件以利辦理入住。#在問候與關心需求之前就索取證件，顯得不夠禮貌且過於程序化。
~引導客人至沙發區休息，並提供飲用水。#首要動作應是基本的問候與需求確認，而非直接引導休息。
~主動詢問客人今日的交通路況是否順暢。#雖然是社交對話，但非SOP規定的首要標準問候話術。
~先幫客人確認停車場是否有空位。#停車指引是後續流程，第一步應是面對面的問候與接待。
}

::OP-FO-06-Q09::若賓客開車抵達且表達有停車需求時，工作人員應如何處理？ {
=參照入住流程中的停車指引及安排進行處置。#迎賓流程中明確指示，停車具體安排應連結至「入住流程」的相關規定。
~請客人在紅線區暫停，由人員代為泊車。#SOP未提到代客泊車，且應依循正式的停車指引辦理。
~告知客人附近有收費停車場，請其自行前往。#應提供飯店規範的停車指引與安排，而非僅是讓客人自行處理。
~先辦理完入住手續，再告訴客人哪裡可以停車。#停車需求通常在車輛抵達時就需要立即給予指引或安排。
~詢問客人車號後，直接開啟地下室鐵捲門。#應先參照標準的停車指引流程，而非在未經引導下直接動作。
}

::OP-FO-06-Q10::在執行迎賓流程時，關於「開自動門」的作業指導，下列何者正確？ {
=應在顧客進門前，主動按開自動門。#主動為客人開門是營造尊榮感與便利性的關鍵服務動作。
~等客人走到感應區門自動開啟後，再點頭致意即可。#SOP要求應「主動按開」，而非依賴感應器自動運作。
~為了節能，非必要不應手動按開自動門。#服務標準優先於一般節能考量，主動服務是流程核心。
~自動門的操作僅限於雨天，以防止雨水噴入。#開門服務是常規流程，與天氣狀況無必然關聯。
~若雙手提行李，則可不執行按開自動門的動作。#人員應安排作業順序，確保顧客進門前的順暢，行李與開門應並重。
}
